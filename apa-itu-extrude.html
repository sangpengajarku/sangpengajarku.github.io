14. dimana terdapat pilihan Interactive extrude tool?​


Jawaban:

1. interactive blend tool:untuk menggabung kan 2 buah objek yang berbeda menjadi satu objek. 2. interactive contour tool:untuk memberi warna dan gradasi pada objek 3. interactive distort tool:untuk memeberikan efek bayangan pada sebuah objek (shadowing) 4. interactive envelope tool:untuk menggeser atau menghapus node supaya bentuk objek berubah 5. interactive extrude tool:untuk memeberikan efek 3 dimensi pada sebuah objek6. interactive transparency tool:untuk memeberikan efek trangparan pada sebagaian atau seluruh objek.


15. Modifier yang dapat digunakan untuk membuat object agar menjadikan object menjadi tumpul pada sudut sudutnya adalah Select one: a. Blend b. MeshSmoth c. Extrude d. PathDeform e. Path


Modifier yang dapat digunakan untuk membuat objek agar menjadikan objek menjadi tumpul pada sudut-sudutnya adalah MeshSmooth.

Blend digunakan untuk membuat efek duplikasi pada objek sekaligus mentranformasi dan mencampurkan baik dari segi bentuk waupun warna antara dua onjek ke bentuk objek atau warna.

Path Deform digunakan untuk mengubah objek menggunakan spline atau kurva.

Extrude digunakan untuk memberikan kesan 3 dimensi pada objek atau teks.

Mapel : Komputer

Kategori : Software

Kata kunci : Modifier, MeshSmooth, Design
