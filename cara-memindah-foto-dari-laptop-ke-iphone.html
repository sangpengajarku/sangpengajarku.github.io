3. bagaiman ma cara memindahkan foto dari hp ke laptop


koneksikan terlebih dahulu antara ponsel hp dengan laptop dengan menggunakan kabel data, kemudian kita perhatikan di layar hp, di sana akan muncul beberapa pilihan saat kabel data kita hubungkan. pilihlah opsi “hidupkan penyimpanan USB” dan tekan “ok”.

untuk mulai memindahkan data, kita bisa langsung memilih lokasi dimana data baik musik, gambar ataupun video itu berada. Apakah di memori internal ponsel ataukah di memori eksternal.Selanjutnya kita pilih beberapa file yang akan kita pindahkan.KIta bisa memindahkan data dengan cara klik kanan dan pilih “copy” pada laptop. Kemudian kita “paste” data tsb. di dalam folder yang berada di harddisk laptop. Maka proses copy data akan berlangsung dan tunggu hingga selesai.Jika kita sudah selesai, matikan penyimpanan USB

4. bagaimana cara memfoto di laptop


kita ke kamera.........!!!Pertama-tama, kita nyalakan laptop dulu, jika sudah kita buka file atau apapun yang ingin kita ambil gambarnya. Jika ingin tampilan layar diperbesar untuk mem-fokuskan tekan CTRL dan + secara bersamaan. Untuk mengecilkannya kembali tekan CTRL dan - secara bersamaan.

Nah, sekarang untuk mengambil gambarnya, 

Kita tekan tombol ALT dan Print Screen secara bersamaan. atau bisa juga CTRL dan Print Screen secara bersamaan. Kemudian buka paint lalu tekan tombol CTRL dan V secara bersamaan.


Jika ada yang bingung mencari tulisan print screen dilaptop --> kadang print screen pada keyboard laptop ditulis prt scr atau prt sc. cari saja tulisan tersebut di keyboard masing-masing.


Nah, sekarang bagaimana jika tombol prt sc bukan merupakan tombol utama di laptop kita?? misalnya saja tombol prt sc berada pada tombol insert, jadi pada satu tombol tersebut ada tulisan insert (di atas) dan prt sc (di bawah). 

Nah, untuk mengambil gambar, 

Kita tekan tombol ALT, fn dan insert secara bersamaan. Kemudian buka paint dan tekan tombol CTRL dan V secara bersamaan.
