12. harga 2 baju dan rok 3 adalah 600.000, sedangkan harga 3 baju dan 2 rok adalah 570.000. harga sebuah baju dan 2 rok adalah


Jawab:

366.000

Penjelasan dengan langkah-langkah:

2x + 3y = 600.000 ...(1)

3x + 2y = 570.000 ...(2)

Untuk mencari harga sebuah baju dan 2 rok, kita perlu mencari nilai x dan 2y.

Kita bisa menggunakan metode eliminasi dengan mengalikan persamaan (1) dengan 2 dan persamaan (2) dengan 3, sehingga didapatkan:

4x + 6y = 1.200.000 ...(3)

9x + 6y = 1.710.000 ...(4)

Kemudian, kurangi persamaan (3) dari persamaan (4) untuk mengeliminasi variabel y:

5x = 510.000

Dengan membagi kedua ruas dengan 5, kita dapatkan:

x = 102.000

Selanjutnya, substitusikan nilai x ke salah satu persamaan awal (misalnya persamaan (1)) untuk mencari nilai y:

2(102.000) + 3y = 600.000

204.000 + 3y = 600.000

3y = 396.000

y = 132.000

Jadi, harga sebuah baju dan 2 rok adalah:

1x + 2y = 102.000 + 2(132.000) = 366.000.

y = 60.000

Masukan nilai y = 60.000 ke pers.1

2x + 5y = 600.000

2x + 5(60.000) = 600.000

2x + 300.000 = 600.000

2x = 600.000 - 300.000

2x = 300.000

x = 300.000/2

x = 150.000

Buktikan nilai x = 150.000 dan y = 60.000 ke masing-masing persamaan :

2x + 5y = 600.000

2(150.000) + 5(60.000) = 600.000

300.000 + 300.000 = 600.000 (benar)

3x + 2y = 570.000

3(150.000) + 2(60.000) = 570.000

450.000 + 120.000 = 570.000 (benar)

Jadi dapat disimpulkan bahwa nilai x dan y memenuhi nilai persamaan.
