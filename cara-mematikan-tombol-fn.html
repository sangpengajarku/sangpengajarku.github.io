18. Untuk mematikan komputer melalui keyboard ,tombol yang di gunakan adalah.


Jawaban:

Untuk mematikan komputer melalui keyboard, tombol yang digunakan tergantung pada sistem operasi yang digunakan. Berikut adalah tombol yang dapat digunakan untuk mematikan komputer pada beberapa sistem operasi:

Windows: Tekan tombol Windows + X, kemudian pilih "Shutdown" atau "Shut down or sign out" dan pilih "Shut down".

Mac OS X: Tekan tombol Control + Option + Command + Power button (atau Control + Option + Command + Eject).

Linux: Tekan tombol Control + Alt + Delete atau Control + Alt + Backspace, kemudian pilih "Shutdown" atau "Power off".

Perlu diingat bahwa sebaiknya melakukan shutdown melalui sistem operasi dan tidak langsung mematikan komputer dengan tombol power karena hal ini dapat menyebabkan kerusakan pada sistem dan file yang sedang diproses.

26. jika fn+1=2 . fn +fn-1untuk n= 2,3,4,.... dengan f1=1 dan f2=1, maka f5=……… f=fungsi tulis caranya juga


f(n+1) = 2.f(n)+f(n-1)
masukan n=2 ----> f(3) = 2.f(2) + f(1) = 2.1 + 1 = 3
masukan n=3 ----> f(4) = 2.f(3) + f(2) = 2.3 + 1 = 7
masukan n=4 ----> f(5) = 2.f(4) + f(3) = 2.7 + 3 = 17
f(5) = 17
