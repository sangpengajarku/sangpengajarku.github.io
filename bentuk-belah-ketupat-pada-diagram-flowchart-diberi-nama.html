4. Buatkan Flowchart untuk mencari Luas belah ketupat​


Flowchart Luas Belah Ketupat

Flowchart merupakan langkah awal dalam membuat program. Dengan adanya flowchart urutan proses kegiatan menjadi lebih jelas. Biasanya flowchart ditampilkan dalam bentuk symbol atau bagan yang berurutan dan memiliki hubungan satu sama lain. Dengan demikian setiap simbol menggambarkan proses tertentu. Sedangkan hubungan antar proses digambarkan dengan garis penghubung.

Berikut ini merupakan urutan dalam membuat flowchart :

START

INPUT

READ

PROCESS

PRINT

END

berikut adalah tahapan dalam membuat flowchart untuk luas belah ketupat yang saya buat ;!

1.START

2.Masukan data (d1,d2)

3.Komputer membaca data yang kita input (d1,d2)

4.Hitung luas belah ketupat = 1/2*d1*d2

5.Cetak luas belah ketupat

6.finally ( selesai )

Moga membantu

dit : D2 ?
