20. 3. Dista, Ririn, salsa, dan Hasna masing masing memegang kartu bilangan bulat. Jumlah nila keempat kartu tersebut -28. Kartu Dista nilainya 36, dan kartu Hasna Nilainya 14. Jumlah nila karti Ririn dan salsa sama. Jumlah nilai kartu Ririn adalah...​


Jumlah nilai kartu Ririn adalah -39. Untuk menyelesaikan soal ini kita gunakan operasi hitung bilangan bulat.

Penjelasan dengan langkah-langkah:

Diketahui:

Dista, Ririn, salsa, dan Hasna masing masing memegang kartu bilangan bulat.

Jumlah nila keempat kartu tersebut -28.

Kartu Dista nilainya 36.

Kartu Hasna Nilainya 14.

Jumlah nila karti Ririn dan salsa sama.

Ditanya:

Jumlah nilai kartu Ririn adalah...​

Jawab:

Untuk menyelesaikan soal ini kita gunakan operasi hitung bilangan bulat.

Dista + Ririn + Salsa + Hasna = -28

36 + Ririn + Salsa + 14 = -28

Ririn + Salsa + 50 = -28

Ririn + Salsa = -28 -50

Ririn + Salsa = -78

Karena nilai kartu Ririn dan Salsa sama maka,

Ririn = -78 : 2

Ririn = -39

Jumlah nilai kartu Ririn adalah -39.

Jawab:

Barisan aritmetika adalah suatu barisan dengan beda atau selisih antara dua suku berurutan selalu tetap atau konstan.

Bentuk umum barisan aritmetika adalah

U₁, U₂, U₃, ..., Un

⇔ a, a + b, a + 2b, ..., a + (n - 1)b

Suku ke-n dari barisan aritmetika, yaitu:

Un = a + (n - 1)b.

Beda atau selisih antara dua suku berurutan, yaitu:

b = Un - U(n - 1).

Deret aritmetika adalah jumlah suku-suku barisan aritmetika.

Bentuk umum deret aritmetika adalah

Sn = U₁ + U₂ + U₃ + ... + U(n - 1) + Un

⇔ Sn = a + a + b + a + 2b + ... + a + (n - 1)b

⇔ Sn = (2a + (n - 1)b)

⇔ Sn = (a + a + (n - 1)b)

⇔ Sn = (a + Un)

S(n - 1) = U₁ + U₂ + U₃ + ... + U(n - 1)

Sn - S(n - 1) = Un.
