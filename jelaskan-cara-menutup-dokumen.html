Setelah dokumen tersimpan, untuk menutupnya bisa di ikon x atau klik menu file -> close.

11. bagaimana cara menutup sebuah dokumen ? apa langkah - langkah nya ?


save / nosave?
kalau save:
1. klik icon X pada ujung kanan atas
2. akan ada dialog. dialog tersebut berisi save, don't save dan cancel
jika kau pilih save, kamu akan menyimpan dokumennya dahulu lalu menutup dokumen
jika kau pilih don't save, kamu akan langsung menutup dokumen tanpa menyimpan dokumen
jika kau pilih cancel, kamu tidak jadi menutup dokumen

13. apa yang terjadi jika menutup dokumen atau menutup aplikasi jika ada dokumen yang sedang dibuka dan belum di simpan


file akan hilang. namun biasanya masih terdapat backup yang dibuat secara otomatis

mapel : TI

kelas : 7

materi : program

kata kunci : menutup dokumen

kode soal : 11

==============================

apa yang terjadi jika menutup dokumen atau menutup aplikasi jika ada dokumen yang sedang dibuka dan belum di simpan?

jawab : biasanya, sebelum menutup program yang isinya adalah dokumen, akan memunculkan kotak dialog program yang menyatakan bahwa dokumen ini belum disimpan dan disarankan untuk disimpan. ini adalah opsi untuk dokumen yang baru dibuat atau yang sudah pernah disimpan dan diubah isinya. jika dokumennya sudah pernah disimpan tidak diubah isinya sama sekali, program yang berisi dokumen tersebut dapat langsung ditutup tanpa ada kotak dialog program yang menyatakan bahwa file ini sudah disimpan sebelumnya. kalau secara otomatis tersimpannya, itu bisa langsung ditutu. fitur ini sangat membantu ketika saat listrik mati
