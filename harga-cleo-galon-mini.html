kl

Pembahasan

Harga 2 galon = Rp 35000

Berapa harga 1 gelas ?

             = 38000 ml

Jika air dalam 2 galon dituang kedalam gelas 200 ml akan menjadi

= 38.000 : 200

= 190 gelas

Harga 1 gelas = Harga 2 galon : 190 gelas

                       = Rp 35.000 : 190 gelas

                       = Rp 182,21/gelas

14. aquades memproduksi air minum isi ulang menghasilkan 120 galon dengan biaya tetap 150.000 dan biaya variabel per galon 2000. Jika harga jual per galon Rp. 4000, hitunglah L/R​


Jawaban:

pendahuluan :

laba atau rugi diketahui dari hasil selisih total penerimaan yang diperoleh dan total biaya yang dikeluarkan atau rumusnya yaitu :

laba/rugi = TR - TC

keterangan :

├──────────────────────────

┣⊱ ➻ TR: Total penerimaan yang diperoleh │

┣⊱ ➻ TC: Total pengeluaran │

(biaya-biaya yang dikeluarkan) │

├──────────────────────────

rumus untuk mencari TR (Total Revenue) adalah TR=P× Q

sementara untuk mencari TC (Total Cost) adalah

TC = FC + VC

keterangan :

├──────────────────────────

┣⊱ ➻ P : Harga per unit │

┣⊱ ➻ Q : Kuantitas barang yang terjual │

┣⊱ ➻ FC: Biaya tetap │

┣⊱ ➻ VC: Biaya variabel │

├──────────────────────────

Pembahasan

Diketahui :

P = Rp 4.000

Q = 120 galon

FC = Rp 150.000

VC = 2.000

Ditanya:

laba/rugi?

Jawab:

laba/rugi = TR - TC

laba/rugi = (P x Q) - (FC + VC)

laba/rugi = (4.000 x 120) - (150.000 + 2.000)

laba/rugi = 480.000 - 152.000

laba/rugi = 328.000 (*laba)

* karena hasilnya positif maka termasuk laba

misal : semen = a

galon cat = b

2a + b = 420.000 |×2| 4a + 2b = 840.000

3a + 2b = 780.000 |×1 | 3a + 2b = 780.000

----------------------------- -

a = 60.000

SUBTITUSI

2a + b = 420.000

2(60.000) + b = 420.000

120.000 + b = 420.000

b = 420.000 - 120.000

b = 300.000

<=> a + b

<=> 60.000 + 300.000

<=> Rp. 360.000.00
