// Sistem Persamaan Linear Dua Variabel //



Misalkan drum minyak tanah ( x ) dan drum minyak goreng ( y ). Maka persamaan yang dapat dibentuk.

[tex]2x + 3y = 8.000.000 \\ x + 2y = 5.000.000 \\ \\ \\ 3x + 4y = ...[/tex]

Eliminasi kedua persamaan. Hilangkan variabel x.

2x + 3y = 8.000.000 [ × 1 ]
x + 2y = 5.000.000 [ × 2 ]

2x + 3y = 8.000.000
2x + 4y = 10.000.000
-------------------------------- --
- y = - 2.000.000
y = Rp 2.000.000,00

Substitusi harga y ke persamaan kedua.

x + 2y = 5.000.000
x + 2(2.000.000) = 5.000.000
x + 4.000.000 = 5.000.000
x = 5.000.000 - 4.000.000
x = Rp 1.000.000,00

Masukkan harga-harga ke dalam persamaan yang diminta.

3x + 4y
= 3(1.000.000) + 4(2.000.000)
= 3.000.000 + 8.000.000
= Rp 11.000.000,00

Jadi, harga 3 drum minyak tanah dan 4 drum minyak goreng adalah Rp 11.000.000,00.
