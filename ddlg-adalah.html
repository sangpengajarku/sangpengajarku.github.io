1) ALTER

Alter digunakan untuk mengubah struktur dari tabel meliputi mengganti nama tabel, menambah kolom, mengubah kolom, menghapus dan memberikan atribut pada kolom.

Contoh perintah menambahkan kolom tanggal_lahir dengan type data date atau tanggal pada table mahasiswa :

ALTER TABLE siswa ADD tanggal_lahir date;

Drop digunakan untuk membuang atau menghapus table dan database.

DROP DATABASE sekolah123;

DROP TABLE siswa;

3) CRATE

Create digunakan untuk membuat database dan table.

23. jelaskan query yang termasuk dalam DDL


DDL adalah kependekan dari Data Definition Language. Query yang termasuk dalam DDL adalah perintah - perintah yang berhubungan dengan skema struktur sebuah database.

Pembahasan

Beberapa contoh query yang termasuk pada Data Definition Language beserta penjelasannya antara lain sebagai berikut :

Create

perintah yang digunakan untuk membuat database dan juga isi di dalamnya seperti tabel atau view. Syntax atau format penulisan query create adalah sebagai berikut

CREATE DATABASE nama_database; CREATE TABLE nama_tabel Â (nama_kolom1, type_kolom1, Â nama_kolom2, type_kolom2,â€¦);

Alter

perintah yang digunakan untuk melakukan perubahan pada tabel yang telah dibuat. Perubahan tersebut seperti menambah kolom, mengubah kolom, menghapus kolom, atau menambahkan primary dan foreign key. Format penulisan query tersebut adalah seperti berikut

ALTER TABLE nama_tabel Â ADD nama_kolom tipe_data;ALTER TABLE nama_tabelDROP COLUMN nama_kolom;ALTER TABLE nama_tabel MODIFY COLUMN nama_kolom tipe_dataALTER TABLE nama_tabel Â ADD PRIMARY KEY (nama_kolom);\

Drop

perintah yang digunakan untuk menghapus objek yang ada di dalam database seperti database, tabel, atau kolom. Contoh penggunaan perintah ini adalah

DROP DATABASE nama_database;DROP TABLE nama_table;

Truncate

perintah yang digunakan untuk mengosongkan atau menghapus semua records atau data yang ada pada sebuah tabel. Contoh penulisan perintah tersebut adalah sebagai berikut

TRUNCATE TABLE nama_tabel;
