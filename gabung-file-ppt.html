PPT

-Format default MS.PowerPoint 2003

-Pengembangan dari PPS

-Berbasis Format XML
-Peningkatan yang lebih baik dari PPS

9. file dalam bentuk gambar yang disertai animasi mempunyai ekstensi file....a. .bmpb. .jpgc. .batd. .ppte. .mdb


File dalam bentuk gambar yang disertai animasi mempunyai ekstensi file jpg. JPG (Joint Phographic Experts Assemble) Merupakan file yang sangat umum digunakan dalam media sosial seperti Facebook dan Instagram. Sebab file nya ringan jadi mudah apabila dibagikan.

Pembahasan :

      Jpg merupakan suatu format yang sangat umum digunakan saat ini, terutama untuk transfer gambar. Format ini digunakan untuk menyimpan gambar yang dikompres menggunakan metode JPEG.  Format JPG/JPEG sendiri merupakan singkatan dari Joint Photography Expert Group. Format file ini dapat memampatkan objek ke tingkat kualitas tergantung pada parameter yang disediakan. Format file JPG biasanya digunakan untuk menyimpan gambar yang umumnya digunakan pada halaman web, multimedia, dan publikasi elektronik lainnya.

     Format file JPG juga dikenal sebagai format gambar yang dapat menyimpan suatu gambar dengan warna yang berbeda. Misalnya, gambar dengan warna RGB, CMYK, dan skala abu-abu. Format file ini tidak hanya dapat berisi saluran alfa, tetapi juga ditujukan untuk publikasi elektronik, sehingga relatif  kecil dibandingkan dengan format file lainnya.  

      Format JPG juga dapat mengompresi file dengan menghapus detail. Format file ini sering digunakan dalam format JPG, PDF, dan format yang menggunakan bahasa PostScript. Kompresi ini sangat cocok untuk gambar dengan nada kontinu, seperti foto.

       Format file JPG memiliki karakteristik yang membedakannya dari format file lainnya. Berikut beberapa ciri-ciri format file JPG, Yaitu:

Ekstensi adalah .jpg atau .jpeg.Kemampuan untuk menampilkan warna dengan kedalaman truecolor 24-bit. kompresi gambar  lossy. Biasa digunakan untuk menyimpan gambar diam. Tampilan Progresif.

18. Bagimana cara menggabungkan beberapa file dengan cara link?Apa perbedaan cara menggabungkan beberapa file melalui Embed dengan Copy-Paste?​


Jawaban:

Install Adobe Acrobat pada komputer Anda. Pilihlah versi Standard atau Pro, dan bukan versi trial. Beberapa komputer biasanya sudah dilengkapi dengan Adobe Acrobat. Akan tetapi, tak ada salahnya Anda menginstall versi terbaru.

Buka software Adobe Acrobat, klik tombol Create yang ada di toolbar utama. Jika Anda tidak menemukan tombol ‘Create PDF’, buka View yang ada di toolbar, lalu pilih tombol Task dan lanjutkan dengan memilih Show All Task Buttons.

Pilih menu Combine Files into a single PDF yang ada di dalam Create. Menu ini berfungsi untuk menggabungkan beberapa dokumen yang ada di komputer Anda menjadi satu file PDF.

Klik Browse di window yang muncul pada layar komputer Anda. Kemudian, carilah dokumen yang akan digabungkan dalam satu file PDF. Klik OK dan dokumen yang dipilih akan tersalin ke kotak putih sebelah kanan.

Sortir dokumen yang sudah dipilih sesuai urutan. Di sini, Anda dapat menggunakan tombol Remove, Move up, dan Move Down untuk merapikan data. Hilangkan pula tanda centang di sebelah tulisan Include all open PDF files jika Anda tidak ingin menggunakan file PDF yang pernah Anda buka sebelumnya

Setelah dokumen sudah diurutkan, klik Combine Files dan tekan OK.

Proses penggabungan dokumen menjadi satu file PDF telah selesai, kini saatnya Anda menyimpan file PDF di ‘Save As..’ dengan nama yang Anda inginkan.

30. File ekstensi pada Corel Draw adalah....  a. ppt d. cdr  e. bmp  b. doc c. exe


File ekstensi pada Corel Draw adalah CDR.

Pembahasan

File ekstensi atau yang lebih sering disebut sebagai ekstensi file adalah beberapa huruf yang terletak setelah nama sebuuah berkas atau folder (biasanya ditempatkan setelah tanda titik) yang menandakan identitas aplikasi yang dapat digunakan untuk membuka berkas tersebut. Ekstensi file atau berkas umumnya berlaku unik kepada aplikasi tertentu. Dengan kata lain, umumnya satu ekstensi berkas hanya dapat dibuka oleh satu aplikasi saja. Namun terdapat pula beberapa contoh ekstensi yang dapat dibuka oleh beberapa aplikasi berbeda seperti aplikasi .txt yang dapat dibuka menggunapakan aplikasi Notepad, Notepad++, hingga peramban internet.

Selain itu, sesuai dengan perkembangannya, satu aplikasi dapat pula mengembangkan beragam ekstensi berkas yang berbeda. Sebagai contoh, aplikasi pengolah kata MS Word tercatat pernah menggunakan setidaknya dua format yaitu .doc dan .docx.
