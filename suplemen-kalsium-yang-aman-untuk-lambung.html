1. kalsium merupakan salah satu unsur penting bagi pertumbuhan tulang dan gigi.Sebuah tablet suplemen kalsium mengandung 800mg kalsium fosfat(Mr=310).200 mg kalsium karbonat(Mr=100),dan 5mg kalsium fluorida(Mr=78).Berapakah kandungan kalsium total dalam suplemen tersebut?


m Ca dalam Ca3(PO4)2
= ( 3 Ar Ca / Mr Ca3(PO4)2 ) × m Ca3(PO4)2
= ( 3(40) / 310 ) × 800 mg
= 309,68 mg

m Ca dalam CaCO3
= ( Ar Ca / Mr CaCO3 ) × m CaCO3
= ( 40 / 100 ) × 200 mg
= 80 mg

m Ca dalam CaF2
= ( Ar Ca / Mr CaF2 ) × m CaF2
= ( 40 / 78 ) × 5 mg
= 2,56 mg

m Ca total
= 309,68 mg + 80 mg + 2,56
= 392,24 mg

2. Kalsium merupakan salah satu unsur penting bagi pertumbuhan tulang dan gigi.Sebuah tablet suplemen kalsium mengandung 800 mg kalsium fosfat(Mr=310),200 mg kalsium karbonat(Mr=100),dan 5mg kalsium fluorida(Mr=78).Berapakah kandungan kalsium total dalam suplemen tersebut?


Materi : Stoikiometri
Kelas : X
Kata kunci : kadar, komposisi

Diketahui:
Massa Ca3(PO4)2 = 800 mg
Massa CaCO3 = 200 mg
Massa CaF2 = 78 mg
Mr masing2 senyawa

Ditanya:
Kandungan total Ca dlm suplemen tsb ???


Pembahasan:

Menghitung berat unsur Ca dlm tiap2 senyawa yg diketahui tsb.

Ar Ca = 40

Ca: (3×40/310 × 800) + (40/100 × 200) + (40/78 × 5)
= 392.24 mg

Kandungan total Ca dlm suplemen tsb = 392.24 mg

Jawaban:

Penjelasan:

Persamaan reaksi yang terjadi pada tablet adalah

CaCO₃(s) + 2HCl(aq) → CaCl₂(aq) + CO₂(g) + H₂O(l)

Ar C = 12

Ar H = 1

Ar O = 16

Ar Ca = 40

maka,

Mr CO₂ = 44

Mr CaCO₃ = 100

Menentukan jumlah mol CO₂

n CO₂ = [tex]\frac{massa}{Mr}[/tex]

n CO₂ = [tex]\frac{0,22}{44}[/tex]

n CO₂ = 0,005 mol

Menentukan jumlah mol CaCO₃

CaCO₃(s) + 2HCl(aq) → CaCl₂(aq) + CO₂(g) + H₂O(l)

n CaCO₃ = n CO₂

n CaCO₃ = 0,005 mol

Menentukan jumlah mol Ca

CaCO₃ ---> Ca²⁺ + CO₃²⁻

n Ca = n CaCO₃

n Ca = 0,005 mol

Menentukan massa Ca

massa Ca = n x Ar Ca

massa Ca = 0,005 x 40

massa Ca = 0,2 gram

Menentukan kadar Ca

% Ca =  x 100%

% Ca =  x 100%

% Ca = 10%

29. suplemen antasid mengandung kalsium karbonat sebagai bahan aktifnya. satu tablet antasid yang memiliki berat 1,998 g direaksikan dengan hcl(aq) berlebih menghasilkan 0,22 g co2. berapakah persentase kalsium dalam tablet?


Jawaban

Persentase kalsium adalah 10%

Pembahasan

Persamaan reaksi yang terjadi pada tablet adalah

CaCO₃(s) + 2HCl(aq) → CaCl₂(aq) + CO₂(g) + H₂O(l)

Ar C = 12

Ar H = 1

Ar O = 16

Ar Ca = 40

maka,

Mr CO₂ = 44

Mr CaCO₃ = 100

Menentukan jumlah mol CO₂

n CO₂ = [tex]\frac{massa}{Mr}[/tex]

n CO₂ = [tex]\frac{0,22}{44}[/tex]

n CO₂ = 0,005 mol

Menentukan jumlah mol CaCO₃

CaCO₃(s) + 2HCl(aq) → CaCl₂(aq) + CO₂(g) + H₂O(l)

n CaCO₃ = n CO₂

n CaCO₃ = 0,005 mol

Menentukan jumlah mol Ca

CaCO₃ ---> Ca²⁺ + CO₃²⁻

n Ca = n CaCO₃

n Ca = 0,005 mol

Menentukan massa Ca

massa Ca = n x Ar Ca

massa Ca = 0,005 x 40

massa Ca = 0,2 gram

Menentukan kadar Ca

% Ca = [tex]\frac{massa~Ca}{massa~tablet}[/tex] x 100%

% Ca = [tex]\frac{0,2}{1,998}[/tex] x 100%

% Ca = 10%
