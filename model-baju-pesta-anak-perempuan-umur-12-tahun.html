3. dalam suatu pesta ulang tahun perbandingan anak laki2 dan anak perempuan yang hadir adalah 7 : 4.apabila 5/8 anak laki2 meniggalkan pesta & 20 anak perempuan datang ke pesta, perbandingan laki2 dan perempuan menjadi 1 : 2. berapa jumlah anak yang hadir dalam pesta itu mula mula ? (laki2 = Laki - laki)


Misalkan
jumlah anak laki-laki semula = a
jumlah anak perempuan semula = b
maka dibuat sistim persamaan liniernya:
a : b = 7 : 4
(1 - 5/8)a : (20 + b) = 1 : 2
atau (3a/8) : (20 + b) = 0,5
Kemudian sistim ini diselesaikan:
a : b = 7 : 4 ⇔ a = 7b/4
a = 7b/4 ⇒ (3a/8) : (20 + b) = 0,5
3(7b/4)/8 : (20 + b) = 0,5
21b/32 : (20+ b) = 0,5
21b/32 = 0,5(20 +b)
21b = 16(20+b)
21b = 320 + 16b
5b = 320
b = 320:5 = 64
b=64⇒ a = 7b/4 = 7(64)/4 =7×16 = 112
sehingga a+b = 64+112 = 176
Jawab
Jumlah anak yang hadir dalam pesta itu mula-mula adalah 176, yaitu terdiri dari
112 laki-laki dan 64 perempuan.

Pelajari lebih lanjut : Keuntungan maksimum → https://brainly.co.id/tugas/14949566

Jawab :

Misal : x = model I

           y = model II

Model matematika dari pernyataan diatas

Kain   Model I   Model II   Persediaan

wol         x             2y               4

satin      2x            y                 5

f (x,y) = 600.000 x + 500.000 y

Berdasarkan tabel tersebut diperoleh model matematika berupa pertidaksamaan sebagai berikut.

x + 2y ≤ 4

2x + y ≤ 5

x ≥ 0

y ≥ 0

Menentukan titik potong sumbu x dan sumbu y

x + 2y ≤ 4

x  |   y  |

0  |  2  |

4  |   0 |

Titik (0, 2) dan (4, 0)

2x + y ≤ 5

x     |   y  |

0    |  5  |

2,5 |   0 |

Titik (0, 5) dan (2,5 , 0)

Apabila koefisiennya positif dan mempunyai tanda ≤, maka daerah arsirannya kebawah.

Menentukan titik potong kedua persamaan garis

Eliminasikan pers I dan II

x + 2y = 4  |×1|   x + 2y = 4

2x + y = 5 |×2| 4x + 2y = 10

                      ------------------ --

                      -3x        = -6

                                x = -6/-3

                                x = 2

Subtitusi

x + 2y = 4

2 + 2y = 4

     2y = 4 - 2

     2y = 2

       y = 1

Titik potong kedua garis adalah (2 , 1)

Untuk gambar grafik dan daerah arsiran terdapat pada lampiran

Menentukan penjualan maksimum, fungsi objektif f(x,y) = 600.000 x + 500.000 y

Uji titik pojok

(0, 2) ⇒ 600.000 (0) + 500.000 (2) = 1.000.000

(2, 1) ⇒ 600.000 (2) + 500.000 (1) = 1.700.000  (MAX)

(2,5 , 0) ⇒ 600.000 (2,5) + 500.000 (0) = 1.500.000

Jadi hasil penjualan maksimum kedua  baju tersebut adalah Rp 1.700.000

Seorang penjahit memiliki persediaan 4 m kain wol dan 5 m kain satin. Dari kain tersebut akan di buat dua model baju. Baju pesta pertama memerlukan 2 m kain wol dan 1 m kain satin, sedangkan baju pesta kedua memerlukan 1 m kain wol dan 2 m kain satin, baju pesta pertama di jual dengan harga Rp600.000,00 dan baju pesta kedua seharga Rp500.000,00. Jika baju pesta tersebut terjual, hasil penjualan maksimum penjahit tersebut adalah Rp1.600.000,00. Soal ini merupakan penerapan materi program linear dalam kehidupan nyata.

Pembahasan

Misal

x = baju model pertama y = baju model kedua

Dengan menggunakan tabel, maka diperoleh model matematikanya:

Kain     Baju 1   |  baju 2 |  persediaan

wol       2x              y              4

satin       x               2y            5

harga 600.000x   500.000y    .... ?

Kain wol:

2x + y ≤ 4

Jika x = 0 maka y = 4 ⇒ (0, 4) Jika y = 0 maka x = 2 ⇒ (2, 0)

Tarik garis yang melalui titik (0, 4) dan (2, 0) lalu arsir ke bawah

Kain satin:

x + 2y ≤ 5

Jika x = 0 maka y = 2½ ⇒ (0, 2½) Jika y = 0 maka x = 5 ⇒ (5, 0)

Tarik garis yang melalui titik (0, 2½) dan (5, 0) lalu arsir ke bawah

x ≥ 0 dan y ≥ 0

Fungsi sasaran (harga jual baju)

f(x, y) = 600.000x + 500.000y

Titik potong kedua garis

2x + y = 4 |.2| 4x + 2y = 8

x + 2y = 5 |.1| x + 2y = 5

                    ------------------- –

                      3x = 3

                      x = 1

2x + y = 4

2(1) + y = 4

2 + y = 4

y = 2

titik potong kedua garis adalah (1, 2)

Setelah kita gambar grafik dan daerah penyelesaiannya, maka diperoleh titik titik sudutnya yaitu (0, 2½), (1, 2) dan (2, 0).

Substitusikan ketiga titik tersebut ke fungsi sasaran

f(x, y) = 600.000x + 500.000y

f(0, 2½) = 600.000(0) + 500.000(2,5) = 1.250.000 f(1, 2) = 600.000(1) + 500.000(2) = 1.600.000 f(2, 0) = 600.000(2) + 500.000(0) = 1.200.000

Jadi hasil penjualan maksimum penjahit tersebut adalah Rp1.600.000,00
