1. Rumus umum suku ke-n untuk barisan 1,1,3,5,7,⋯ adalah ⋯⋅⋯⋅Rumus umum suku ke-n untuk barisan 1,1,3,5,7,⋯ adalah ⋯⋅⋯⋅​


Jawaban:

-1, 1, 3, 5, 7

Nilai a = -1

Nilai b = 3 - 1 = 2

Maka, rumus suku ke n barisan

Un = a + (n - 1)b

Un = -1 + (n - 1)(2)

Un = -1 + 2n - 2

Un = 2n - 3 ✔

semoga membantu☯

➸ *Pertanyaan* : rumus umum suku ke-n untuk barisan -1,1,3,5,7,... adalah....

➸ *Jawaban* : Un=a+(n-1)bUn=-1+(n-1)2

Un=-1+2n-2Un= 2n-3

➸ *Pertanyaan* : Rumus umum suku ke-n untuk barisan −1,1,3,5,7,⋯ adalah ⋯

➸ *Jawaban* : a = -1 b = 1-(-1) = 1+1 = 2 maka, a - b = -1 - 2 = -3 Un = 2n - 3

242 + 3 = 5n²
245 = 5n²

 49    = n²
√49   = n
   7    = n
Un = 5n² - 3

Karena Un = 242, maka

Un = 242
⇒ 5n² - 3 = 242
⇒ 5n² - 3 - 242 = 0
⇒ 5n² - 245 = 0
⇒ 5(n² - 49) = 0
⇒ 5(n - 7)(n + 7) = 0

maka n = 7 atau n = -7

Karena n > 0, maka diambil n = 7
