autocad-perintah-line

autocad-shortcut-circle

Perintah ARC: Digunakan untuk membuat objek arc atau lingkaran tidak penuh. Untuk menggunakannya ketik ARC (enter)

autocad-shortcut-arc

autocad-shortcut-fillet

autocad-shortcut-chamfer

Perintah Trim: Digunakan untuk memotong objek yang saling bersinggungan antara satu dengan yang lainnya. Ketik tr atau trim (enter) untuk menggunakannya

autocad-shortcut-trim

Perintah offset: Digunakan untuk membuat duplikasi objek secara offset. Ketik offset (enter) untuk menggunakannya

autocad-shortcut-offset

Perintah mirror: Digunakan untuk melakukan duplikasi objek secara cepat dengan efek seperti cermin (sebaliknya). Ketika Mi (enter) untuk menggunakannya

autocad-shortcut-mirror

Perintah Copy: digunakan untuk menduplikasi objek ditentukan dengan titik pertama dan titik tujuan. Ketik Co (enter) untuk menggunakannya

11. jika ingin membuat segilima dengan panjang sisi 3mm .maka perintah perintah yang digunakan adalah (dalam autocad)​


1.Cara membuat segi banyak di autocad dengan Polygon teknik circumscribed.

Jika panjang sisi 3 mm

History..

1. Ketik “POL” di keyboard

2. Ketik Spasi

3. Masukkan nilai “3” (Jumlah sisi polygon dalam contoh)

4. Ketik “cen” untuk menentukan titik tengah lingkaran

5. Ketik spasi kemudian klik objek circle

6. Klik “Circumsribed” untuk memilih opsi sisi Luar lingkaran

7. Masukkan nilai radius (Contoh 20.1)

8. Ketik spasi untuk mengakhiri perintah

Catatan :

pada langkah ke-4 anda tidak mesti menggambar circle dulu untuk membuat polygon, objek circle di tutorial ini hanya menrupakan contoh yang membuktikan bagaimana perintah polygon dengan teknik “circumscribed” berjalan dengan baik.

2. Cara membuat segi banyak di autocad dengan Polygon teknik inscribed

Polygon dengan teknik ini yaitu dengan cara menentukan penggambaran polygon berada di dalam lingkaran yang anda tentukan jari-jarinya.

History..

1. Ketik “POL” di keyboard

2. Ketik Spasi

3. Masukkan nilai “3” (Jumlah sisi polygon dalam contoh)

4. Ketik “cen” untuk menentukan titik tengah lingkaran

5. Ketik spasi kemudian klik objek circle

6. Klik “Inscribed” untuk memilih opsi sisi Luar lingkaran

7. Masukkan nilai radius (Contoh 20.1)

8. Ketik spasi untuk mengakhiri perintah

Catatan :

pada langkah ke-4 anda tida mesti menggambar circle dulu untuk membuat polygon, objek circle di tutorial ini hanya menrupakan contoh yang membuktikan bagaimana perintah polygon dengan teknik “Inscribed” berjalan dengan baik.

3. Cara membuat segi banyak di autocad dengan Polygon teknik edge

Teknik ini dilakukan dengan cara menggambar polygon dengan menentukan panjang sisi polygon, maka keseluruhan sisi akan memiliki panjang yang sama berdasarkan nilai yang anda kehendaki.

History..

1. Ketik “POL” di keyboard

2. Ketik spasi

3. masukkan nilai 3 (jumlah sisi pada contoh tutorial ini

4. Ketik spasi

5. Ketik “e” untuk opsi Edge

6. Masukkan nilai 3mm (Panjang sisi pada contoh tutorial ini)

Catatan :

Jika ingin polygon tersebut memiliki putaran sudut pada bagian bawahnya anda tinggal memasukkan nilai sudut setelah angka panjang sisi polygon yang anda kehendaki contoh panjang sisi 50 sudut 45 derajat maka ketikkan “50<45” pada langkah ke 6, jika anda ingin lebih jauh bagaimana system putaran sudut ini bekerja pada softwere tinjaulah konten System koordinat dan putaran sudut.

21. 21. Tuliskan tipe extensen untuk program?​


Berikut extensi untuk pemrograman

1. doc dan .docx file dengan extension ini biasanya berupa lembar kerja pengolah data Microsoft Word. Kedua file tersebut merupakan file dari aplikasi yang sama namun dengan seri yang berbeda. .doc merupakan extensi untuk Microsoft Office 2003. Sedangkan .docx merupakan File Microsoft Office 2007, 2010, dan 2013. File Microsoft Office 2003 dapat dibuka dengan aplikasi Microsoft Office 2007. Namun Tidak sebaliknya. Microsoft Office 2007 Tidak bisa dibuka dengan Office 2003. Namun demikian, masih ada cara, agar File .docx bisa dibuka dengan aplikasi Office 2003. Yakni dengan menginstal aplikasi File Format Converter.

2. xls dan .xlsx. File ini juga merupakan produk dari Microsoft Office namun untuk file ini merupakan aplikasi berupa lembar kerja pengolah angka yakni Microsoft Office Excel. mengenai penjelasan tentang aplikasi ini sama dengan file .doc dan .docx

3. .ppt dan .pptx File jenis ini juga masih dalam keluarga besar Microsoft. extensi ini merupakan file yang biasa digunakan sebagai media presentasi. untuk membuka aplikasi ini disebut dengan Microsoft Office Power Point. Penjelasanya sama dengan dua file yang sudah saya jelaskan sebelumnya.

4. .rar dan .zip File ini merupakan file untuk mengompres file. File ini bisa dibuka dengan aplikasi WinZip atau dengan aplikasi WinRar. Aplikasi ini biasa digunakan untuk mengumpulkan beberapa file dalam satu wadah. Sehingga memudahkan jika akan di kirim sebagai attachment email.

5. .pdf merupakan program dokumen yang bisa dibuka dengan program atau aplikasi Adobe Reader atau dengan Nitro pdf.

6. .flv Exstension jenis ini merupakan alamat untuk sebuah file Video. File jenis ini biasanya file hasil download dari Youtube. untuk membuka file ini anda bisa menggunakan aplikasi VLC atau dengan K-Lite.

7. .mkv. File jenis ini merupakan file video sama seperti .flv namun biasanya file dengan jenis ini mempunyai kualitas video yang lebih baik dari .flv. untuk aplikasi pembuka file yang berextensi ini juga bisa menggunakan VLC atau K-lite.

8. .mp3 file jenis ini banyak kita jumpai di kehidupan sehari-hari. File dengan extension ini merupakan file musik tanpa gambar. Untuk bisa membuka file tersebut anda bisa menggunakan Winamp atau Media player yang lain. Karena file dengan extensi ini kompatibel dengan beberapa aplikasi multimedia.

9. .avi file jenis ini masih merupakan file Multimedia. File ini merupakan file video. dan banyak juga kompatibel dengan banyak aplikasi multimedia yang ada.

10. .mp4 masih merupakan file multimedia. File jenis ini merupakan video yang sudah kompatibel dengan aplikasi yang ada di handphone. 

11. .jpg. File exstensi ini merupakan bentuk file gambar. bisa dibuka dengan aplikasi Microsoft Office Picture Manager atau aplikasi editing foto seperti Potoshop.

12. .cdr file dengan extension .cdr merupakan file lembar kerja Corel Draw yang nerupakan aplikasi editing gambar yang biasanya berbasis Vektor.

13. PSD extension ini merupakan jenis file yang bisa dibuka dengan aplikasi Potoshop. aplikasi ini berguna untuk editing foto.

14. DWG extension ini mungkin tak banyak ditemui pengguna komputer umum. Karena file tersebut merupakan lembar kerja Auto Cad. Yakni aplikasi yang digunakan untuk membuat sketsa bangunan. Aplikasi ini biasanya digunakan oleh perusahaan yang bergerak  dibidang konstruksi.

15. ODF extensi file ini tak kalah asing dengan .dwg. File dengan extensi ini bisa dibuka dengan aplikasi open office. Meski aplikasi ini bisa didapatkan secara cuma-cuma alias gratis, namun tak banyak yang menggunakan aplikasi ini karena lebih familiar dengan aplikasi Microsoft Office.

Perintah layer adalah perintah untuk mengatur penggunaan layer. Layer adalah suatu fasilitas yang memungkinkan penggunaan autocad mengorganisasi objek-objek gambar autocad.Tujuh pengelompokkan objek gambar ke dalam layer adalah untuk memudahkan proses menggambar apabila gambar yang sedang ditangani berupa gambar yang kompleks. Dengan menggunakan layer, anda dapat memilah-milah gambar yang ada ke dalam beberapa lapisan (layer=lapisan).Setiap layer diidentifikasi dengan nama layer, warna layer, tipe garis, dan lain-lain. Jika satu layer hendak dihapus atau diedit, kita dapat mengatur sedemikian rupa sehingga layer lain yang terdapat pada gambar yang sama tidak ikut terhapus atau teredit (dengan cara mengunci layer tidak ingin diedit).
