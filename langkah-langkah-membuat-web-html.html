2. langkah langkah membuat web


Cara Membuat Website. 1.Kunjungi situs penyedia Layanan pembuatan Website,disini akan saya beri contoh menggunakan situs penyedia layanan yg cukup populer di Indonesia,yaitu IDHostinger.Langsung saja kunjungi http://www.idhostinger.com/ 2.Setelah terbuka,selanjutnya klik Buat Akun.Letaknya ada di pojok kanan atas,dekat form login. 3.Di halaman ini Anda diwajibkan mengisi.Nama, E-mail dan Password Anda.Isikan sesuai tempatnya.Jangan lupa centang,Saya setuju dengan Ketentuan Penggunaan Layanan. 4.Selanjtunya,kita buka Tab Baru dan masuk ke Email kita,untuk mengaktivasi Akun kita.Nanti ada dua Email dari yg dikirim oleh IDHostinger,Anda lihat saja yg pertama/yg paling bawah.Tingal Anda klik atau salin link tersebut ke Address Bar dan klik Enter. 5.Disini kita akan disuruh memilih Paket Hosting yg akan kita pakai nanti untuk Membuat Website.Karena kita hanya akan Membuat Web untuk Belajar saja,maka kita Order saja yg Paket Gratis. 6.Kalau diatas kita sudah memesan Hostingan makn setelah itu kita akan diberi hadiah SubDomain.Untuk SubDomainnya ada banyak pilihan,silahkan Anda pilih yg paling cocok/sesuai dengan selera Anda.Lalu klik Buat. 7.SubDomain + Hosting Anda akan dibuat, jadi tunggu hingga muncul pemeberitahuan 'Akun telah dibuat 100%' di pojok kanan bawah.Selanjutnya silahkan Anda klik Lihat Hasil. 8.Dari sini kita sudah berhasil membuat akun untuk membuat Websitenya,namun kita belum memesan Hosting + SubDomain/dengan kata lain suah berhasi Mendaftar/Membuat Website.Namun kita harus melanjutkan untuk mengisi website kita dengan Artikel/File.Caranya dengan mengklik tombol Kelola. 9.Maka Anda akan dibawa ke Cpanel (Control Panel).Disinilah tempat Anda mengatur segala sesuatu yg berkaitan dengan website Anda. Mulai dari : Menambah Subdomain Parkir Domain Impor Website Impor Database Mengatur Halaman Error Dan masih banyak lagi.Namun kita tidak akan membahas fitur2 yg ada di Cpanel ini (butuh watu berhari-hari).Tujuan kita masuk Cpanel adalah untuk mengambil Data mengenai : 'Nama pengguna'.

3. klik pilihan Open With ⇒ Internet Explorer

6. langkah-langkah membuat program web


Langkah-langkah membuat Program Web di Komputer Lokal:
1. Install aplikasi web server seperti Wamp atau Xampp.
2. Tentukan Framework yang akan mempermudah dalam pembuatan aplikasi web (misal Zen framework, Codeigniter, Laravel, Yii, dll)
3. Bila tidak menggunakan Framework, Anda bisa memulai menulis code dari awal dengan bahasa pemrograman PHP, HTML, CSS, Ajax, dll.
4. Semua kode yang Anda buat, diletakkan di folder htdocs (misal: c:/xampp/htdocs) dengan file halaman utaman diberinama index.php atau index.html
5. Mulai berkreasi membuat program web. ^^

8. langkah langkah untuk menyimpan sebuah halaman web yang ditampilkan


Menyimpan sebuah halaman web

Pada bagian atas jendela Firefox, klik pada menu File, lalu pilih Save Page As.... Jendela Save As akan muncul.Gunakan jendela Save As untuk memilih lokasi penyimpanan halaman Web sesuai dengan yang Anda kehendaki.Pada panel dropdown Save As Type, pilih format penyimpanan halaman yang diinginkan: 

￼
Web page, complete: Jika Anda memilih opsi ini maka seluruh halaman web akan disimpan bersama dengan seluruh gambar yang Anda di halaman tersebut. Opsi ini dapat digunakan agar Anda untuk melihat halaman web yang telah Anda simpan sebagaimana aslinya, walaupun struktur tautan HTML akan berbeda dibandingkan halaman aslinya. Firefox akan membuat direktori baru tempat seluruh gambar dan berkas yang diperlukan untuk membuka halaman yang akan disimpan.Web page, HTML only: Opsi ini memungkinkan Anda menyimpan halaman asli namun tanpa disertai dengan gambar. Seluruh struktur tautan HTML akan disimpan pada satu berkas sebagaimana halaman aslinya.Text files: Jika opsi ini dipilih, halaman asli akan disimpan sebagai sebuah berkas teks. Struktur tautan HTML tidak akan tersimpan. Akan tetapi dengan opsi ini, Anda dapat membuka hasil simpanan halaman web dengan menggunakan program editor teks.All files: Pada dasarnya sama, opsi ini sama halnya dengan dengan opsi "Web page, HTML only", hanya saja Anda dapat menentukan sendiri ekstensi berkasnya (misal: ".htm" atau ".shtml").Klik Save. Salinan halaman web akan tersimpan sesuai dengan tipe yang telah Anda pilih di lokasi yang telah Anda tentukan.

20. jelaskan langkah langkah untuk mencari web


1.Buka Internet Explorer atau Mozilla Firefox, ketik www.google.co.id pada dialog Address dan tekan Enter
2.Ketik kata kunci informasi yang ingin dicari pada dialog pencarian & tekan Enter.
3.Pilih diskripsi yang paling mendekati isi informasi yang dicari dengan cara mengeklik judul yang bergaris bawah, dalam contoh ini yang kita pilih yang paling atas sendiri (Internet-Wikipedia bahasa indonesia .....).
4.Setelah website ditampilkan blok informasi yang dicari, kemudian klik menu Edit klik Copy
5.Buka program Ms. Word, klik menu Edit → Paste, aturlah sedemikian sehingga informasi mudah dipahami. Di bagian bawah tulis nama pengutib/penyadur dan sumber informasinya (Alamat website).
6.Simpan informasi, klik File → Save → ketik nama Filenya → tekan Enter
7.Keluar dari Ms. Word dan keluar dari Internet Explorer atau Mozilla Firefox dengan cara klik icon close (X) di pojok kanan atas.
*Maafkalo salah 1. buka web mesin pencari. misal : google.com, bing.com, duckduckgo.com, dll
2. masukkan kata kunci yang berhubungan dengan web terkait. misal : jelaskan langkah langkah untuk mencari web.
3. tambahkan dengan tanda petik ("") untuk hasil lebih akurat dan diinginkan. misal : "jelaskan langkah langkah untuk" mencari web
4. jelajahi hasil pencarian hingga ditemukan web yang dimaksud
