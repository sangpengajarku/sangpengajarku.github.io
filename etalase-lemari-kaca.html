19. Berdasarkan ukuran etalase yang dibuat Pak Arya, berapa banyaknya kaca yang dibutuhkan untuk membuat etalase kecil? *


Soal diatas dapat diselesaikan menggunakan rumus luas permukaan balok. Jumlah kaca yang dibutuhkan untuk membentuk etalase kecil adalah 9000 cm².

Penjelasan dengan langkah-langkah:

Diketahui :

Ukuran etalase kecil adalah

p = 50 cm

l = 20 cm

t = 50 cm

Ditanya :

Jumlah kaca yang dibutuhkan

Jawab :

Untuk menyelesaikan soal ini, kita asumsikan bahwa etalase ini nantinya akan berbentuk balok yang memiliki 6 sisi. Maka untuk menentukan jumlah kaca yang dibutuhkan, kita dapat menghitung menggunakan rumus luas penampang balok yang memiliki rumus :

Luas penampang balok = (2 x p x l ) + (2 x p x t ) + (2 x l x t )

Sehingga, Jumlah kaca yang dibutuhkan :

Luas penampang balok = (2 x p x l ) + (2 x p x t ) + (2 x l x t )

                                       = (2 x 50 x 20) + (2 x 50 x 50) + (2 x 20 x 50)

                                       = 2000 + 5000 + 2000

                                       = 9000 cm²

Sehingga, jumlah kaca yang dibutuhkan untuk membuat etalase kecil adalah 9000 cm².
