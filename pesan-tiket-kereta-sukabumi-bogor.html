1. kereta api "bumi geulis" berangkat dari stasiun bogor pada pukul 07.55.kereta itu sampai di stasiun sukabumi pukul 09.55. apabila jarak antara kedua stasiun 57km,berapa kecepatan kereta api tersebut?


Kereta api "bumi geulis" berangkat dari stasiun bogor pada pukul 07.55.kereta itu sampai di stasiun sukabumi pukul 09.55. Apabila jarak antara kedua stasiun 57 km, maka kecepatan kereta api tersebut adalah 28,5 km/jam.

Pembahasan

Kecepatan rata-rata adalah jarak rata-rata yang ditempuh setiap satuan waktu.

Misalkan kecepatan rata-rata sebuah motor 60 km/jam, kecepatan ini bukan berarti motor itu melaju dengan kecepatan 60 km/jam terus, namun ada kalanya motor itu melaju dengan kecepatan kurang dari 60 km/jam ada kalanya motor itu melaju melebihi 60 km/jam. dan kemudian kecepatan yang kurang ditambahkan dengan kecepatan yang melebihi dibagi rata sehingga menghasilkan kecepatan rata-rata.

Rumus kecepatan rata-rata:

Kecepatan rata-rata = jarak tempuh : waktu tempuh

Jarak tempuh = Kecepatan x waktu

Waktu = Jarak : Kecepatan

Penyelesaian

Diketahui:

Waktu yang dibutuhkan = 09.55 - 07.55 = 2 jam

Jarak yang dapat ditempuh = 57 km

Ditanya:

Kecepatan = ...

Jawab:

Kecepatan rata-rata = jarak tempuh : waktu tempuh

                                = 57 km : 2 jam

                                = 28,5 km/jam
