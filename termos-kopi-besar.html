13. Dalam sebuah termos terdapat 1000 g larutan kopi pada suhu 180°C. Kemudian sebanyak 40 g larutan susu pada suhu 10°C ditambahkan ke dalamnya.Kalor jenis larutan susu dan kopi sama besar. Maka suhu akhir campuran tersebut adalah...°C


m1= 1000g
m2= 40g
t1= 180°C
t2= 10°C
c1=c2
tc=?
Qlepas=Qterima
m1. c. Δt1 = m2. c. Δt2
1000 g. (180°C-tc) = 40 g. (tc- 10°c)
180.000°c - 1000 tc = 40tc - 400°c
180.400°c = 1040 tc
tc= 180.400 : 1040
tc = 173,46... °cQ lepas = Q terima
m1.c1.ΔT1= m2.c2.ΔT2
m1.c.ΔT1 = m2.c.ΔT2
m1(T1 - Tc) = m2(Tc - T2)
m1T1 - m1Tc = m2Tc - m2T2
-m1Tc - m2Tc = -m2T2 - m1T1
m1Tc + m2Tc = m2T2 + m1T1
(m1 + m2)Tc = m2T2 + m1T1
Tc = (m2T2 + m1T1) / (m1 + m2)

Tc = (40.10 + 1000.180) / (40 + 1000)
    = (400 + 180000)/1040
    = 173,47°C

Diket:
m1 = 230 g
t1= 90'C
m2 = 20 g
t2 = 5 C
cair=c kopi = c susu = 1 kalor °c
Dit: tc....?

dijwb :
tc = (m1 x t1xc) + (m1 x t2xc)/(m1 + m2)
tc = [(230 x 90 x1 ) + (20x5)] / (230+5)
tc = [20.700 + 100]/235
tc = 20.800/235
tc = 88,51 °

30. dalam sebuah termos terdapat 250 g larutan kopi pada suhu 90⁰c. kemudian sebanyak 70 g larutan susu pada suhu 5⁰c ditambahkan ke dalamnya. berapakah suhu akhir campuran jika dianggap kalor jenis larutan kopi dan susu sama besar?​


Jawaban:

Dik : m1 = 250 g = 0,25 kg

        m2 = 20 g = 0,02 kg

        Δt1 = 90'C

        Δt2 = 5'C

        C1 = C2

Dit : t campuran ?

Jwb :

m1.c1.Δt1 = m2.c2.Δt2

0,25 kg.(90'C - t campuran) = 0,02 kg.(t campuran - 5'C)

22,5 kg'C - 0,25 g t campuran = 0,02 kg t campuran - 0,1 kg'C

22,5 kg'C + 0,1 kg'C = 0,02 kg t campuran + 0,25 kg t campuran

22,6 kg'C = 0,27 kg t campuran

t campuran = 22,6 kg'C / 0,27

                 = 83,703'c
