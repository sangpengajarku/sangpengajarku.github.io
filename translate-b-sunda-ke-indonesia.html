19. Terjemahkan dari bahasa Sunda ke Indonesia× Tidak boleh menggunakan translate× Tidak boleh Asal✓ Jawab semua​


Jawaban:

1. Orang tua saya merasa bangga karena kaka saya menjadi juara

2. Pekerjaan bapak saya menjadi guru

3. maranehna keur arameung di teras.

Penjelasan:

mohon maaf jika saya salah tolong di benarkan, karna manusia tak luput dari yang namanya kesalahan

Jawaban:

1. Orang tua saya merasa bangga karena kakak saya menjadi juara

2. Pekerjaan Ayah saya adalah Guru

3. Arajeunna nuju maèn di padang rumput.

Penjelasan :

1. Orang tua -> Sepuh

Saya -> abdi

merasa -> ngarasa

reueus -> bangga

Kakak -> lanceuk

Menjadi -> jadi

juara -> juara

2. Pekerjaan -> Pakasaban

Ayah -> Bapa

saya -> abdi

Jadi -> jadi

Guru -> guru

pada nomor 2, artinya bukan " pekerjaan ayah saya jadi guru " karena pada kalimat tersebut bila dibaca tidak nyambung.

Maka, kita ubah kalimat nya menjadi " pekerjaan ayah saya adalah guru "

3. Arajeunna -> mereka

Nuju -> sedang

Maén -> bermain / main

di padang rumput -> di halaman.

Pamugi ku kasumpingan urang sadaya kénging ridho ti Alloh SWT.

Bapa sareng Ibu Guru,sareng réréncangan sakola sadaya anu dipikacinta,

Bapa sinareng ibu guru anu dipikacinta, cunduk waktu ninggang mangsa, ...... taun teu karaos sim kuring saparakanca diatik dididik sagalaning elmu pangaweruh anu teu kinten seueurna jasa bapa miwah ibu guru sadaya. Teu hilap ka adi-adi sadaya nu masih kénéh aya di bangku sakola ti kelas ... dugi ka kelas ... , mugi sing getol diajar, sareng sing taat kana piwuruk bapa sareng ibu guru.
