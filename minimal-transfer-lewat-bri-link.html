2. Jelaskan prospek usaha link Bri


BRILink adalah layanan perbankan tanpa kantor yang disediakan BRI melalui agen. BRI Link menawarkan layanan transaksi keuangan real time online dengan memanfaatkan Electronic Data Capture (EDC) atau aplikasi BRILink Mobile.

Agen BRILink memperoleh keuntungan dari biaya transaksi yang dikenakan pada customer. Keuntungan dari biaya transaksi ini dibagi ke Bank BRI dengan konsep sharing fee 50:50.

13. Apa fungsi dari : 1. Radio transfer : 2. baseband link controller : 3. Link Manager : 4. Baseband processing :


Jawaban:

1. Radio Transceiver : berfungsi untuk menerima san mengirim signal berupa gelobang radio

2. Baseband Link Controller : berfungsi untuk menghubungkan perangkat keras radio ke baseband processing dan physical layer

3. Link Manager : berfungsi untuk melakukan autentikasi dan konfigurasi perangkat bluetooth yang terhubung

4. Baseband Processing : Perangkat dalam antarmuka jaringan yang mengelola semua fungsi radio, biasanya di wifi/bluetooth
