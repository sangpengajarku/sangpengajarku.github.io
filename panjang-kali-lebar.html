27. jika panjang persegipanjang ditambah 7cm, maka panjangnya menjadi 2 kali lebarnya, tetapi jika lebarnya ditambah 16cm maka lebarnya menjadi 2 kali panjangnya. maka luas persegi panjang tersebut adalah?


Misal :
Panjang = P
Lebar = L

P + 7 cm = 2 x L
→ P + 7 = 2L
→ P - 2L = -7 ...(Persamaan 1)

L + 16 cm = 2 x P
→ L + 16 = 2P
→ -2P + L = -16 ..(Persamaan 2)

Eliminasi & Substitusi
P - 2L = -7 [Kali 1]
-2P + L = -16 [Kali 2]
..
P - 2L = -7
-4P + 2L = -32
___________(+)
-3P = -39
P = -39 / -3
P = 13 cm ← (Panjang)

Substitusikan (P = 13 ) ke pers (1)
P - 2L = -7
13 - 2L = -7
-2L = -7 - 13
-2L = -20
L = -20 / -2
L = 10 cm ← (Lebar)

Jadi, Luas persegi panjang tsb :
= P x L
= 13 cm x 10 cm
= 130 cm² ← (Jawaban)
