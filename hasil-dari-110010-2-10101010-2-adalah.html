2. hasil dari 110010(2) + 10101010(2) adalah...pakai cara ya


Secara umum penjumlahan bilangan biner dan penjumlahan bilangan desimal adalah sama. Setiap angka akan dijumlahkan satu per satu, namun pada bilangan biner perlu diikuti aturan sebagai berikut

0  +  0  =  0

1   +  0  =  1

0  +   1  =  1

1   +   1  =  1  (carry out 1)

Yang dimaksud dengan carry out adalah penyimpanan nilai untuk angka berikutnya, seperti layaknya pada bilangan desimal ketika 14+7 maka akan menghasilkan 1 dengan carry out 1, sehingga diperoleh 21.


Pembahasan

Dengan memperhatikan aturan penjumlahan biner di atas lakukan penjumlahan untuk masing - masing angkat sehingga

        1  1  0  0  1  0

9. berapa hasil penjumlahan 100101+110010


X = 100101

Y = 110010

Jika dilakukan operasi penjumlahan 100101 + 110010 hasilnya adalah 1010111

Langkah-langkahnya sebagai berikut

1. Ubah angka biner 100101 ke desimal

Cara mengubahnya adalah sebagai berikut

- Tulis angka binernya 100101

- Kalikan tiap digit angka biner dengan pangkat 2

1x2^5 + 0x2^4 + 0x2^3 + 1x2^2 + 0x2^1 + 1x2^0

- Dapatkan hasil pangkat

1x32 + 0x16 + 0x8 + 1x4 + 0x2 + 1x1 = 32 + 0 + 0 + 4 + 0 + 1

- Dapatkan hasil penjumlahan diatas

32 + 0 + 0 + 4 + 0 + 1 = 37

2. Ubah angka biner 110010 ke desimal

Cara mengubahnya adalah sebagai berikut

- Tulis angka binernya 110010  

- Kalikan tiap digit angka biner dengan pangkat 2

1x2^5 + 1x2^4 + 0x2^3 + 0x2^2 + 1x2^1 + 0x2^0

- Dapatkan hasil pangkat

1x32 + 1x16 + 0x8 + 0x4 + 1x2 + 0x1 = 32 + 16 + 0 + 0 + 2 + 0

- Dapatkan hasil penjumlahan diatas

32 + 16 + 0 + 0 + 2 + 0 = 50

3. Setelah mendapatkan nilai desimal, lakukan penjumlahan

37 + 50 = 87

4. Ubah angka 87 menjadi angka biner

Cara mengubahnya adalah sebagai berikut

- 2 dibagi dengan 87 hingga mendapat nilai 0

87/2 = 43, sisa 1

43/2 = 21, sisa 1

21/2 = 10, sisa 1

10/2 = 5, sisa 0

5/2 = 2, sisa 1

2/2 = 1, sisa 0

1/2 = 0, sisa 1

- Baca dari bawah (MSB) ke atas (LSB) sehingga mendapat 1010111

Mapel : TIK

Kategori : Operasi Dasar pada Sistem Operasi

Kata kunci : Desimal, Biner

11. Diketahui bilangan binernya 10101010. Hitunglah bilangan desimalnya

(10101010)₂ = (170)₁₀

Penjelasan:

Cara mengkonversikannya cukup mudah, yaitu setiap bilangan biner dikali dengan 2 dan pangkatnya dimulai 0 dengan urutan dari digit paling kanan, jadi karena binernya 8 digit dan indexnya dimulai dari 0 maka pangkatnya 0-7. Lebih jelasnya seperti ini:

(10101010)₂ = (1 × 2⁷) + (0 × 2⁶) + (1 × 2⁵) + (0 × 2⁴) + (1 × 2³) + (0 × 2²) + (1 × 2¹) + (0 × 2⁰) = (170)₁₀


12. Hasil konversi bilangan 110010 (2) menjadi bilangan desimal adalah … *


Bilangan berbasis dua dapat dikategorikan sebagai bilangan biner. Adapun hasil konversi bilangan 110010 (2) menjadi bilangan desimal adalah 50.

Penjelasan dengan langkah-langkah

Bilangan biner merupakan bilangan berbasis dua. Bilangan biner terdiri atas dua angka, yaitu 0 dan 1. Dalam sistem komputer, bilangan biner dipresentasikan dengan ada atau tidaknya listrik. Untuk mengubah bilangan biner kedalam bilangan berbasis 10 (bilangan desimal), maka dapat dilakukan langkah sebagai berikut:

Kalikan setiap angka pada bilangan biner dari paling kanan ke paling kiri berurutan menggunakan perumusan [tex]\boxed{x=a \times 2^{n}}[/tex] dengan a = angka pada biner dan n = urutan pada angka binerMenambahkan hasil perkalian pada langkah pertamaPenyelesaian :

110010

0 => 0 x 2⁰ = 0 x 1 = 0

1 => 1 x 2¹ = 1 x 2 = 2

0 => 0 x 2² = 0 x 4 = 0

0 => 0 x 2³ = 0 x 8 = 0

1 => 1 x 2⁴ = 1 x 16 = 16

1 => 1 x 2⁵ = 1 x 32 = 32

32 + 16 + 2

= 48 + 2

= 50

Kesimpulan :

Dari sini dapat disimpulkan bahwa hasil konversi bilangan 110010 (2) kedalam bilangan desimal adalah 50
