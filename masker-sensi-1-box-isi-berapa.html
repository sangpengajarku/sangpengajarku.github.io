19. 24. Rata-rata penjualan masker selama seminggu 72 box. Setelah penjualan masker dua hari berikutnya dimasukkan, rata-rata menjadi 74 box. Penjualan hari kedelapan lebih banyak 30 box dari hari kesembilan. Penjualan masker pada hari kedelapan adalah ....tolong pakai cara ya kak maaf ​


Penjualan masker pada hari ke delapan adalah 111 box masker

Penjelasan dengan langkah-langkah:

Soal tersebut merupakan soal matematika yang berkaitan dengan pengolagan suatu data. Rata-rata adalah hasil pembagian dari banyaknya frekuensi dengan banyaknya data.

Untuk menyelesaikan soal tersebut kita harus mencari banyaknya penjualan selama satu minggu dan banyaknya penjualan selama sembilan hari. Setelah kita mengetahui jumlah penjualan kita bisa mengetahui berapa penjualan pada hari ke delapan

Persamaan mencari penjualan selama satu minggu

S1 = X1 × n1

Persamaan mencari penjualan selama sembilan hari

S2 = X2 × n2

Persamaan mencari penjualan di hari ke delapan

x = S2-S1/2 + 30

Dengan :

S1 = penjualan selama satu mingguS2 = penjualan selama sembilan hariX1 = rata-rata penjualan satu mingguX2 = rata-rata penjualan sembilan harin1 = lama penjualan n2 = lama penjualan

Penyelesaiana soal :

Diketahui :

X1 = 72 boxX2 = 74 boxn1 = 7n2 = 9

Ditanyakan :

Berapa penjualan pada hari ke - 8 (x)?

Jawab :

Mencari penjualan selama satu minggu

S1 = X1 × n1S1 = 72 × 7S1 = 504

Mencari penjualan selama sembilan hari

S2 = X2 × n2S2 = 74 × 9S2 = 666

Mencari penjualan hari ke 8

x = S2-S1/2 + 30x = (666-504)/2 + 30x = 162/2 +30x = 81 + 30 x = 111 box

Jadi, penjualan pada hari ke 8 adalah 111 box masker

Model matematikanya

h >= 0
