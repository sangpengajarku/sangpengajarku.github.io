15. Suatu Cuplikan hanya mengandung nitrogen ( Ar N=14 ) dan oksigen ( Ar O = 16 ) bila dalam 216 gram cuplikan terdapat 56 gram nitrogen , maka rumus empiris senyawa tersebut adalah

                Ar  O = 16 gr/ mol

Ditanya : Rumus empiris senyawa ?
Jawab   : Massa Cuplikan - Massa Nitrogen
              216                   - 56
              160 Gram

[tex] \frac{Massa \ N}{Ar \ N} \ \ \ : \frac{Massa \ O}{Ar \ O} \\ \\ \frac{56 \ Gram}{14 \ ^{gram}/_{mol}} : \frac{160}{16 \ ^{gram}/_{mol}} \\ \\ 4 \ \ \ \ \ \ \ \ \ \ \ \ \ : 10 \\ \\ 2 \ \ \ \ \ \ \ \ \ \ \ \ \ : 5 \\ \\ [/tex]

Jadi Rumus Empiris Senyawanya adalah
[tex]N_2 \ O_5 [/tex]

21. Tentukan perbandingan massa nitrogen dan oksigen dalam no2 jika diketahui ar N=14 dan ar O =16?​

Penjelasan:

Jawab:

Menentukan perbandingan massa dalam NO2 dengan cara:

massa C : massa O = jumlah atom C x Ar N : jumlah atom O x Ar O

= 1 x 14 : 2 x 16

= 14 : 32

= 7 : 16

Maka, besarnya perbandingan NO₂ adalah 7 : 16

27. suatu cuplikan hanya mengandung nitrogen (Ar N=14) dan oksigeb (Ar O=16) . Bila dalan 216 gram cuplikan terdapat 56g Nitrogen, maka rumus empiris senyawa tersebut


Kelas         : X
Pelajaran   : Kimia
Kategori     : Stoikiometri
Kata Kunci : rumus empiris, nitrogen, oksigen

Diketahui
Suatu cuplikan (sampel) mengandung N dan O
Dalam 216 gram cuplikan terdapat 56 gram N.
Ar N = 14
Ar O = 16

Ditanya
Rumus empiris 

Penyelesaian

Step-1
Tentukan massa O dalam cuplikan

Massa O = massa cuplikan - massa N
Massa O = 216 - 56
Massa O = 160 gram

Step-2
Tentukan perbandingan mol N dan O

Mol = massa / Ar

Perbandingan mol N dan O adalah 

= [massa N/ Ar N] : [massa O/ Ar O]

= [⁵⁶/₁₄] : [¹⁶⁰/₁₆]

= 4 : 10

=  2 : 5

Setelah menjadi perbandingan paling sederhana, susun menjadi rumus empiris

Jadi rumus empirisnya adalah N₂O₅


28. TOLONG INI BERAPA YA JAWABANNYA??kalau kata guruku, nanti nitrogennya saat mau dicari molnya Pakenya 1×Ar dari nitrogen... itu kenapa ya? padahal nitrogenkan diatomik (N2) kenapa tidak boleh ar N × 2 jadinya 28?.. tolong ya temans...​


Jawaban:

(D.) 60%

Penjelasan:

Diketahui,

• Massa molar nitrogen = 14 gr/mol

• Massa molekul relatif urea = 60 gr/mol

° ° °

Pertama-tama, kita tentukan kadar nitrogen dalam urea murni.

[tex]\% Kadar\ N=\frac{Massa\ Nitrogen}{Massa\ Urea}\times 100\%[/tex]

Kita tahu rumus senyawa dari urea itu adalah CO(NH₂)₂. Dalam rumus tersebut, ada dua buah atom nitrogen. Dengan demikian, massa nitrogen menjadi, [tex]2\times 14\ gram=28\ gram[/tex]

Jadi, besar kadarnya adalah:

[tex]\% Kadar\ N=\frac{28}{60}\times 100\%[/tex]

[tex]\% Kadar\ N=0,4667\times 100\%[/tex]

[tex]\% Kadar\ N=46,67\%[/tex]

° ° °

Kemudian, kita temukan persen kemurnian (g/g) nitrogen dalam sampel urea tersebut.

[tex]\% Kadar\ N=\frac{Massa\ Nitrogen\ dalam\ pupuk}{Massa\ sampel\ urea}\times100 \%[/tex]

[tex]\% Kadar\ N=\frac{280\ gram}{1000\ gram}\times100 \%[/tex]

[tex]\% Kadar\ N=0,28\times100 \%[/tex]

[tex]\% Kadar\ N=28\%[/tex]

° ° °

Terakhir, kita hitung persen kemurnian dari sampel urea tersebut.

[tex]\% Kemurnian\ sampel=\frac{\%\ Kemurnian\ Nitrogen\ sebenarnya}{\%\ Kemurnian\ Nitrogen\ teoritis}\times 100\%[/tex]

[tex]\% Kemurnian\ sampel=\frac{28}{46,67}\times 100\%[/tex]

[tex]\% Kemurnian\ sampel=59,99\%[/tex]

[tex]\boxed{\bold{\% Kemurnian\ sampel\approx 60\%}}[/tex]

° ° °

[tex]\star\ Semoga\ Membantu\ \star[/tex]
