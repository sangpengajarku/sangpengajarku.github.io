Seorang gamer mendownload file sebesar 120 GB, dengan kecepatan download sebesar 50 Mbps, Ketika download sudah mencapai 75%, maka speed download turun menjadi 12 Mbps, dan stabil sampai file 100% didownload. Jawaban:

Penjelasan:

Ketika speed 50Mbps

75% x 120GB = 90GB

90GB X 8bit = 720Gb

720Gb = 720000Mb

720000Mb/50Mbps = 14400s

Ketika speed 12Mbps

120GB-90GB= 30GB

30GB X 8bit = 240Gb

240Gb = 240000Mb

240000Mb/12Mbps = 20000s

Total Waktu Download

14400s+20000s = 34400s

34400s = 9 jam 33 menit 20 detik

23. Seorang gamer mendownload file sebesar 81 GB, dengan kecepatan download sebesar 45 Mbps, Ketika download sudah mencapai 79%, maka speed download berubah menjadi 35 Mbps, dan stabil sampai file selesai didownload. Berapa lama waktu yang dibutuhkan untuk mendownload file tersebut? Jawaban harus disertai dengan cara perhitungan, hasil akhirnya ditampilkan dalam satuan berapa jam, berapa menit, berapa detik.


Jawaban:

31 menit 48 detik

Penjelasan:

Diket

- s = 81GB = 81000mb

- v0 = 45mb/s

- v1 = 35mb/s

- Menjadi glbb diperlambat

- Saat 79% v = v1

Ditanya

t = ?

Jawab

t = s/v

t0 = 81000(79%)/45

t0 = 1422 s

t1 = 81000(100%-79%)/35

t1 = 486 s

t = t0+t1

= 1422+486

= 1908 s

= 31 menit 48 detik
