6. Bu Titik mendapat pesanan menjahit baju kurung dan kebaya. Jumlah kedua pesanan baju tidak lebih dari 8 potong. Banyak baju kurung tidak kurang dari banyak kebaya. Banyak baju kurung yang dipesan tidak lebih dari 3 kali banyak kebaya. Jika upah menjahit satu baju kurung Rp. 30.000 dan satu baju kebaya Rp. 60.000, upah maksimum yang diterima Bu Titik ...


Dengan:
x = Kurung
y = Kebaya

Kasus jumlah:
x + y ≤ 6

Kasus banyaknya baju:
x ≤ 3y

Dengan titik pojok untuk x ≥ 0, y ≥ 0
(0,0), (0,6), (4.5, 1.5)

Cek pada fungsi objektif:
f(x,y) = 30.000x + 60.000y

Diperoleh yang maksimum ketika (0,6)
Dengan keuntungan Rp 360.000,-

k = 2b

2k + 3b = 560,000
2(2b) + 3b = 560,000
7b = 560,000
b = 80,000
k = 160,000

k+2b 
160,000 + 2(80,000)

Kebaya = k
Baju batik = b
1k = 2b
2k + 3b = 560000
1k + 2b = ?
Cara:
Harga 1 kebaya + Harga 2 batik

Harga 1 kebaya:
560000 = 2k + 3b (diubah ke k = 3b : 2 = 1.5k)
560000 = 2k + 1.5k
560000 = 3.5k
1k = 560000 : 3.5k
1k = Rp160000

Harga 1 batik:
1k = 2b
160000 = 2b
1b = 160000 : 2
1b = Rp80000

Harga 1 kebaya + Harga 2 batik:
160000 + 80000(2)
160000 + 160000
160000 x 2
Rp320000
