4. kuota lokal adalah.............


Kuota lokal adalah kuota data yang hanya dapat digunakan pada lokasi tertentu. Yaitu dimana pelanggan melakukan aktivasi paket maka hanya disitulah paket data ini data ini dapat digunakan. Contoh: jika pelanggan melakukan aktivasi paket lokal di kota P, maka paket data ini hanya dapat digunakan selama pelanggan berada di kota P tersebut. Sebaliknya jika pelanggan berada di luar kota P, maka paket data ini tidak dapat digunakan. Dengan demikian pelanggan akan dikenakan paket nasional (jika ada) atau bisa juga dikenakan biaya normal.kuota data yang dapat digunakan haanya pada lokasi tertentu dimana pelanggan melakukan aktivasi paket.

8. apa yg terjadi jika kuota lokal habis?​


Yang terjadi jika kuota lokal kita habis adalah kita tidak dapat mengakses sebuah layanan pada internet dan daerah yang sesuai di-daftarkan maupun tidak, karena kuota lokal kita sudah habis terpakai / masa berlaku habis.

Pembahasan

Paket Internet merupakan sebuah informasi yang memiliki data yang di-salurkan oleh komunikasi digital (Digital Communication) di atas jaringan yang di-transmisikan.

Kuota lokal merupakan paket internet yang disediakan khusus untuk suatu daerah yang telah di-daftarkan sehingga kita dapat merasakan layanan kuota lokal pada internet provider kita. Pada saat kita berada  di-daerah luar yang kita pakai buat mendaftarkan kuota lokal maka paket internet tersebut tidak dapat dipakai.

Perbedaan kuota lokal dengan kuota reguler yang sering ditanyakan

Q = Question | A = Answer

Q: Jika kita menggunakan kuota lokal dapatkah kita menggunakan kuota tersebut di daerah / kota lain?

A: Tidak bisa, kuota lokal hanya dapat diakses pada daerah yang kita daftarkan, sehingga hanya pada daerah dan kota yang kita daftarkan dapat digunakan.

Pada contoh-nya jika kita membeli paket internet 5 Gigabytes lokal dengan tempat yang di-daftarkan yaitu Magelang, kita hanya dapat mengakses paket internet tersebut di kota Magelang. Sedangkan di kota Jakarta kita tidak dapat menggunakan layanan kuota lokal.

13. apa yang dimaksud kuota reguler dan kuota lokal dan perbedaannya ! jelaskan berikan contoh


Kuota REGULER berbeda dengan kuota LOKAL. Kuota regular ini secara umum adalah kuota yang bisa digunakan dengan batas waktu tertentu dan di mana saja. Adapun kuota lokal adalah paket internet yang hanya bisa dipergunakan di lokasi tertentu di mana pembelian dan aktivasi dilakukan.

Ketahui lebih banyak lagi mengenai pengertian kuota https://brainly.co.id/tugas/20000341

» Pembahasan

Aturan penggunaan kuota regular dan kuota lokal ini berbeda-beda pada masing-masing provider. Seperti misalnya provider TRI di mana pengguna harus menghabiskan kuota lokal terlebih dahulu baru kemudian kuota reguler. Terkadang ada beberapa provider yang menawarkan paket internet seperti Telkomsel misalnya yang mencakup kuota lokal dan kuota reguler.

17. File yang berisi konfigurasi lokal DNS server untuk meletakkan zona lokal adalah


Jawaban:

 

File yang berisi konfigurasi lokal DNS server untuk meletakkan zona lokal adalah:

 

etc/bind/ named.conf.local

 

Pembahasan:

 

Pada server jaringan komputer berbasis UNIX/LINUX, pengaturan konfgurasi DNS (Domain Name Seer) dan pengaturan jaringan lainya dilakukan dengan BIND (Berkeley Internet Name Domain).

 

BIND (dan variannya seperti BIND9) adalah server DNS yang paling umum digunakan di Internet, khususnya pada sistem operasi bertipe UNIX/LINUX yang secara de facto merupakan standar.

 

Konfigurasi pengaturan DNS pada bind ini dilakukan dalam serangkaian file.

 

File-file konfigurasi DNS yang tersimpan dalam direktori:

 

/etc/bind

 

Direktori atau folder ini tersusun dari beberapa file. File konfigurasi utama dalam folder ini adalah file:

 

/etc/bind/ named.conf.

 

Sedangkan file-file lainya adalah:

 

/usr/sbin/ named = file binary untuk menjalankan bind9

 

/etc/init.d/ bind9 = file init untuk memudahkan start/stop/restart named

 

/etc/bind/ named.conf.options = file untuk mengatur forwarder untuk keperluan cache nameserver

 

/etc/bind/ named.conf.local = file yang isinya adalah konfigurasi lokal dns server, disini kita akan meletakkan zona lokal.

 

/etc/bind/ named.conf.default-zones = file yang isinya default zones (db.local, db.0, db.127, db.255)

 

Kelas: X

Mata pelajaran: Teknologi Informasi

Materi: Jaringan   

Kata kunci: Konfigurasi DNS

20. Andi membeli ke konter voucer kuota telkomsel 2G dan voucer kuota Indosat 1G, dia harus membayar Rp. 28.000. Di konter yang sama Budi membeli voucer kuota telkomsel 3G dan voucer kuota Indosat 2G. Jumlah uang yang harus dibayarkan Budi Rp. 46.000 . Harga voucer kuota telkomsel 2G dan voucer kuota Indosat 3G adalah …. A. Rp 44.000 B. Rp 40.000 C. Rp 36.000 D. Rp 28.000 E. Rp 24.000​


Jawaban:

A. 44.000

Penjelasan dengan langkah-langkah:

misal: voucher kuota Telkomsel = X

voucher kuota Indosat = Y

Eleminasi X dan Y:

3X+2Y=46.000

2X+Y =28.000

X+Y =18.000

X=18.000-Y

Substitusi X=18.000-Y:

2X+Y = 28.000

2(18.000-Y)+Y = 28.000

36.000-2Y+Y= 28.000

-2Y+Y=28.000-36.000

-Y= -8.000

Y= 8.000

Jika Y=8.000 maka:

2X+Y=28.000

2X+8.000=28.000

2X=28.000-8.000

2X=20.000

X=20.000/2

X=10.000

jadi harga untuk 2G kuota Telkomsel dan 3G Kuota Indosat adalah:

2X+3Y=2(10.000) + 3(8.000)

=20.000+24.000

=44.000
