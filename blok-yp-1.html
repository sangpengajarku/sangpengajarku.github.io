1. Fungsi f(x) = x² - 6x + 8 memiliki titik puncak (xp, yp) maka hasil dari xp + yp adalah ….*02- 113​


Jawaban:

2

Penjelasan dengan langkah-langkah:

Untuk menemukan titik puncak (xp, yp) dari fungsi f(x) = x² - 6x + 8, kita dapat menggunakan rumus xp = -b/2a dan yp = f(xp), di mana a, b, dan c adalah koefisien dari persamaan kuadratik umum y = ax² + bx + c.

Dalam kasus ini, a = 1, b = -6, dan c = 8, sehingga kita memiliki:

xp = -b/2a = -(-6)/(2 x 1) = 3

yp = f(xp) = xp² - 6xp + 8 = 9 - 18 + 8 = -1

Jadi, titik puncak dari fungsi f(x) adalah (3, -1). Dan hasil dari xp + yp adalah:

xp + yp = 3 + (-1) = 2

Jadi, hasil dari xp + yp adalah 2.
