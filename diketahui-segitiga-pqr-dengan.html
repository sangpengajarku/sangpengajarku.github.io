6. Diketahui segitiga PQR adalah segitiga sama sisi dengan panjang sisi masing-masing adalah 8 cm luas dari segitiga PQR adalah ... cm2​


Jawab: 16√3 cm²

Penjelasan dengan langkah-langkah:

Tinggi = 8² - 4² = √48 = 4√3

Luas = alas x tinggi : 2

Luas = 8 x 4√3 : 2

Luas = 32√3 : 2

Luas = 16√3 cm²

Jadi, luasnya 16√3 cm²

20. Diketahui segitiga PQR dengan PQ = 8cm, QR=12cm, dan sudut PQR = 120°. Keliling segitiga PQR adalahh/...................


Bab Trigonometri
Matematika SMA Kelas XI

aturan cosinus
PR = √(PQ² + QR² - 2 x PQ x QR x cos PQR)
PR = √(8² + 12² - 2 x 8 x 12 x cos 120°)
PR = √(64 + 144 - 2 x 96 x (-1/2))
PR = √(208 + 96)
PR = √304
PR = 4√19 cm

keliling = PQ + QR + PR
keliling = 8 + 12 + 4√19
keliling = (20 + 4√19) cm
