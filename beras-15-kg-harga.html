Dik : 1A+2B+3C = Rp. 19.500 ....... persamaan (1) 
2A+3B = Rp. 19.500 ............. persamaan (2) 
1A+1C = Rp. 6.250 .............. persamaan (3) 
Dit : Harga beras jenis mana yang paling mahal ? 

Penyelesaian : 

Persamaan (1) dan (3) 

1A+2B+3C = Rp. 19.500 
1A+1C = Rp. 6.250 
_______________________ _ 

2B + 2C = Rp. 13. 250 ..... (4) 

Persamaan (2) dan (3) 

2A+3B = Rp. 19.500 (x 1) => 2A+3B = Rp. 19.500 
1A+1C = Rp. 6.250 (x 2) => 2A+2C = Rp. 12.500 
__________________ ____________________ _ 
3B - 2C = Rp. 6.500 ........ (5) 

Mencari harga beras jenis B yaitu dengan persamaan (4) dan (5) 

2B + 2C = Rp. 13. 250 
3B - 2C = Rp. 6.500 
____________________ _ 
5B = 19.750 
B = 19.750 / 5 
B = 3.950 
Jadi harga beras jenis B adalah Rp. 3.950 

Untuk mencari harga beras jenis A , subtitusikan harga beras jenis B ke persamaan (2) . 
Maka didapat : 

2A + 3B = Rp. 19.000 
2A + 3(3.950) = 19.000 
2A + 11.850 = 19.000 
2A = 19.000 - 11.850 
2A = 7.150 
A = 7.150 / 2 
A = 3.575 
Jadi, harga beras jenis A adalah Rp. 3.575 

Dan yang terakhir mencari harga beras jenis C mensubtitusikan harga beras jenis A ke dalam persamaan (3). 
Maka didapat : 

1A + 1C = Rp.6.250 
1(3.575) + 1C = 6.250 
C = 6.250 - 3.575 
C = 2.675 

Jadi, harga beras jenis C adalah Rp. 2.675 

Dari penyelesaian di atas, maka didapat : 
Harga beras jenis A = Rp. 3.575 , B = Rp. 3.950 , C = Rp. 2.675 

Jadi jenis harga beras yang paling mahal yaitu harga beras jenis B yaitu Rp. 3.950

Persamaan (1) dan (3) 

1A+2B+3C = Rp. 19.500 
1A+1C = Rp. 6.250 
_______________________ _ 

2B + 2C = Rp. 13. 250 ..... (4) 

Persamaan (2) dan (3) 

2A+3B = Rp. 19.500 (x 1) => 2A+3B = Rp. 19.500 
1A+1C = Rp. 6.250 (x 2) => 2A+2C = Rp. 12.500 
__________________ ____________________ _ 
3B - 2C = Rp. 6.500 ........ (5) 

Mencari harga beras jenis B yaitu dengan persamaan (4) dan (5) 

2B + 2C = Rp. 13. 250 
3B - 2C = Rp. 6.500 
____________________ _ 
5B = 19.750 
B = 19.750 / 5 
B = 3.950 
Jadi harga beras jenis B adalah Rp. 3.950 

Untuk mencari harga beras jenis A , subtitusikan harga beras jenis B ke persamaan (2) . 
Maka didapat : 

2A + 3B = Rp. 19.000 
2A + 3(3.950) = 19.000 
2A + 11.850 = 19.000 
2A = 19.000 - 11.850 
2A = 7.150 
A = 7.150 / 2 
A = 3.575 
Jadi, harga beras jenis A adalah Rp. 3.575

Maka didapat : 

1A + 1C = Rp.6.250 
1(3.575) + 1C = 6.250 
C = 6.250 - 3.575 
C = 2.675 

Jadi, harga beras jenis C adalah Rp. 2.675 

Dari penyelesaian di atas, maka didapat : 
Harga beras jenis A = Rp. 3.575 , B = Rp. 3.950 , C = Rp. 2.675 

Jadi jenis harga beras yang paling mahal yaitu harga beras jenis B yaitu Rp. 3.950
