20. Seorang pedagang membeli sepeda bekas dengan harga Rp 250.000,00 dan biaya perjalanan Rp 50.000,00 Kemudian sepeda bekas tersebut dijual dengan memperoleh untung 20% Berapa harga penjualan sepeda tersebut​


Harga penjualan sepeda tersebut adalah Rp 360.000,00.

Pembahasan

Diketahui :

Harga pokok = Rp 250.000Biaya perjalanan = Rp 50.000Persentase keuntungan = 20%

Ditanyakan :

Berapa harga penjualan sepeda tersebut?

Penyelesaian :

» Menentukan Harga Beli

= Harga pokok + Biaya perjalanan

= Rp 250.000 + Rp 50.000

= Rp 300.000

» Menentukan Keuntungan

= Persentase untung x Harga beli

= 20% x Rp 300.000

= 20/100 X Rp 300.000

= 20 x Rp 3.000

= Rp 60.000

» Menentukan Harga Jual

= Harga beli + Keuntungan

= Rp 300.000 + Rp 60.000

= Rp 360.000 ✅

Kesimpulan :

Jadi, Harga penjualan sepeda tersebut adalah Rp 360.000,00.

21. Pak Dandi membeli sebuah sepeda bekas. Sepeda tersebut diperbaiki. Lalu dijual kembali dengan harga Rp585.000. Dari penjualan sepeda tersebut,pak Dandi mendapat keuntungan 30%. Harga pembelian sepeda bekas tersebut adalah ...


Pembahasan

Pak Dandi membeli sebuah sepeda bekas. Sepeda tersebut diperbaiki. Lalu dijual kembali dengan harga Rp585.000. Dari penjualan sepeda tersebut,pak Dandi mendapat keuntungan 30%. Harga pembelian sepeda bekas tersebut adalah ...

---------------------------------------------------------------------------------------------------------------

Menentukan untung/rugi dari penjualan suatu barang :

        [tex]\boxed{\bold{Untung = Harga \ Jual - Harga \ beli}}[/tex]

        [tex]\boxed{\bold{Rugi = Harga \ beli - Harga \ Jual}}[/tex]


Balik pada soal :

Diketahui :

    Harga Jual   = Rp. 585.000

    Keuntungan = 30%


Ditanya :

    Harga Beli sepeda = ____?


Penyelesaian :

Anggap bahwa awalnya harga pembelian sepeda tersebut adalah :

    Harga Beli = 100% x n → n adalah harga beli awal

    Keuntungan = 30%

    Harga Jual   = 100% + 30%

                         = 130% x n


Karena Pak Dandi memperoleh keuntungan, Maka :

Harga jual = 585.000    

130% x n   = 585.000

             n   = 585.000 : 130%

             n   = 585.000 : [tex]\frac{130}{100}[/tex]

             n   = 585.000 x [tex]\frac{100}{130}[/tex]

             n   = Rp. 450.000


Jadi, Harga Beli sepeda bekas tersebut adalah Rp. 450.000
