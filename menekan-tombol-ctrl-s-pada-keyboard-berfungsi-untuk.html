1. apakah fungsi dari menekan tombol shift + Ctrl + S pada keyboard?​


Jawaban: CTRL + S Berfungsi untuk menyimpan dokumen).

Penjelasan:

SEMUGA MEMBANTU MAAF KALAU SALAH

Jawaban:

Fungsi tombol shift pada tombol keyboard yaitu :

Membuat huruf kapital dan menjadikan lowercase pada kondisi capslock on Sebagai shortcut membuka Recycle Bin Sebagai cara menghilangkan tampilan loading ketika membuka Microsoft Office Sebagai shortcut mencari kata/kalimat pada website dan dokumen Sebagai cadangan shortcut jika tidak bisa klik kanan pada mouse Sebagai shortcut pada browser untuk mengakses kembali link yang tertutup baru saja Sebagai tindakan represif saat komputer hang/freeze/error

Pembahasan:

1) Membuat huruf kapital dan menjadikan lowercase pada kondisi capslock on sekaligus mengakses simbol diatas keyboard angka

Pada kondisi Capslock off, jika kita menekan shift lalu menekan salah satu huruf dikeyboard, maka akan muncul menjadi huruf kapital pada komputer. Hal tersebut akan berlaku sebaliknya jika capslock dalam kondisi on, maka huruf kapital akan menjadi lowercase.

Pada keyboard, simbol diatas angka dapat dimunculkan dengan menekan shift + angka maka akan muncul simbol yang ada diatas tombol angka di keyboard.

2) Sebagai shortcut menghapus permanen file

Dengan menekan tombol Shift + Delete / Del pada file yang sudah di select, file tersebut akan terhapus secara permanen dan tidak akan ditemukan di Recycle Bin

3) Sebagai cara menghilangkan tampilan loading ketika membuka Microsoft Office

Tampilan loading saat membuka microsoft office dapat dihilangkan selain menggunakan tombol minimize adalah dengan menekan shift beberapa lama sebelum membuka program.

4) Sebagai shortcut mencari suatu kata/kalimat pada website dan dokumen

Biasanya dalam mencari kata/kalimat kita menggunakan Ctrl + F, akan tetapi dapat dilakukan juga dengan tombol Shift yaitu kombinasi Shift + F3.

5) Sebagai cadangan shortcut jika tidak dapat melakukan klik kanan pada mouse

Jika sewaktu - waktu mouse maupun touchpad kita memiliki kendala untuk mengakses 'klik kanan' maka dapat dilakukan dengan menekan tombol Shift + F 10

6) Sebagai shortcut pada browser untuk mengakses kembali link yang tertutup baru saja

Setelah menutup tab yang terdapat link secara sengaja maupun tidak dan ternyata kita membutuhkan kembali dapat dilakukan dengan menekan Shift + Ctrl + T, maka link sebelumnya yang tertutup akan kembali

7) Sebagai sarana tindakan represif saat komputer hang/freeze/error

Saat komputer mengalami hang/freeze/error dapat diatasi dengan menekan Ctrl + Shift + Esc. Cara ini cukup mirip dengan penggunaan Ctrl + ALt + Del yang mengakses Task Manager maupun Alt + F4 dalam melakukan force close program.
