B ( 2 , 1 , -1 )

C ( 2 , 0 , -3 )

vektor AB = i + j + k = 1i + 1j + 1k

vektor AC = i - k = 1i + 0j -1k

[vektor AB] . [vektor AC] = (1).(1) + (1).(0) + (1).(-1) = 1 + 0 - 1 = 0

Rumus sudut 2 vektor :

cos α = ( [vektor AB] . [vektor AC] ) ÷ ( |vektor AB| . |vektor AC| )

Karena dot product (perkalian titik) dr kedua vektor tersebut = 0, maka kita tidak perlu menghitung perkalian besar/panjang kedua vektor, karena bilangan "0" dibagi dengan bilangan berapapun hasilnya akan 0.

Maka,

cos α = 0

α = 90°
