Bilangan bulat adalah bulangan yang terdiri atas bilangan negatif , bilangan nol , dan bilangan positif.

Contoh Bilangan bulat

Negatif => -n

Nol => 0

Positif => n

Garis Bilangan :

Operasi Hitung Pada Bilangan Bulat :

PenjumlahanPenguranganPerkalianPembagian

Sifat-sifat Penjumlahan & Pengurangan :

Sifat Tertutup

Contoh :

–8 + 5 = –3

Hasil penjumlahan bilangan-bilangan bulat juga merupakan suatu bilangan bulat.

Sifat Komutatif ( Pertukaran )

Untuk sembarang bilangan bulat a dan b berlaku

a + b = b + a

Sifat Asosiatif ( Pengelompokan )

Untuk sembarang bilangan bulat a , b , dan c berlaku

(a + b) + c = a + (b + c)

Unsur Identitas

Unsur identitas untuk penjumlahan dan pengurangan adalah 0 karena bilangan apapun yang dijumlahkan atau dikurangkan dengan 0 hasilnya tetap bilangan itu sendiri

Sifat-sifat Perkalian :

Sifat komutatif ( Pertukaran )

Untuk setiap a dan b bilbul berlaku

a × b = b × a

Sifat Asosiatif ( Pengelompokan )

Untuk setiap a , b dan c bilbul berlaku

(a × b) + c = a × (c + b)

Sifat Distributif ( Penyebaran )

Untuk setiap a , b dan c bilbul berlaku

a × (b + c) = (a × b) + (a × c)

29. persamaan lingkaran yang pusatnya tertelat pada 2x-4y-4 =0 serta menyinggung sumbu x negatif dan sumbu y negatif adalah


Lingkaran.
Kelompok peminatan kelas XI kurikulum 2013 revisi 2016.

2x - 4y - 4 = 0 → x - 2y - 2 = 0 ... (*)
x dan y harus sama agar menyinggung sumbu X negatif dan sumbu Y negatif. Substitusi x ke y di (*) menghasilkan:
x - 2x - 2 = 0 → x = -2
Pusat lingkarannya (-2, -2), dan r = x² = y², sehingga
(x - a)² + (y - b)² = r²
(x + 2)² + (y + 2)² = 4
x² + y² + 4x + 4y + 4 = 0
