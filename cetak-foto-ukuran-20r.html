5. Ubah suhu berikut ini a. 40 c :........f b. 20r :.........c c. 75 c:.........k tolonh bantu dong besok di kumpul soal nya


Jawaban :

A. ) 40 °C = 104 °F

B. ) 20 °R = 25 °C

C. ) 75 °C = 348,15 K

Penjelasan :

Bagian A

→ Rumus Konversi Celsius ke Fahrenheit

9/5 × C + 32

→ Penyelesaian

40 °C =

= 9/5 × C + 32

= 9/5 × 40 + 32

= 360/5 + 32

= 72 + 32

= 104 °F

Bagian B

→ Rumus Konversi Reamur ke Celsius

5/4 × R

→ Penyelesaian

20 °R =

= 5/4 × R

= 5/4 × 20

= 100/4

= 25 °C

Bagian C

→ Rumus Konversi Celsius ke Kelvin

C + 273,15

→ Penyelesaian

75 °C =

= C + 273,15

= 75 + 273,15

= 348,15 K

Semoga Membantu

19. Sebuah benda homogen berupa silinder dilubangi bagian atasnya berupa kerucut,letak titik berat benda dihitung dari dasar silinder adalah ? A. (0,13/20r) B. (2,20/13r) C. (0, 17/20r) D. (0,20/17r) E. (0,17/13r) Jawabnya C. Tolong dong untuk caranya gimana sih gw bingung. Terimakasih


Anggap benda 1 adalah silinder dengan alas berjari-jari R dan tinggi 2R, maka volumenya = 2*π*R³, dengan y1 = R

Anggap benda 2 adalah kerucut yang hilang volumenya = negatif (1/3)*π*R³, dengan y2 = 2R - (1/4)R = (7/8)R

y = (y1*V1 + y2*V2)/(V1+V2)
= (R * 2*π*R³ + (7/8)R * (-1/3)*π*R³) / (2*π*R³ + (-1/3)*π*R³) = (17/20)R

jawaban:

17p: n = 3, l= 1, m= 0, s= -1/2

6Q: n= 2, l= 1, m= 0, s= +1/2

20R: n=4, l= 0, m= 0, s= -1/2

penjelasan:

17p: 1s2 2s2 2p2 3s2 3p5

hanya diambil elektron terakhir yaitu 3p5

n= 3 ( dari kulit terluarnya yaitu 3 pada 3p5)

l = 1 ( bil.kuantum l untuk s=0, p=1, d=2, f=3)

m=0 ( karena elektron pada orbital berakhir pada kotak tengah yang mana nilainya adalah 0

n= -1/2 ( elektron mengarah kebawah dalam diagram orbital)

6Q= 1s2 2s2 2p2

n=2 ( diambil dari kulit terluar dari elektron terkahir yaitu 2 pada 2p2)

l=1 ( krna berakhir pada sub kulit p)

m=0 ( karna berakhir di kotak kedua pada diagram orbital)

s=+1/2 ( karna elektronnya mengarah keatas)

20R= 1s2 2s2 2p6 3s2 3p6 4s2

n= 4 ( diambil dari kulit terluar yaitu 4 apda 4s2)

l = 0 ( karna berakhir pada sub kulit s)

m=0 ( pada sub kulit s hanya terdapat 1 kotak dengan nilai 0)

s= -1/2 ( elektron mengarah ke bawah)
