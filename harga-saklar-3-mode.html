6. Mode pengambilan keputusan Strategi Mintzberg 1. Mode Kewirausahaan 2. Mode Perencanaan 3. Mode Logika Inkrementalisme​


Jawaban:

Enam mode pengambilan keputusan Henry Mintzberg adalah:

   Mode Kewirausahaan: ditandai dengan pengambilan keputusan cepat dan berorientasi pada peluang berdasarkan intuisi dan pengalaman.    Mode Perencanaan: ditandai dengan pengambilan keputusan tertentu dan menyeluruh berdasarkan penelitian dan analisis.    Mode Inkremental: ditandai dengan pengambilan keputusan bertahap dan terpecah-pecah berdasarkan pendekatan tahap demi tahap dan analisis terbatas.    Mode Konfigural: ditandai dengan pengambilan keputusan fleksibel dan holistik yang memperhitungkan konteks keseluruhan dan interaksi antar variabel.    Mode Politik: ditandai dengan perjanjian politik dan permainan kekuasaan, dimana keputusan diambil berdasarkan distribusi kekuasaan dan pengaruh antar pemangku kepentingan.
