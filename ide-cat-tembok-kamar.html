Biaya pengecatan

= Rp2.000,00/m² * 21m²

= Rp42.000,00/m²

28. tembok bagian dalam kamar dengan ukuran seperti di bawah akan dicat. pintu kamar berukuran 0,8m x 2,2m,dan jendela berukuran 1,2m x 1,6m. jika tiap 1 m persegi permukaan tembok membutuhkan 1 kaleng cat,maka banyak cat yang dibutuhkan adalah?? tolong di bantu ya jawabannya


luas tembok = 2 (l x t) + 2(p x t)
= 2 (3,5 x 3) + (3 x 3)
= 2 (10,5) + 2 (9)
= 21 + 18
= 29 m²

luas pintu = l x p
= 0,8 x 2,2
=1,76 m²

luas jendela = p x l
= 1,2 x 1,6
= 1,92 m²

luas tolat = luas tembok - luas pintu - luas jendela
= 29 - 1,76 - 1,92
= 25, 32 m²

banyak cat yang sigunakan adalah = 25, 32 kaleng = 26 kaleng
