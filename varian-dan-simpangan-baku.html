10. 1. Jelaskan pengertian dari mean, median, modus, desil, persentil, simpangan rata- rata, varian, simpangan baku (standar devisiasi)maaf tolong di bantu ​


Jawab:

Berikut adalah penjelasan singkat mengenai beberapa istilah dalam statistika:

1. Mean atau rata-rata adalah nilai tengah dari kumpulan data. Untuk mendapatkan nilai rata-rata, jumlahkan semua nilai data dan dibagi dengan jumlah data.

2. Median adalah nilai tengah dari kumpulan data yang telah diurutkan. Jika jumlah data ganjil, maka median adalah nilai tengah. Jika jumlah data genap, median dihitung dengan menjumlahkan dua nilai tengah dan kemudian dibagi dua.

3. Modus adalah nilai yang muncul paling sering dalam kumpulan data.

4. Desil adalah nilai dari data yang membagi 10% dari total data. Desil ke-3 (D3) misalnya, membagi data menjadi tiga bagian yang masing-masingnya terdiri dari 30% data.

5. Persentil adalah nilai dari data yang membagi 1% dari total data. Persentil ke-70 (P70) misalnya, adalah nilai yang membagi data menjadi dua bagian, dengan 70% data bernilai di bawah nilai tersebut dan 30% data di atasnya.

6. Simpangan rata-rata adalah jarak rata-rata antara setiap nilai data dengan nilai rata-rata. Untuk menghitungnya, kurangkan setiap nilai data dengan nilai rata-rata, kemudian jumlahkan hasilnya dan dibagi dengan jumlah data.

7. Varian adalah ukuran seberapa jauh sebaran data dari nilai rata-ratanya. Untuk menghitung varian, kuadratkan setiap selisih antara setiap nilai data dengan nilai rata-rata, kemudian jumlahkan hasilnya dan dibagi dengan jumlah data.

8. Simpangan baku atau standar deviasi adalah akar kuadrat dari varian. Simpangan baku menggambarkan seberapa jauh nilai-nilai data tersebar di sekitar nilai rata-ratanya. Semakin tinggi simpangan baku, semakin besar keragaman data yang diamati.

Nilai mean, median, modus berturut-turut untuk data tahun 2018 adalah 380, 350, 350. Nilai mean, median, modus berturut-turut untuk data tahun 2018 adalah 345.71, 320, 400.

Jawab :

Data tahun 2018

Soal nomor 1

Data yang sudah diurutkan : 240, 260, 350, 350, 420, 510, 530

Banyak data (n) = 7

Nilai max = 530

Nilai min = 240

Langkah 1

Mean = Jumlah data / Banyak data

         = (240 + 260 + 350 + 350 + 420 + 510 + 530) / 7

         = 2660 / 7

         = 380

Langkah 2

Median = X(n+1)/2

            = X(7+1)/2

            = X(8)/2

Langkah 3

Data dengan frekuensi tertinggi adalah 350. Maka modus = 350

Kuartil bawah atau Q₁

= 1/4 x (n + 1)

= 1/4 x (7 + 1)

= 1/4 x 8

= 2 → artinya Q₁ terletak di data ke-2

Maka Q₁ = 260

Langkah 5

= 6 → artinya Q₃ terletak di data ke-6

Maka Q₃  = 510

Soal nomor 2

Langkah 1

Rumus varians S² =  (Jumlah setiap suku – rata-rata)^2 / n

S² = {(240 - 380)² + (260 - 380)² + (350 - 380)² + (350 - 380)² + (420 - 380)² + (510 - 380)² + (530 - 380)²} / 7

    = {(-140)² + (-120)² + (-30)² + (-30)² + 40² + 130² + 150²} / 7

    = (19600 + 14400 + 900 + 900 + 1600 + 16900 + 22500) / 7

    = 76800 / 7

    = 10971.42857

Langkah 2

Standar deviasi = √varian

                         = √ S²

                         = √ 10971.42857

                         = 104.7445873

Langkah 3

Range = Nilai max – Nilai min = 530 – 240 = 290

Langkah 4

Interquartile range = Q₃ - Q₁ = 510 – 260 = 250

Langkah 5

Coefficient of Variation = (Standar deviasi / mean) x 100%

                                    = (104.7445873 / 380) x 100%

                                    = (0.275644) x 100%

                                    = 27,56% atau 28%

Langkah 6

Rumus Z-score = (xi – rataan)/standar deviasi, dengan xi = nilai data tertentu

Z-score untuk datum  240 = (240 – 380)/ 104.7445873 = -1.33658

Z-score untuk datum  260 = (260 – 380)/ 104.7445873 = -1.14564

Z-score untuk datum  350 = (350 – 380)/ 104.7445873 = -0.28641

Z-score untuk datum  510 = (510 – 380)/ 104.7445873 = 1.241114

Z-score untuk datum  530 = (530 – 380)/ 104.7445873 = 1.432055

Karena kemiringan/skewness bernilai positif, maka data tersebut berdistribusi miring kanan

15. Peluang rusaknya sebuah bola lampu adalah 0,4. Apabila diambil 550 bola lampu, berapa diharapkan dijumpai bola lampu yang rusak, berapa variannya dan simpangan bakunya


Jumlah bola lampu yang diharapkan rusak adalah sebanyak 220 bola lampu. Adapun varian dari kasus ini adalah 132. Sedangkan, simpangan baku terhitung adalah 11,49.

Penjelasan dengan langkah-langkah

Diketahui:

Jumlah bola lampu:
n = 550.Peluang rusaknya bola lampu:
p = 0,4.

Ditanyakan:

Jumlah bola lampu rusak = N = ?Varian data = σ² = ?Simpangan baku = σ = ?

Penyelesaian:

Langkah 1
Perhitungan jumlah bola lampu yang diharapkan rusak.

N = p x n
N = 0,4 x 550
N = 220.

Langkah 2
Perhitungan varian data.

Peluang bola lampu tidak rusak = q.

q = 1 - p
q = 1 - 0,4
q = 0,6.

Perhitungan varian:

[tex]\begin{array}{ll} \sf \sigma^2 &\sf = n\times p \times q\\\\\sf \sigma^2 &\sf = 550 \times 0,4 \times 0,6\\\\\sf \sigma^2 &\sf = 132.\end{array}[/tex]

Langkah 3
Perhitungan simpangan baku.

[tex]\begin{array}{ll} \sf \sigma &\sf = \sqrt{\sigma^2}\\\\\sf \sigma &\sf = \sqrt{132}\\\\\sf \sigma &\sf = 11,49.\end{array}[/tex]

Simpangan baku adalah salah satu ukuran sebaran dalam statistika. Nama lain dari simpangan baku adalah standar deviasi. Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data. Untuk menentukan simpangan baku, kita harus mencari nilai rata-rata data dan nilai ragam (varians) terlebih dahulu. Simpangan baku dirumuskan sebagai akar kuadrat dari ragam.

Data: 3,3,4,4,5,5,5,6,6,7,7

Rata-rata Data

x̄ = 3+3+4+4+5+5+5+6+6+7+7 / 11

  = 55/11

  = 5

SR = Σ |xi - x̄| / n

    = |3-6|+|3-6|+|4-6|+|4-6|+|5-6|+|5-6|+|5-6|+|6-6|+|6-6|+|7-6|+|7-6| / 6

    = 15/6

    = 2,5

s² = Σ (xi - x̄)² / n

   = (3-6)²+(3-6)²+(4-6)²+(4-6)²+(5-6)²+(5-6)²+(5-6)²+(6-6)²+(6-6)²+(7-6)²+(7-6)² / 6

   = 30/6

s² = 5

s = √5

  = 2,23

27. Apakah yang dimaksud dengan varian overhead? Bagaimanakah varian overhead pada umumnya diperlakukan?


Varians Overhead Pabrik adalah selisih biaya overhead pabrik aktual dengan biaya overhead pabrik berdasarkan standar yang diperkenankan.

Dalam menganalisis biaya overhead pabrik ini dapat dilakukan dengan :

###Metode Dua Varians

1) Varians Terkendali (Controllable Variance), adalah varians antara biaya overhead pabrik aktual dengan total anggaran fleksibel pada aktivitas standar yang diperbolehkan .

Rumus :

(BOP Aktual) - (BOP tetap aktifitas normal + BOP Variabel aktifitas standar)

2) Varians Volume (Volume Variance), adalah varians antara anggaran fleksibel pada aktivitas standar dari dasar alokasi yang diperbolehkan dengan standar biaya overhead pabrik yang dibebankan ke produk.

Varians ini mengidentifikasikan aktivitas yang tersedia tetapi tidak digunakan secara efisien.

Rumus :

(Anggaran Fleksibel Aktivitas Standar) - (Aktivitas standar x Tarif total)

###Metode Tiga Varians

Metode ini memperhitungkan varians pengeluaran, varians kapasitas menganggur dan varians efisiensi.

1) Varians Pengeluaran (Spending Variance), adalah varians biaya overhead pabrik aktual dengan jumlah anggaran fleksibel yang disesuaikan dengan aktivitas aktual yang diperkenankan .

Rumus :

(BOP Aktual) - (BOP tetap aktivitas normal + BOP variabel aktivitas aktual)

dimana,
BOP Tetap = Aktivitas Normal x Tarif Tetap
BOP Variabel = Aktivitas Aktual x Tarif Varabel

2) Varians Kapasitas Menganggur (Idle Capacity Variance), adalah perbedaan antara biaya overhead yang dianggarkan pada kapasitas sesungguhnya dengan biaya overhead pabrik yang dibebankan kepada produksi pada kapasitas sesungguhnya dengan tarif standar.

Rumus :

Varians Kapasitas Menganggur (F/UF) = Jumlah Anggaran berdasarkan jam kerja aktual - (Jam Kerja Aktual x Tarif overhead pabrik)

3) Varians Efisiensi (Efficiency Variance), adalah varians biaya overhead pabrik variabel dengan membandingkan overhead pabrik variabel menggunakan aktivitas aktual dengan overhead pabrik menggunakan aktivitas standar.

Varians ini mencerminkan pemakaian yang efisien dan tidak efisien untuk dijadikan dasar oleh perusahaan dalam membebankan biaya overhead pabrik.

Rumus :

Varians Efisiensi (F/UF) = (Jam kerja aktual x tarif overhead standar) - Overhead pabrik standar yang dibebankan ke produksi (jam kerja standar x total tariff overhead)
