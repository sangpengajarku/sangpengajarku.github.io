2. Apa fungsi Polygon tool di corel draw


Fungsi Polygon tool  yaitu untuk menggambar dengan bentuk persegi secara banyak, sekedar dengan melakukan atau mengaktifkan drag dan klik mouse. Memiliki turunan 4 macam tools, yaitu :

Star tool : berfungsi untuk menggambar berbagai macam bentuk bintang.

Complex Star tool : berfungsi untuk menggambar bentuk bintang yang mempunyai banyak potongan sisi.

Graph Paper tool : berfungsi  untuk menggambar bentuk rangkaian  susunan kotak-kotak seperti jala-jala.

Spiral tool : berfungsi  untuk menggambar bentuk lingkaran spiral secara simetris atau logaritmis

Pengertian poligon tool

Polygon Tool adalah sejenis tool yang  bisa dipakai untuk membuat beragam format objek pada lembar kerja diCorelDRAW, yang berbentuk sebuah polygonal. Dalam Polygon Tool ini, tersembunyi 5 macam tool yang fungsinya saling berkaitan, yaitu : Polygon, Star, Complex Star, Graph Paper, dan juga Spiral. Tombol shortcut untuk Polygon Tool ialah Y, sedangkan tombol yang dapat digunakan untuk mengakses Tool Graph Paper adalah D, serta Tombol A untuk Spiral Tool.

28. 1. Sebutkan 5 macam mode warna pada fill tool? 2. Fungsi pick tool, shape tool, polygon tool, outline tool 3. Fungsi weld, trim, intersect, back minus front


1. -Uniform fill (shift + F11)
-Fountain fill
-pattern fill
-Texture fill
-Outline pen

2. FUNGSI PICK TOOL :
1. Memindahkan posisi objek
2. Mengubah ukuran objek (memperbesar/ memperkecil)
3. Merotasi sebuah objek
4 Memiringkan sisi objek (Skew)
5. Memilih satu atau lebih sebuah objek secara bersamaan
FUNGSI SHAPE FOOL :
1. Shape, adalah sebuah tool yang berfungsi untuk merubah bentuk sebuah garis atau object.
2. Smudge.brush, sering penyebutannya dengan istilah smudge brush tool, yaitu berfungsi sebagai penghalus garis.
3.Roughen brush, yaitu berfungsi untuk membuat garis berefec kasar.
Free transform tool, yaitu berfungsi untuk memutar sebuha objec secara bebas.
FUNGSI POLYGON TOOL :
berfungsi untuk menggambar bentuk persegi banyak, hanya dengan drag dan klik mouse. 
FUNGSI OUTLINE TOOL :
dalam Outline Tool, tool tool tersebut fungsinya saling berkaitan, yakni Outline Pen, Outline Color, No Outline, Hairline Outline, Beberapa ukuran garis outline ( 0.5 pt, 0.75 pt, 1 pt, 1.5 pt, 2 pt, 3 pt, 4 pt, 8 pt, 1 pt) dan juga Colour. Tombol shortcut / jalan pintas untuk mengakses Outline Pen secara cepat adalah dengan menekan tombol F12. Sedangkan untuk mengakses Outline Color secara cepat, dapat dilakukan dengan menekan tombol kombinasi Shift+F12.

3. - Shaping adalah teknik yang digunakan untuk membentuk objek yaitu menggabungkan (Weld), memotong (Trim, Intersect, Simplify, dll) pada dua objek atau lebih yang bersinggungan.
- Weld Yang pertama ada Weld, fungsi Weld dalam CorelDRAW digunakan untuk menggabungkan 2 buah objek atau lebih menjadi satu. Berbeda dengan Group, jika Group hanya mengelompokan objek saja namun Weld menjadikan beberapa objek tersebut menjadi satu. 
- Trim berfungsi untuk memotong sebuah objek dengan objek lain yang bersinggungan atau menempel pada objek tersebut. Cara kerjanya, objek pertama yang diseleksi atau dipilih akan memotong objek yang kedua. 
- Intersect diguanakan untuk membuat atau mendapatkan objek baru dari hasil persinggungan dari dua buah objek. Jadi kalau sobat membuat dua buah objek kemudian kedua objek tersebut sobat singgungkan atau tumpuk, dengan mengklik  icon intersect maka persinggungan objek tersebut akan menghasilkan objek baru. 
- Simplify, Fasilitas ini hampir sama dengan Trim, namun bedanya pada Simplify objek yang di atas akan memotong objek yang dibawah tidak berpengaruh objek mana yang di seleksi duluan.
