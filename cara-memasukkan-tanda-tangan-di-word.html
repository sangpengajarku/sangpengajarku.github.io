15. 5 langkah memasukkan bingkai pada aplikasi word


Jawaban:

1. Klik pada tab menu "Page Layout".

2. Klik icon "Page Borders" pada grup Page Background.

3. Setelah muncul tampilan Borders and Shading, atur jenis bingkai pada bagian kiri ( Setting ) dan atur tampilan bingkai bagian tengah ( Style, Color, Width, Art ).

- Style : pengaturan bingkai yang berupa garis.

- Art : pengaturan bingkai yang berupa gambar.

4. Pada bagian preview pilih halaman yang akan diatur bingkainya.

5. Klik tombol OK untuk menerapkan bingkai pada halaman dokumen.

28. apakah ada perbedaan pada saat memasukkan table dan gambar di microsoft word?​


Jawaban:

Iya, terdapat perbedaan diantara keduanya, untuk memasukkan Tabel kita harus menggunakan menu Table pada tab Insert, sedangkan untuk memasukkan Gambar kita harus menggunakan menu Picture pada tab Insert

Penjelasan:

Pada tab Microsoft Word terutatama pada tab "Insert" terdapat ribbon yang terdiri dari beberapa grup yaitu: Pages, Table, Illustrations, Add-ins, Media, Links, Comments, Header & Footer, Text, dan Symbols. Untuk kali ini kita akan membahas memasukan tabel dan gambar

Untuk memasukan tabel pada Microsoft Word:

Klik tab Insert

pada grup "Table", klik Table

Terdapat pilihan menu tabel, berikut penjelasan:

Kotak-kotak tabel kecil: Untuk membuat tabel secara instan

Insert Table: Untuk membuat tabel dengan mengisi jumlah kolom dan baris tabel yang diinginkan

Convert Text to Table: Menu ini hanya akan tersedia apabila Anda ingin mengubah teks menjadi tabel secara instan

Excel Spreadsheet: Untuk membuat tabel dengan Microsoft Excel (Sehingga tabel yang dibuat, akan membuka aplikasi Microsoft Excel)

Quick tables: Untuk membuat tabel secara instan dari pilihan setelan yang diberikan

Untuk memasukan gambar pada Microsoft Word:

Klik tab Insert

pada grup "Illustration", klik Pictures

Terdapat pilihan menu gambar pada versi Microsoft Office yang lebih baru (Microsoft Office 2019 atau yang lebih baru), berikut penjelasan:

This Device: Untuk memilih gambar dari perangkat Anda

Stock Online: Untuk memilih gambar dari gambar yang sudah Microsoft Office berikan sendiri

Online Pictures: Untuk memilih gambar dari internet

Jawaban:

1.Buka dokumen Word. Klik dua kali pada dokumen agar terbuka.

Anda dapat membuka dokumen dengan cara lain. Buka Word (di menu Windows dalam PC atau folder Aplikasi dalam Mac), klik File, klik Open, kemudian klik ganda nama berkas.

2.Aktifkan pelabelan 'anchor'. Ini akan membantu Anda memonitor letak gambar-gambar.

Klik menu File.

Klik Options.

Klik Display.

Centang kotak di sebelah “Object anchors.”

Klik OK.

3.Masukkan gambar yang Anda inginkan.

4.Klik kanan pada gambar. Sebuah menu akan muncul.

Jika mouse Anda tidak memiliki tombol kanan, tahan tombol Ctrl sembari mengeklik menggunakan tombol kiri.

5.Klik Size and Position

6.Tentukan posisi absolut gambar.

7.Klik Text Wrapping.

8.Pilih opsi wrapping.

10.Klik OK. Anda seharusnya dapat melihat ikon jangkar di pojok kiri atas gambar. Artinya, gambar telah melekat di posisi tersebut.
