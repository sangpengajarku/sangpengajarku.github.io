Misal:
x = harga 1 buah roti coklat
y = harga 1 buah roti keju

4x + 2y = 23.000 |×4| 16x + 8y = 92.000
3x + 4y = 26.000 |×2| 6x + 8y = 52.000

Eliminasi
16x + 8y = 92.000
6x + 8y = 52.000
_______________-
10x = 40.000
x = 4.000

Substitusi
4x + 2y = 23.000
4(4.000) + 2y = 23.000
16.000 + 2y = 23.000
2y = 23.000 - 16.000
2y = 7.000
y = 3.500

Uang yang di bayar Ari
= 7x
= 7(4.000)
= 28.000

Uang kembalian yang diperoleh
= 50.000 - 28.000
= 22.000
