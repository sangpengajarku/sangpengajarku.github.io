19. Bagian puncak dari tiang pagar rumah berbentuk limas dengan alas berukuran 30cm dan 39cm dan tinggi 20cm.Luas permukaan puncak tiang pagar tersebut adalah?


Luas permukaan limas = Luas alas + Luas selubung limas
                                     = Luas alas + (1/4 x keliling alas x tinggi segitiga)
                                     = (p x l) + {1/4 x 2(p + l) x tinggi segitiga}
                                     = (39 cm x 30 cm) + {1/4 x 2(39 cm + 30 cm) x 20 cm}
                                     = (1.170 cm²) + {1/4 x 2(69 cm) x 20 cm}
                                     = (1.170 cm²) + (1/4 x 138 cm x 20 cm)
                                     = (1.170 cm²) + (1/4 x 2.760 cm²)
                                     = (1.170 cm²) + (690 cm²)
                                     = 1.860 cm²

21. pembentukan karat pada pagar besi, paku, dan tiang-tiang besi disebabkan oleh?


Karat merupakan hasil korosi, yaitu oksidasi suatu logam. Besi yang mengalami korosi membentuk karat dengan rumus Fe2O3.xH2O. Korosi atau proses pengaratan merupakan proses elektro kimia. Pada proses pengaratan, besi (Fe) bertindak sebagai pereduksi dan oksigen (O2) yang terlarut dalam air bertindak sebagai pengoksidasi. Persamaan reaksi pembentukan karat sebagai berikut: Anode: Fe(s) → Fe2+(aq) ) 2e-
Katode: O2(g) + 4H+(aq) + 4e- → 2H2O(l)
