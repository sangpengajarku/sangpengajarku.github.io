9. QUIZ3n - 2 = -1note: gak bisa buka ig di chrome ;)​


Nilai n adalah 1/3

PENJELASAN

Persamaan Linear Satu Variabel

Diketahui:

3n - 2 = -1

Ditanyakan:

Nilai n

Jawab:

3n - 2 = -1

3n = -1 + 2

3n = 1

n = 1/3

Jadi, nilai n adalah 1/3

PEMBUKTIAN

3n - 2 = -1

3(1/3) - 2 = -1

3/3 - 2 = -1

1 - 2 = -1

-1 = -1 (Benar)

3n - 2 = -1

3n = -1 + 23n = 13n/3 = 1/3n = 1/3 ✔
