18. UM UGM Ujian Masuk UGM


y = cosx - sinx
y(π/4) = cos(π/4) - sin(π/4)
= √2/2 - √2/2
= 0

y' = -(sinx + cosx)
y'(π/4) = -(sin(π/2) + cos(π/2))
= -(√2/2 + √2/2)
= -(2√2/2)
= -√2

garis melalui (π/4, 0) dengan gradien -√2

y - b = m(x - a)
y - 0 = -√2(x - π/4)
y = -√2 x + √2 π/4
y(0) = -√2 (0) + √2 π/4
= π/4 √2

24. tolong bantu jawab..jarak antara kampus A dan B adalah dua kali harak kampus C dan D. kampus E terletak di tengah antara kampus C dan D. selanjutnya diketahui bahwa jarak kampus C ke kampus A sama dengan jarak kampus D ke B, yaitu setengah jarak kampus A ke kampus B. kampus yang jaraknya paling jauh adalah?


A ke B  = 2 x ( C ke D)
E ditengah C dan D
AC = DB
Urutan:
A           C            E           D           B
Misalkan
A ke C adalah 1
maka DB = 1
CD = 2
AB = 1 + 2 + 1 = 4
Jadi jarak paling jauh adalah A ke B
