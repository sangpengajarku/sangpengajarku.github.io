19. (SPLTV)pak hamid,bu indah,dan pak rudi masing masing membeli tiga jenis pupuk. Pak hamid membelu 3kg pupuk NPK,2kg pupuk urae,1kg pupuk ZA seharga Rp10.100,00. Bu indah membeli 2kg pupuk NPK,3kg pupuk urea,dan 2kg pupuk ZA seharga Rp11.400,00. Sedangkan pak rudi membayar uang sebanyak Rp9.700,00 untuk membeli 2kg pupuk NPK,1kg pupuk urea,dan 3kg pupuk ZA. Berapakah harga tiap kg pupuk tersebut?​


Jawaban:

Pupuk urea = 1.500 per kg

Pupuk ZA = 1.300 per kg

Pupuk NPK = 2.150 per kg

Penjelasan dengan langkah-langkah:

Langkah pertama yaitu kita ibaratkan:

Pak Hamid = H

Bu indah = I

Pak Rudi = R

Pupuk NPK = x

Pupuk urea = y

Pupuk ZA = z

kemudian ganti yang diketahui dengan pengibaratan tersebut, menjadi

H = 3x + 2y + 1z = 10.100 (persamaan 1)

I = 2x + 3y + 2z = 11.400 (persamaan 2)

R = 2x + 1y + 3z = 9.700 (persamaan 3)

Eliminasi persamaan 2 dengan persamaan 3

2x + 3y + 2z = 11.400

2x + 1y + 3z = 9.700

_________________ -

2y - z = 1.700 (persamaan 4)

Eliminasi persamaan 1 dengan persamaan 3

3x + 2y + 1z = 10.100 (dikali 2)

2x + 1y + 3z = 9.700 (dikali 3)

6x + 4y + 2z = 20.200

6x + 3y + 9z = 29.100

_________________ -

y - 7y = - 8.900 (persamaan 5)

Eliminasi persamaan 4 dengan persamaan 5

2y - z = 1.700 (dikali 1)

y - 7y = - 8.900 (dikali 2)

2y - z = 1.700

2y - 14y = - 17.800

_______________ -

13y = 19.500

y = 1.500 (harga 1kg pupuk urea)

Substitusi y ke persamaan persamaan 4

2y - z = 1.700

2 (1.500) - z = 1.700

3.000 - z = 1.700

- z = -1.300

z = 1.300 (harga 1kg pupuk ZA)

Substitusi y dan z ke persamaan 2

2x + 3y + 2z = 11.400

2x + 3 (1.500) + 2.600 = 11.400

2x + 4.500 + 2.600 = 11.400

2x + 7.100 = 11.400

2x = 4300

x = 2.150 (harga 1kg pupuk NPK)
