13. Ayam betina 51 ekor lebih banyak daripada ayam jantan.perbandingan ayam betina dengan ayam jantan adalah 7:4 maka jumlah seluruhnya adalah?


Jawab:

Penjelasan dengan langkah-langkah:

perbandingan.

B : J = 7 : 4

B - J = 51

B + J = ( 7+4)/(7-4)  x 51

.        = 11/3  x 51

.        = 187

atau

B : J = 7x : 4x

B - J = 51

7x - 4x = 51

3x = 51

x = 51/3

x = 17

B + J = 7x + 4x

.        = 11x

.        = 11 ( 17 )

.        = 187

jadi jumlah ayam betina dan jantan adalah 187
