7. cara menghapus database dan tabel (pake MYSQL)


Menghapus Tabel
DROP TABLE [IF EXIST]  table_name [, table_name, ...]
keterangan : parameter [IF_EXIST] berupa opsional yang berguna menghilangkan pesan error jika tabel tidak ada.
cara pakai : mysql> Drop Table [IF EXIST]  tabel_mahasiswa;

Menghapus Database
DROP DATABASE [IF_EXIST] Database_name;
Keterangan : parameter [IF_EXIST] berupa opsional yang berguna menghilangkan pesan error jika database tidak ada.
cara pakai : mysql> DROP DATABASE [IF_EXIST] DB_Penjualan;

8. pada system operasi apakah MySQL database srver dapat di instal?​


Jawaban:

MySQL adalah Relational Database Management System (RDBMS) yang sangat populer digunakan. MySQL pertama kali diluncurkan pada tahun 1995 dan dikelola oleh perusahaan MySQL AB. Pada tahun 2008 MySQL dibeli oleh Sun Microsystems dan pada tahun 2010 Oracle Corporation membeli Sun, sehingga otomatis saat ini MySQL kini dimiliki oleh Oracle. Setelah Sun dibeli oleh Oracle, salah satu pembuat MySQL, Michael “Monty” Widenius, merasa bahwa akan muncul conflict of interest antara MySQL dengan Oracle database, sehingga Dia memutuskan untuk membuat fork dari source code MySQL dan membuat proyek baru bernama MariaDB.

Saat instalasi anda akan diminta memasukkan password untuk user root. Secara default, user root memiliki hak akses tertinggi dalam database MySQL, jadi pastikan password yang anda gunakan cukup kuat.

Setelah instalasi selesai kita dapat menjalankan sebuah perintah sederhana untuk menghilangkan beberapa konfigurasi default yang rentan menjadi vulnerability sistem anda, yaitu

$ mysql_secure_installation

Jawablah tiap pertanyaan pada yang muncul sesuai dengan tingkat keamanan yang anda inginkan. Langkah ini sebenarnya bersifat optional, namun langkah ini dapat meningkatkan keamanan server MySQL anda.

11. Tuliskan Langkah2 Pembuatan Database Mysql !  di PHPMyadmin. Nama Datase : Sekolah Jawaban tulis di kolom Komentar ....


Jawaban:

1.  Masuk ke phpmyadmin di XAMPP dengan mengetikkan alamat localhost/phpmyadmin di browser

2. Pilih menu "Databases" dan masukan pada bagian nama "Sekolah" (tanpa tanda kutip) kemudian, klik tombol "Create"

3. Masuk ke database yang sudah dibuat. Setelah itu, klik nama database tersebut kemudian klik menu “Privileges”. Lanjutkan dengan klik “Add user”.

4. Setelah itu, akan muncul menu form pembuatan user baru. Pada kolom “username”, Anda bisa mengisi username dengan nama apapun yang Anda inginkan. Kemudian isikan juga “host”, gunakan “use text field” saja, lalu isikan dengan “localhost”. Setelah itu, isikan “password” 2 kali (harus sama!). Terakhir, ceklist “grant all privileges on database".

5. Di bagian bawah, anda akan menemukan opsi “Global Privileges” yang berguna untuk memberikan hak akses kepada user untuk mengelola semua database, klik “Check All”. Setelah itu, klik tombol “Add User”.

Setelah melakukan cara tadi, maka akan terbentuk database dengan nama Sekolah. Secara default, XAMPP sudah memiliki user bernama "root" tanpa password.

15. menjelaskan teori tentang apa itu MySQL database connection


Jawaban:

MySQL adalah sebuah database management system (manajemen basis data) menggunakan perintah dasar SQL (Structured Query Language) yang cukup terkenal. Database management system (DBMS) MySQL multi pengguna dan multi alur ini sudah dipakai lebih dari 6 juta pengguna di seluruh dunia.

MySQL adalah DBMS yang open source dengan dua bentuk lisensi, yaitu Free Software (perangkat lunak bebas) dan Shareware (perangkat lunak berpemilik yang penggunaannya terbatas). Jadi MySQL adalah database server yang gratis dengan lisensi GNU General Public License (GPL) sehingga dapat Anda pakai untuk keperluan pribadi atau komersil tanpa harus membayar lisensi yang ada.

Seperti yang sudah disinggung di atas, MySQL masuk ke dalam jenis RDBMS (Relational Database Management System). Maka dari itu, istilah semacam baris, kolom, tabel, dipakai pada MySQL. Contohnya di dalam MySQL sebuah database terdapat satu atau beberapa tabel.

SQL sendiri merupakan suatu bahasa yang dipakai di dalam pengambilan data pada relational database atau database yang terstruktur. Jadi MySQL adalah database management system yang menggunakan bahasa SQL sebagai bahasa penghubung antara perangkat lunak aplikasi dengan database server.

23. MemasUkkan perintah untuk mengakses database mysql server melalui


Jawaban:

Buka command prompt atau terminal pada komputer Anda.Masuk ke direktori instalasi MySQL Server.Masuk ke folder "bin" pada direktori MySQL Server.Jalankan perintah "mysql -u [username] -p [password]".

         a. Ganti [username] dengan username MySQL Anda.

         b. Ganti [password] dengan password MySQL Anda.

Setelah Anda menjalankan perintah di atas, Anda akan diarahkan ke shell MySQL.Selanjutnya, Anda dapat menggunakan perintah SQL untuk melakukan berbagai operasi seperti membuat database, membuat tabel, mengisi data, dan sebagainya.

Penjelasan:

Berikut adalah contoh perintah untuk membuat database baru:

CREATE DATABASE nama_database;

Berikut adalah contoh perintah untuk memilih database yang sudah ada:

USE nama_database;

Berikut adalah contoh perintah untuk menampilkan tabel yang sudah ada dalam database:

SHOW TABLES;
