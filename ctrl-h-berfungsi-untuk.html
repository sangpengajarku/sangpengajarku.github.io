Jawaban:

Keyboard adalah suatu alat input yang tergolong dalam perangkat keras komputer dan memiliki bermacam-macam tombol yang memiliki fungsi yang berbeda-beda pada saat ditekan juga dapat menghasilkan suatu proses yang sesuai dengan keinginan pengguna. Keyboard merupakan hal pokok dalam menunjang kinerja komputer, karena apabila komputer tersebut tidak memiliki keyboard maka proses memasukkan data agar sangat terhambat, walaupun sebenarnya masih bisa dilakukan dengan menggunakan Keyboard Virtual.

Salah satu tombol yang ada di keyboard adalah tombol CTRL. Tombol CTRL bila digabungkan dengan tombol lain akan menghasilkan sebuah perintah. Tombol CTRL berfungsi untuk mempermudah dalam melakukan perintah pada aplikasi tertentu dalam sebuah pekerjaan.

Fungsi Control (CTRL) Pada Keyboard :

CTRL + A : Berfungsi untuk memblok seluruh teks/file

CTRL + B : Berfungsi untuk membuat teks Tebal

CTRL + C : Untuk mengcopy/menyalin teks maupun File

CTRL + D : Untuk membuka menu/kotak dialog Font

CTRL + E : Untuk rata tengah teks

CTRL + F : Untuk mencari File dan membuka menu/kotak dialog Find

CTRL + G : Untuk membuka menu/kotak dialog Go To

CTRL + H : Untuk membuka menu/kotak dialog Replace

CTRL + I : Untuk membuat teks Miring

CTRL + J : Untuk membuat teks rata kanan-kiri

CTRL + K : Untuk membuka menu/kotak dialog Insert Hyperlink

CTRL + L : Untuk membuat teks rata Kiri

CTRL + M : Untuk menggeser First Line Indent dan Left Tab pada garis dokumen

CTRL + N : Untuk membuka dokumen baru

CTRL + O : Untuk membuka menu/kotak dialog Open

CTRL + P : Untuk membuka menu/kotak dialog Print

CTRL + Q : Untuk menghilangkan nomor urut

CTRL + R : Untuk membuat teks rata Kanan

CTRL + S : Untuk menyimpan dokumen

CTRL + T : Untuk menggeser Left Tab pada mistar dokumen

CTRL + U : Untuk membuat teks garis bawah

CTRL + V : Untuk menampilkan hasil teks yang sudah di Copy

CTRL + W : Untuk menyimpan dan langsung menutup program Microsoft Office yang di aktif

CTRL + X : Untuk memotong teks

CTRL + Y : Untuk mengembalikan perintah yang telah dibatalkan

CTRL + Z : Untuk membatalkan perintah sebelumnya
