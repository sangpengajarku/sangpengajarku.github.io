3. Sebanyak 6,4 gram belerang (S) direaksikan dengan fluor (F2) menghasilkan belerang heksafluorida (SF6) dengan peramaan S + F2 => SF6 (reaksi belum setara). Hitunglah berapa gram Fluorida yang diperlukan agar belerang habis bereaksi. ( Ar S = 32 ; F = 19)​


Jawaban:

Massa F₂ yang dibutuhkan agar belerang habis bereaksi adalah 22,8 gram . Langkah penyelesaiannya adalah

Hitung mol SBuat persamaan reaksi setara Hitung Mr F₂ Hitung mol F₂ Hitung massa F₂

Penjelasan:

Langkah I : menghitung mol S

mol S = [tex]\frac{Massa S}{Ar S}[/tex]

          = [tex]\frac{6,4 gram}{32 gram/mol}[/tex]

          = 0,2 mol

Langkah II : Buat persamaan reaksi setara

S + 3 F₂ ⇒ SF₆

Langkah III : hitung Mr F₂

Mr F₂ = jumlah Fx Ar F

         = 2 x 19 gram/mol

         = 38 gram/mol

Langkah IV : Hitung mol F₂

mol F₂ = [tex]\frac{koefisien F_2}{Koefisien S}[/tex] x mol S

           = [tex]\frac{3}{1}[/tex] x 0,2 mol

           = 3 x 0,2 mol

           = 0,6 mol

Langkah V : menghitung massa F₂

massa F₂ = mol F₂ x Mr F₂

                = 0,6 mol x 38 gram/mol

                = 22,8 gram

7. Sebanyak 12,8 gram belerang dibakar dengan oksigen berlebihan menghasilkan 25,6 gram belerang oksida. Berapa gram oksigen yang bereaksi dengan belerang dan tentukan perbandingan massa belerang dan oksigen dalam belerang dioksida?


massa S = 12,8 gr , Ar S = 32
massa SO2 = 25,6 gr , Mr SO2 = 64

S + O2 => SO2

mol S = 12,8 gr / 32 gr/mol = 0,4 mol
mol SO2 = 25,6 gr / 64 gr/mol = 0,4 mol
perbandingan mol S : SO2 = 1 : 1

mol O2 = 1/1 x mol S = 1/1 x 0,4 mol = 0,4 mol
massa O2 = 0,4 mol x 32 gr/mol = 12,8 gram

perbandingan massa S : O2 =  32 : 32  = 1 : 1

25 gram

Penjelasan:

Perbandingan S : O = 2 : 3

massa S yg bereaksi = 2/3 x 15 gram => 10 gram

massa O yg bereaksi = 3/3 x 15 gram => 15 gram

massa S : O yg bereaksi = 10 gram : 15 gram ( tepat bereaksi mengikuti perbandingan 2 : 3 )

massa SO3 = massa S + massa O

massa SO3 = 10 gran + 15 gram => 25 gram
