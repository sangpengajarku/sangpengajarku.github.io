1. Apa itu simetris bilateral?


Simetri Bilateral adalah hewan yang bagian tubuhnya tersusun bersebelahan dengan bagian lainnya . Jika di ambil garis memotong dari depan ke belakang , maka akan terlihat bagian tubuh - tubuh yang sama antara kiri dan kanan . Hewan yang bersimetri bilateral selain memiliki puncak { oral } dan sisi dasar { aboral } juga memiliki sisi atas { dorsal } dan sisi bawah { ventral } sisi kepala { anterior } , dan sisi ekor { posterior } , serta di sisi samping { lateral } . Triploblastik slomata : Tidak memiliki rongga tubuh 
Triploblastik pseudoselomata : Memiliki rongga tubuh
