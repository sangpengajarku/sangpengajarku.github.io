2. cara mengaktifkan hidden ​


PEMBAHASAN

Setelah Control Panel terbuka, klik view by menjadi Category dan pilih Appearance and Personalization. Kemudian akan terbuka menu baru lagi, klik Folder Options untuk menampilkan folder yang di hidden. Untuk menampilkan folder yang di hidden, pilih Show hidden files, folder, dan drive. 8. cara menyimpan dokumen excel dengan menggunakan nama baru dan cara mengirimkan data excel melalui email


Misalkan dokumen excel 2007:
- bukalah dokumen escel yang ingin diganti dengan nama baru
- klik tombol office button dan klik tombol Save As
- isikan File Name dengan nama baru dan klik tombol Save

Misalkan mengirim data excel melalui email yahoo:
- akses situs www.ymail.com dan login dengan username serta passwordmu
- klik tombol Compose mail (tulis email baru)
- isikan To (tujuan email) dan Subject (judul email)
- klik tombol lampiran dan pilih Lampirkan file dari komputer
- saat tampil jendela pemilihan file, silakan dobel klik file excel yang ingin dikirim
- klik tombol Kirim (yg berwarna biru)

# untuk lebih jelas dalam melampirkan file di email, silakan perhatikan gambar berikut ini

24. Bagaimana cara mengatasi file yang sudah di atur hidden tetapi selalu muncul dengan sendirinya?


Pada windows, mungkin karena pengaturan Folder Option belum maksimal.
Agar file hidden benar-benar tersembunyi, coba langkah berikut ini:
1. Masuk ke dalam Windows Explorer
2. Klik menu tools
3. Klik menu Folder Options
4. Klik tab. View
5. Berikan pilihan "Don't Show hidden files, folder or drives"
berikan tanda cawang di 3 pilihan di bawahnya
6. Klik tombol OK

# langkah di atas, untuk Windows 7

27. Bagaimana cara menutup file yang ter-hidden​


Jawaban:

Windows 8,10

Di My Computer atau ketika membuka folder, klik tab "View" yand ada diatas. Lalu apabila box "Hidden files" ada tanda centangnya, klik supaya tidak bisa dilihat, klik lagi kalau ingin hidden files bisa dilihat.

Windows 7 :

Ketika membuka "My computer" atau folder apa saja, klik "organize" yang ada di atas kiri, lalu klik "folder and search options" lalu klik "don't show hidden files, folders, and drivers" klik ok.
