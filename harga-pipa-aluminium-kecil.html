Jawaban:

t = 74,83 °C

Penjelasan:

Panjang pipa baja

Lb = 40 + 0,01 = 40,01 m

Jika dipanaskan panjang pipa aluminium dan baja

La = Loa + Loa.α₁.Δt

La = 40 + 40. 23. 10⁻⁶. Δt

Lb = Lob + Lob. α₂.Δt

Lb = 40,01 + 40,01. 12. 10⁻⁶. Δt

Suhu agar selisih panjang 15 mm ketika dipanaskan

La - Lb =  40 + 40. 23. 10⁻⁶. Δt - 40,01 - 40,01. 12. 10⁻⁶. Δt

0,015 = -0,01 + ( 40. 23.10⁻⁶ - 40,01. 12.10⁻⁶ ). Δt

0,015 = -0,01 + ( 439.88 .10⁻⁶ ). Δt

0,025 = ( 439.88. 10⁻⁶ ). Δt

Δt = 0,025/( 439.88. 10⁻⁶ )

Δt = 56,83 ⁰

t - 18 = 56,83 ⁰

t = 74,83 °C

19. pipa aluminium pipa besi dan pipa tembaga memiliki panjang yang sama pada suhu t1.ketiga logam dipanaskan hingga suhu t2.pipa yg pling pnjg hingga yg terpendek pda suhu t2 adlah


Jawaban:

urutan Pipa yang memiliki panjang mula - mula sama pada suhu awalnya , ketika dipanaskan pada suhu akhir dari yang terpanjang adalah tembaga, besi dan aluminium. (Opsi B).

Penjelasan:

Diketahui :

Panjang pipa aluminium, besi dan tembaga memiliki panjang awal yang sama.

Data untuk pipa aluminium :

α = Koefisien muai panjang = 0,000025/°C

t1 = Suhu awal = 30°C

t2 = Suhu akhir 50°C

Data untuk pipa besi :

α = Koefisien muai panjang = 0,000012/°C

t1 = Suhu awal = 20°C

t2 = Suhu akhir 80°C

Data untuk pipa tembaga :

α = Koefisien muai panjang = 0,000017/°C

t1 = Suhu awal = 40°C

t2 = Suhu akhir 90°C

Ditanyakan :

Urutan pipa yang terpanjang hingga terpendek pada suhu akhir = ...

Penyelesaian :

Pertambahan panjang dirumuskan dengan :

[tex]\boxed{\triangle{L}=Lo\times\alpha\times\triangle{t} }[/tex]

ΔLa : ΔLb : ΔLt

Lo x α(a) x Δt(a) : Lo x α(b) x Δt(b) : Lo x α(t) x Δt(t)

Karena panjang awal sama bisa disederhanakan

α(a) x Δt(a) : α(b) x Δt(b) : α(t) x Δt(t)

(0,000025 x (50 - 30)) : (0,000012 x (80 - 20)) : (0,000017 x (90 - 40))

Disederhanakan lagi :

(25 x 20) : (12 x 60) : (17 x 50)

500 : 720 : 850

Jadi urutan dari yang terpanjang adalah tembaga, besi dan aluminium.
