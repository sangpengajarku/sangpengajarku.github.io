2. harga bagi 3kg nanas dan 5kg nanas ialah rm29.Beza harga antara 6kg nanas dan 4kg pisang ialah rm2. Berapakah jumlah harga, dalam rm, bagi 1kg nanas dan 1kg pisang? ​


Cara kerja

misal

a = nanas

b = pisang

bentuk matematika dalam penjelasan diatas

3a + 5b = 29

6a + 4b = 2 atau 3a + 2b = 1

ditanyakan a + b = ??

Menggunakan rumus eliminasi

3a + 5b = 29

3a + 2b = 1

____________-

3b = 28

b = 28/3

b = 9,3

harga 1 kg pisang rm 9,3

Menggunakan rumus subtitusi

3a + 2b = 1

3a + 2(9,3) = 1

3a = 1 - 18,6

a = -17,6 ÷ 3

a = -5,86 (tidak boleh negatif maka positif)

harga 1 kg nanas rm 5,86

jadi harga

= 1 kg nanas + 1 kg pisang

= 5,86 + 9,3

= rm 15,16

Terdapat kesalahan pada soal

Mapel : Matematika

Bab : Sistem persamaan

Kode : 8.2.3

Harga beli = Rp600.000

Hitung harga jual keranjang nanas dengan keuntungan 20%:

Harga jual = Harga beli + (20% x Harga beli)

= Rp600.000 + (0.2 x Rp600.000)
