18. seorang pembuat ukiran kayu,setiap harinya dapat membuat 2 jenis ukiran kayu tidak lebih dari 4.Harga bahan untuk membuat satu ukiran kayu jenis pertama sama dengan Rp.50.000 dan satu ukiran jenis ke dua sama dengan Rp.100.000.setiap harinya dia menyediakan uang sebesar Rp.300.000 untuk membeli bahan.jika harga jual ukiran kayu jenis pertama adalah Rp.65.000 dan harga jual kayu jenis kedua adalah Rp.125.000.Tentukan pendapatan maksimum yang akan di buat ukiran kayu?


Model Matematika

Misal,
ukiran 1 = x
ukuran 2 = y

x+y ≤ 4
..
50.000x+100.000y ≤ 300.000
x + 2y ≤ 6
..
Fungsi objektifnya :
f(x,y) = 65.000x + 125.000y

• x+y ≤ 4
untuk x=0, → y = 4 (0,4)
untuk y=0, → x = 4 (4,0)
• x+2y ≤ 6
untuk x=0, → y = 3 (0,3)
untuk y=0, → x = 6 (6,0)
..
x+y = 4
x+2y = 6
------------- --
-y = -2
y = 2, → x = 2 (2,2)

Subtitusi ke :
f(x,y) = 65.000x + 125.000y
f(0,4) = 65.000(0)+125.000(4) = 500.000 ★
f(4,0) = 65.000(4)+125.000(0) = 260.000
f(0,3) = 65.000(0)+125.000(3) = 375.000
f(6,0) = 65.000(6)+125.000(0) = 390.000
f(2,2) = 65.000(2)+125.000(2) = 380.000

Dapat dilihat sendiri, nilainya maksimumnya Rp500.000,-
