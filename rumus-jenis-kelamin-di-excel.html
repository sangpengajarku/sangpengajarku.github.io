5. Sebutkan jenis-jenis rumus sederhana yang dapat dilakukan Microsoft Excel​


Jawaban:

Syntax: =SUM(number1,[number2],...)

Fungsi SUM digunakan untuk menjumlahkan data-data di cell. Fungsi SUM bisa digunakan untuk menambahkan data per cell, data dalam satu range, maupun data dalam beberapa range sekaligus.

2. IF.

Syntax: =IF(logical_test,[value_if_true],[value_if_false])

Fungsi IF digunakan untuk mengembalikan atau menampilkan suatu nilai jika sebuah kondisi atau pernyataan bernilai BENAR, dan menampilkan nilai lain jika kondisi atau pernyataan bernilai SALAH. Fungsi IF juga bisa digunakan untuk banyak kondisi dengan cara menggabungkan beberapa fungsi IF sekaligus atau yang disebut dengan IF bersarang (fungsi IF di dalam IF). Fungsi IF bisa digunakan secara bersama sampai dengan 64 fungsi IF.

3. LOOKUP.

Syntax: =LOOKUP(lookup_value,array)

Fungsi LOOKUP digunakan untuk mencari sebuah nilai data pada suatu kolom maupun suatu row. Fungsi LOOKUP ini sebenaranya sudah jarang digunakan dan sudah beralih ke fungsi VLOOKUP dan HLOOKUP.

9. berjenis kelamin apakah lalat dengan rumus kromosom 3AAXXX​


Lalat yang memiliki rumus kromosom AAXXX maka, akan memiliki jenis kelamin betina super. Penentuan jenis kelamin pada lalat buah yaitu menggunakan perbandingan kromosom X dan A. Pada lalat Buah, kromosom X atau Y tidak menentukan jenis kelamin jantan ataupun jenis kelamin betina.

Penjelasan:

Penentuan jenis kelamin pada lalat buah

Kromosom X pada serangga menentukan kemampuan hidup atau viabilitas, sedangkan kromosom Y menentukan tingkat kesuburan atau fertilitas. Dalam hal ini penentuan jenis kelamin berdasarkan jumlah perimbangan antara kromosom X dengan autosom atau A.

Lalat buah dikatakan berjenis kelamin betina jika angka perimbangn kromosom X dengan A adalah 1,0. Sedangkan berkelamin jantan jika angka perimbangannya adalah 0,5.

Contoh:

Rumus= X/A, maka

AAXXX= 3/2= 1,50 sehingga menghasilkan betina super karena melebihi angka 1,0.

Lalat betina super dan jantan super memiliki kelainan dan kemunduran pada alat tubuh. Kedua lalat tersebut seringkali steril atau tidak berumur panjang. Lalat intersek merupakan lalat yang memiliki jaringan tubuh yang mosaik artinya terdapat campuran antara jaringan jantan dan betina.

10. buatlah rumus penghitungan angka ketergantungan dan rumus untuk menghitung perbandingan berdasarkan jenis kelamin!​


PENDAHULUAN Angka Beban Ketergantungan

Angka yang menunjukkan besarnya beban tanggungan bagi penduduk usia produktif terhadap penduduk usia belum atau tidak produktif.

Angka Jenis Kelamin

Merupakan perbandingan banyaknya penduduk laki-laki dan perempuan pada suatu daerah dalam jangka waktu tertentu.

PEMBAHASAN Angka Beban Ketergantungan

[tex]\tt Rumus: \frac{P(0 - 14) + P(65 + )}{P(15 - 64)} \times 100[/tex]

Keterangan:

P(0-14) = penduduk belum produktif

P(65+) = penduduk tidak produktif

P(15-64) = penduduk produktif

Angka Jenis Kelamin

[tex]\tt Rumus: \frac{M}{F} × 100[/tex]

Keterangan:

M = Jumlah laki-laki

F = Jumlah wanita

100 = konstanta

Untuk menyaring data pada Microsoft Excel dan memasukkan hanya data perempuan pada kolom jenis kelamin, berikut adalah langkah-langkah yang perlu dilakukan:

Buka dokumen Excel yang berisi data yang ingin Anda saring.

17. pengertian excel dan rumus rumusnya


excel versi pribadi : tempat pengolahan data baik berupa text dan angka. excel sendiri bisa dibilang sebagai database termudah untuk digunakan.

Rumusan Umum :

Tentang Sum
Sum() = Penjumlahan seluruh data dari sel 1 dan sel ke 2 misal : Sum(A1:A5) artinya menjumlahkan data dari sel A1 sampai A5.

Tentang IF
IF() = Logika percabangan, misal kita mau buat rumus jika cel A1 = 1 maka sel yg kita beri rumus muncul teks "Anda memasukan Angka Satu" tapi selain A1 maka "Bukan Angka Satu"

kode rumus : =IF(A1=1,"Anda memasukan Angka Satu","Bukan Angka Satu")

Tentang Count
Kalau kita mau menghitung banyaknya data yg terisi di cel cukup buat rumus dgn ini (hanya dalam penggunaan angka)

=count(A1:A9)

jika km mengisi angka di antra cel 1 sampai 9 maka akan muncul bnyaknya data yg terisi.

Tentang CountIf
Sama halnya dengan count tetapi bedanya ini menjumlahkan data berdasarkan text.

contoh di cel A1 ditulis "Sapta"
cel A2 ditulis "Dewi"
cel A3 ditulis "Sapta"

jika km menulis rumus di cel lain :
=countif(A1:A3,"Sapta") maka akan muncul angka 2.

Tentang Min dan Max
Jika kamu mau mencari data angka terkecil pada cel kamu bisa memanfaatkan =Min() dan sebaliknya =Max tapi jgn lupa isi parameter cel-nya sperti dicontoh sebelumnya.

26. 4 rumus jenis Microsoft Excel dan jelaskan fungsi nya masing-masing


1. SUM
Rumus SUM ini paling sering di gunakan, fungsi utamanya adalah untuk penjumlahan /menjumlahkan. Misal :
=SUM(B12:B25) artinya menjumlahkan data di kolom B12 sampai B25
2. AVERAGE
Fungsi untuk mengabil nilai rata-rata suatu variable, misal :
= AVERAGE(D7:E7) artinya menngambil nilai rata-rata dari D7 sampai E7.
3. AND
Fungsi Menghasilkan nilai TRUE jika semua argument yang di uji bernilai BENAR DAN FALSE jika semua atau salah satu argument bernilai SALAH.
Contoh kasus : Peserta ujian dinyatakan lulus jika nilai ujian teori dan praktik masing-masing harus di atas 7, jika kurang dari 7 maka dinyatakan gagal.
4. NOT
Fungsi not kebalikan dari fungsi AND, yakni menghasilkan TRUE jika kondisi yang di uji SALAH dan FALSE jika kondisi yang di uji BENAR

30. apa saja rumus-rumus dalam microsoft excel?
