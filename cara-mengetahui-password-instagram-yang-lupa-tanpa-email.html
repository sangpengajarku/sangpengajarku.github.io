wulandari...

[tex]2x + 3x - 15 \\ = \red{5x - 15} \\ \\ 5x - 15 = 0 \\ 5x = 15 \\ x = \frac{15}{5} \\ \red{x = 3} \\ \\ (2x + 3x - 15) \\ = (2(3) + 3(3) - 15) \\ = \red{(6 + 9 - 15)}[/tex]

sandy...

[tex](3x - 2x + 2 - 4) \\ = \red{x - 2} \\ \\ x - 2 = 0 \\ \red{x = 2} \\ \\ (3x - 2x + 2 - 4) \\ = (3(2) - 2(2) + 2 - 4) \\ = \red{(6 - 4 + 2 - 4)}[/tex]

hasil bagi...

[tex]( \frac{5x - 15}{x - 2}) = 0 \\ (5x - 15) = (x - 2)0 \\ \frac{(5x - 15)}{1} = \frac{(x - 2)}{1} 0 \\ ( \frac{5x}{1} - \frac{15}{1}) = (\frac{x}{1} - \frac{2}{1} )0 \\ ( \frac{5x}{1} - \frac{15}{1} ) \times \frac{1}{x} = ( - \frac{2}{1} )0 \\ ( \frac{5x}{x} - \frac{15}{x} ) = ( - \frac{2}{1} )0 \\ ( \frac{5}{1} - \frac{15}{x} ) = ( - \frac{2}{1} )0 \\ ( \frac{5}{1} - \frac{15}{x} ) ( - \frac{1}{2} ) = 0 \\ ( - \frac{5}{2} + \frac{15}{2x} ) = 0 \\ \frac{15}{2x} = \frac{5}{2} \\ x = \frac{15}{5} \\ \red{x = 3} \\ \\ \frac{5x - 15}{x - 2} = \frac{5(3) - 15}{(3) - 2} \\ = \frac{15 - 15}{3 - 2} = \frac{0}{1} \\ = \red{0}[/tex]

27. Bagaimana cara mengetahui password router netis jika lupa sandi, tanpa merestart router?


Cara mengetahui password router netis jika lupa sandi, tanpa merestart router adalah dengan membaca buku manual dari router tersebut. Selain itu, kita bisa mencari pada internet untuk pasword default dari router netis. Router bisa berupa perangkat jaringan komputer atau berupa sistem operasi seperti mikrotik.

Pembahasan

Router merupakan salah satu perangkat jaringan komputer. Router berfungsi untuk menghubungkan jaringan komputer yang satu dengan jaringan komputer lainnya. Router biasanya dilengkapi user dan password untuk mengamankan pengaturan dari router tersebut. Beberapa password default dari router netis adalah sebagai berikut

user admin password adminuser guest password guestuser netis password passworduser unknown password unknown

Dengan mengetahui user dan password default dari sebuah router kita bisa mengakses router dengan mudah. Oleh karena itu untuk seorang admin jaringan hendaknya merubah user dan password routernya sehingga tidak menggunakan password default lagi agar mudah diakses oleh orang lain.

Pelajari lebih lanjut

Demikian pembahasan mengenai cara untuk mengetahui password router netis, kita bisa membacanya pada buku manual atau mencarinya di internet. Untuk mempelajari materi lebih lanjut mengenai perangkat jaringan komputer lainnya dapat dibaca pada link berikut
