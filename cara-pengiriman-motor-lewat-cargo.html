9. 7. Sebuah jasa pengiriman akan mengirimkan paket Cargo dan paket Regular dari kota A ke kota B, jumlah paket Regular lebih banyak 50 dari paket Cargo. Jika jumlah paket seluruhnya adalah 70 paket. Tentukan jumlah paket Cargo yang di kirim dari kota A ke kota B!​


Jawaban:

paket Regular = x

paket Cargo = y

( I ) y - x = 50

( II ) y + x = 70

x - y = 50

x + y = 70

0 - 2y = -20

- 2y = -20

y = -20/-2

y = 10

Jadi, jumlah paket Cargo yang dikirim dari Kota A ke Kota B adalah 10 paket.
