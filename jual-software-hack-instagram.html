6. Kuis ⁽⁴⁰⁾ Banyaknya uang yang didapat perusahaan software sebanding dengan banyaknya copy software yg dijual. Perusahaan tersebut mendapat Rp. 2.800.000,- setelah menjual 8 copy software. Banyaknya uang yang didapat perusahaan dari penjualan 38 copy software Rp. _________.


Untuk mencari banyaknya uang yang didapat perusahaan dari penjualan 38 copy software, kita dapat menggunakan rumus:

(banyaknya copy software yang dijual) * (harga per copy software) = total uang yang didapat

Kita dapat mencari harga per copy software dengan menggunakan informasi yang sudah ada, yaitu:

(banyaknya copy software yang dijual) * (harga per copy software) = total uang yang didapat

8 copy software * harga per copy software = Rp. 2.800.000,-

harga per copy software = Rp. 2.800.000,- / 8 copy software = Rp. 350.000,-

Sekarang kita dapat menggunakan harga per copy software tersebut untuk mencari total uang yang didapat dari penjualan 38 copy software:

(banyaknya copy software yang dijual) * (harga per copy software) = total uang yang didapat

38 copy software * Rp. 350.000,- = total uang yang didapat

total uang yang didapat = 38 copy software * Rp. 350.000,- = Rp. 13.300.000,-

Jadi, banyaknya uang yang didapat perusahaan dari penjualan 38 copy software adalah Rp. 13.300.000,-.
