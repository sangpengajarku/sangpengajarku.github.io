Soto ayam 40 porsi dan soto daging 40 porsiBagaimana cara menyelesaikan soal tersebut dengan menggunakan program linier?Terima kasih sebelumnya


kendala :
x + y ≤ 80
x ≥ 20
y ≤ 60
fungsi tujuan = 5000x + 6000y
gambar daerah solusi yg memenuhi ketiga sistem di atas, diperoleh 3 titik vertex :
A(20, 0), B(20,60) dan C(80,0)
untuk mendapatkan penghasilan maksimum, bandingkan nilai tujuan berikut :
A : 5000(20) + 6000(0) = 100.000 (minimum)
B:  5000(20) + 6000(60) = 100.000 + 360.000 = 460.000 (maksimum)
C: 5000(80) + 6000(0) = 400.000

jadi, akan mendapatkan keuntungan maksimal jika menjual soto ayam 20 porsi dan daging sebnyak 60 porsi.
