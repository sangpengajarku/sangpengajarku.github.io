8. Cara membuat background di html


Jawaban:

Untuk membuat background di HTML, Anda bisa menggunakan atribut "style" pada tag "body". Berikut adalah cara membuat background berwarna sederhana:

Penjelasan:

Buka file HTML Anda menggunakan editor teks seperti Notepad.Temukan tag "body" di halaman Anda.Tambahkan atribut "style" pada tag "body" seperti ini: <body style="background-color: #000000;">.Ganti warna "#000000" dengan warna yang Anda inginkan. Anda bisa menggunakan kode warna HTML atau nama warna seperti "red" atau "green".Simpan file HTML Anda dan buka dengan browser web untuk melihat perubahan.

Untuk membuat background dengan gambar, Anda bisa menambahkan atribut "background-image" ke tag "body". Berikut adalah contoh:

*css

<body style="background-image: url('background.jpg');">

!Catatan: Pastikan file gambar berada pada lokasi yang benar dan memiliki format yang dapat diterima oleh browser web seperti JPEG, PNG, atau GIF.

Untuk membuat background pada HTML, gunakan atribut "style" dan "background-color" pada tag "body". Ini adalah contoh sederhana:

<!-- Isi halaman HTML Anda di sini -->

</body>

28. Cara Membuat Form Login Dan Register Dengan HTML, PHP


Untuk HTML, silahkan buat tag <form>.
Di dalam tag <form> tersebut, nanti ada tag lain yang berfungsi mendapat data dari user, yaitu <input>. <input> ini punya attribute type="" yang bisa menentukan apa yang diisi oleh user. Misal, <input type="password"> akan membuat apa yanh diketik user berubah jadi bulatan² seperti password pada umumnya. <input type="date"> akan mengisi tanggal. Jika ingin lebih tahu lagi, ada banyak attribute <input> yang lainnya silahkan lihat W3Schools.

Untuk PHPnya, nanti kamu harus mengintegrasikan HTML tag <form> dengan PHPmu. Yaitu dengan attribute action="namafilephp.php". File php ini berisi fungsi yang bisa membuat inputan user disimpan ke database, yaitu MySQL. Silahkan cari sumber lain lagi untuk lebih lengkapnya, karena kode PHP tiap orang beda².
