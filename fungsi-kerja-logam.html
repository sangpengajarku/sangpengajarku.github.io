1. Apakah fungsi kerja logam dengan energi ambang sama


Iya sama, fungsi kerja logam (atau energi ambang) menggambarkan seberapa energi minimum yang diperlukan agar elektron tetap berada/menempel di dalam logam tersebut.

Fungsi kerja logam/energi ambang dirumuskan secara matematis melalui rumus berikut.

[tex] {W}_{o} = h f_{o}[/tex]


2. Suatu logam tertentu mempunyai fungsi kerja 3,7 eV. Maka besar frekuensi ambang logam adalah


Frekuensi ambang logam tersebut adalah 8,9 × 10¹⁴ Hz.

Diketahui

Wo = 3,7 eV1 eV = 1,6 × 10⁻¹⁹ Jh = 6,626 × 10⁻³⁴ J.s

Ditanya

Frekuensi ambang (fo)

JAWAB

[tex] \boxed{fo = \frac{Wo}{h}} [/tex]

[tex]fo = \frac{3,7(1,6 \times {10}^{ - 19}) }{6.626 \times {10}^{ - 34} } [/tex]

[tex]fo = \frac{5,92 \times {10}^{ - 19} }{6,626 \times {10}^{ - 34} } [/tex]

[tex]fo = 8,9345 \times {10}^{14} \: Hz[/tex]

[tex]fo = 8,9 \times {10}^{14} \: Hz[/tex]

Kesimpulan

Berdasarkan pembahasan di atas, frekuensi ambang logam tersebut adalah 8,9 × 10¹⁴ Hz.

4. Panjang gelombang ambang bagi efek fotolistrik dalam suatu logam tertentu adalah 259 nm. a. Berapakah fungsi kerja logam tersebut? b. Apakah efek fotolistrik akan teramati untuk landa>259 nm ataukah untuk landa​


Jawaban:

[tex]W = 7,765 \times 10^{-19} Joule[/tex]

Efek foto listrik akan teramati untuk lambda kurang dari 256 nm

Penjelasan:

Rumus umum efek fotolistrik Einstein :

[tex]E_{k} = hf - hf_{o}[/tex]

Fungsi kerja Logam :

[tex]W = hf_{o} \\W = \frac{hc}{\lambda}[/tex]

Konstanta planck : 6,626 x 10^(-34) J.s

Kecepatan cahaya : 3 x 10^8 m/s

Panjang gelombang ambang : 256 nm = 256 × 10^(-9) m

Maka :

W = (6,626 × 10^(-34) J.s) × (3 × 10^8 m/s) / (2,56 × 10^(-7) m)

W = 7.765 x 10^(−19) J

Untuk keteramatan efek foto listrik maka :

[tex]hf > hf_{o} \\[/tex] , frekuensi cahaya harus diatas frekuensi ambang

Karena : lambda berbandi terbalik dengan frekuensi, maka :

Efek foto listrik akan teramati jika lambda kurang dari lambda fungsi kerja logam yaitu < 256 nm

7. cahaya ultraviolet yang dipergunakan untuk menyinari permukaan logam memiliki panjang gelombang 2000 A. elektron yang terlepas dari permukaan logam memiliki energi kinetik sebesar 5.945 eV. fungsi kerja dari logam tersebut adalah​


Jawaban:

dik:

panjang gelombang (λ) = 2000 A = 2.000 x 10⁻¹⁰ m = 2 x 10⁻⁷ m

energi kinetik (Ek) = 5,945 eV

konstanta Planck (h) = 6,6 x 10⁻³⁴ Js

cepat rambat cahaya (c) = 3 x 10⁸ m/s

dit:

fungsi kerja (W₀)....?

jawab:

E = W₀ + Ek

hc / λ = W₀ + Ek

(6,6 x 10⁻³⁴ Js x 3 x 10⁸ m/s) / (2 x 10⁻⁷

m) = W₀ + 5,945 eV

9,9 x 10⁻³⁴⁺⁸⁻⁽⁻⁷⁾ J= W₀ + 5,945 eV

9,9 x 10⁻¹⁹ J = W₀ + 5,945 eV

ubah satuan Joule ke eV

1 eV = 1,6 x 10⁻¹⁹J

(9,9 x 10⁻¹⁹ J) / (1,6 x 10⁻¹⁹ J) = W₀ + 5,945 eV

6,1875 eV = W₀ + 5,945 eV

[tex]\boxed{W_{0}=hf_{0}}[/tex]

Persamaan energi sinar adalah :

[tex]\boxed{E=hf}[/tex]

   Konversi satuan

Konversi satuan fungsi energi ke dalam satuan Joule :

E = 1,8 eV = 1,8 x 1,6 x 10⁻¹⁹ J = 2,88 x 10⁻¹⁹ J


   Frekuensi

Frekuensi gelombang tersebut bisa dicari dengan menggunakan persamaan :

E = h f

atau

f = E / h

dimana konstanta plang (h) = 6,63 x 10⁻³⁴ J.s, maka :

f = 2,88 x 10⁻¹⁹ J / 6,63 x 10⁻³⁴ Js

f = 0,434 x 10¹⁵ Hz

Jadi frekuensinya sebesar 0,434 x 10¹⁵ Hz.


   Panjang gelombang

Besar panjang gelombang bisa dicari menggunakan persamaan :

c = λ f

atau

λ = c / f

diaman c = 3 x 10⁸ m/s, maka :

λ = 3 x 10⁸ m/s / 0,434 x 10¹⁵ Hz

λ = 6,91 x 10⁻⁷ m

atau

λ = 691 nm

Jadi panjang gelombangnya sebesar 691 nm.


[tex]\boxed{Kesimpulan}[/tex]

Besar Panjang gelombang maksimum cahaya yang dapat dijatuhkan pada permukaan logam cesium tanpa mengeluarkan elektron dari permukaan logam adalah 691 nm
