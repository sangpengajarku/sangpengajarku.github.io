1. rumus matematika mencari perbandingan umur


Rumus-rumus mencari perbandingan umur, dapat disimak pembahasannya.

Pembahasan

Dalam melakukan perbandingan ada syarat yang harus kita perhatikan, apa syarat dari perbandingan?

Syarat perbandingan adalah:

1) Samakan satuan

2) Buat dalam bentuk paling sederhana

Samakan satuan maksudnya, jika membandingkan sebuah panjang maka satuan panjangnya harus sama, jangan cm dibandingkan dengan m.

Rumus-rumus dalam perbandingan:

Misal perbandingan A dan B adalah m : n

m berarti rasio untuk A, dan n adalah rasio untuk B

Jumlah perbandingan (jumlah rasio) = m + n

Selisih perbandingan (selisih rasio) = m - n dengan m > n

Jumlah A dan B adalah K

Selisih A dan B adalah L

Maka berlaku rumus ↓

A = [tex]\frac{m}{m+n}[/tex] x K

B = [tex]\frac{n}{m+n}[/tex] x K

A = [tex]\frac{m}{m-n}[/tex] x L

B = [tex]\frac{n}{m-n}[/tex] x L

A = [tex]\frac{m}{n}[/tex] x B

B = [tex]\frac{n}{m}[/tex] x A

jika a = x

a = 6 pkerja

b = 8 pekerja x = 300.000

a = x

b = b / a * x

6 pekerja = 300. 000

8 pekerja = 8 / 6 * 300.000

= 400.000

Jadi, jumlah uang yang harus dikeluarkan

sebesar Rp 400.000 ,00

jika a = x

maka b = a / b * x

a = 5 pekerja

b = 8 pekerja

x = 20. 000

a = x

b = b / a * x

5 pekerja = 20. 000

8 pekerja = 8 / 5 * 20.000

= 32.000

Jadi, harga apel nya adalah Rp 32.000,00
