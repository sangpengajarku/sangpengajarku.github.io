3. tinggi sebuah kamar mandi 2 meter, lantai sebuah kamar mandi berbentuk persegi dengan setiap sisi kamar mandi 1,5 meter, ukuran pintu kamar mandi 1×2 meter. berapa kebutuhan keramik lantai dan dinding kamar mandi?​


Jawaban:

keramik 20cm x 20cm = 2 duz.

Untuk keramik 20cm x 25cm = 8 duz.

keramik 20cm x 10cm = 1 duz.

Penjelasan dengan langkah-langkah:

keramik 20cm x 20cm = 2 duz.

Untuk keramik 20cm x 25cm = 8 duz.

keramik 20cm x 10cm = 1 duz.

jumlah keliling list keramik = 1,35m+1,35m+1,35m+0,65m = 4,7m.

tinggi pasangan list keramik = 10cm = 0,1m.

luas pasangan list keramik = 4,7 m x 0,1m= 0,47 m.

faktor keamanan 3% = 3/100 x 0,47 m2 = 0,0141m2.

Total kebutuhan keramik = 0,47m2 + 0,0141 m2 = 0,4841 m2 = 1 duz keramik ukuran 20cm x 10cm.

Karena bentuk lantainya persegi empat maka untuk menghitung luasnya kita gunakan rumus panjang x lebar.

Ukuran lantai panjang 1,35m lebar 1,35m

Luas pasangan keramik = 1,35m x 1,35m = 1,8225 m2.

faktor keamanan 3% = 3/100 x 1,8225 m2 = 0,054675 m2.

Total kebutuhan keramik = 1,877175 m2 = 2 duz keramik ukuran 20cm x 20cm. Caranya kita hitung dulu jumlah keliling dinding yang akan dipasang keramik, kemudian kita kalikan dengan tinggi pemasangan.

Jumlah keliling dinding yang dipasang keramik = 1,35+1,35+1,35+0,65 = 4,7m. angka 0,65 berasal dari panjang dinding 1,35m dikurangi lebar pintu 0,7m.

keramik 20cm x 25cm dipasang setinggi 1,5m.

jadi luas pasangan keramik = 4,7m x 1,5m = 7,05 m2.

faktor keamanan 3% = 3/100 x 7,05 m2 = 0,2115m2.

Total kebutuhan keramik = 7,05m2 + 0,2115 m2 = 7,2615 m2 = 8 duz keramik ukuran 20cm x 25cm.
