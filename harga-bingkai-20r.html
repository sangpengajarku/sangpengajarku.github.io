4. tentukan nilai bilangan kuantum unsur unsur berikut 17p 6Q 20R​


jawaban:

17p: n = 3, l= 1, m= 0, s= -1/2

6Q: n= 2, l= 1, m= 0, s= +1/2

20R: n=4, l= 0, m= 0, s= -1/2

penjelasan:

17p: 1s2 2s2 2p2 3s2 3p5

hanya diambil elektron terakhir yaitu 3p5

n= 3 ( dari kulit terluarnya yaitu 3 pada 3p5)

l = 1 ( bil.kuantum l untuk s=0, p=1, d=2, f=3)

m=0 ( karena elektron pada orbital berakhir pada kotak tengah yang mana nilainya adalah 0

n= -1/2 ( elektron mengarah kebawah dalam diagram orbital)

6Q= 1s2 2s2 2p2

n=2 ( diambil dari kulit terluar dari elektron terkahir yaitu 2 pada 2p2)

l=1 ( krna berakhir pada sub kulit p)

m=0 ( karna berakhir di kotak kedua pada diagram orbital)

s=+1/2 ( karna elektronnya mengarah keatas)

20R= 1s2 2s2 2p6 3s2 3p6 4s2

n= 4 ( diambil dari kulit terluar yaitu 4 apda 4s2)

l = 0 ( karna berakhir pada sub kulit s)

m=0 ( pada sub kulit s hanya terdapat 1 kotak dengan nilai 0)

s= -1/2 ( elektron mengarah ke bawah)

= 6(1,2 + 0,7)

= 6(1,9)

= 11,4 m

Harga Bahan

= 11,4 x Rp. 30.000

= Rp. 342.000
