2. Bagaimana cara menggabungkan gambar Clip Art ke dokumen Miscrosoft Word?


   Sisipkan gambar terlebih dahulu dalam dokumen anda.

   Kemudian buat agar layout setiap gambar dalam mode In front Of Text atau Behind Text. Caranya: klik kanan pada gambar, lalu pada popup menu yang muncul pilih Text Wrapping, kemudian klik item In Front of Text atau Behind Text. Lakukan untuk semua gambar yang akan anda satukan.

   Bila Gambar anda terlalu jauh jaraknya, sebaiknya anda dekatkan terlebih dahulu

   Selanjutnya atur posisi dan penempatannya sesuai dengan keinginan anda.

11. Posisi gutter pada sebuah dokumen di Ms. word hanya terdapat pada bagian ....... dan .......kertas / dokumen


Posisi gutter terdapat pada bagian page setup dan berada di batas kanan, kiri dan atas bawah

Penjelasan:

Gutter adalah salah satu menu pada aplikasi pengolah kata seperti MS Word, digunakan untuk mengatur batas atas, bawah, kiri dan kanan menggunakan penggaris virtual yang terletak di bagian atas dan kiri  dokumen. Fitur ini  kurang lebih berfungsi seperti mengatur margin di aplikasi yang sama. Perbedaannya, bagaimanapun, adalah bahwa parameter margin selalu membutuhkan input angka tertentu, sedangkan parameter gutter mengharuskan pengguna untuk memindahkan batas virtual secara manual atau tidak memasukkan angka tertentu. Fungsi talang adalah untuk memberikan ruang kosong yang dapat digunakan untuk mengikat setiap sisi dokumen dan meningkatkan estetika dan kerapian karya.

20. langkah2 menyimpan dokumen word


*selesaikan dlu file/dokumen yg tlah dibuat
*klik icon save pada ujung layar
*tulis nama file/dokumen tersebut
*lalu klik ok[jawaban]

Mata Pelajaran: Teknologi Informasi
Kata kunci: Microsoft Word

• Jawaban pendek:

Langkah2 menyimpan dokumen word

1. Setelah dokumen selesai dibuat, Klik menu file

2. Klik Save

3. Ketika dialog Save File muncul, pilih tempat folder penyimpanan dokumen

4. Atur nama file pada kolom File Name

5. Klik Save, dan dokumen telah tersimpan pada tempat folder yang telah ditentukan tadi.

• Penjelasan:

Untuk menyimpan dokumen, dapat juga menggunakan Shortcut Ctrl + S

Dan jika ingin menyimpan dokumen dengan nama, format lain, atau tempat penyimpanan berbeda, gunakan File > Save As atau dengan shortcut Ctrl + Shift + S
