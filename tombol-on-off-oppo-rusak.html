mapel : ti

kata kunci : wifi, switch
kode soal : 11
kode kategorisasi : 12.11

pembahasan :
Q : bagaimana cara mengatasi tombol switch wifi tak bekerja?

menurut saya, ada penyebab tertentu mengapa tombol switchmu tidak bekerja. penyebabnya bisa saja :
1. tombol di toolbar freeze
2. belum instal driver untuk kartu jaringannya (mungkin karena habis diinstal ulang)

cara mengatasinya :
1. a. coba buka settings => change pc settings => network => airplane mode => wifi => geser tombolnya
b. ubah konfigurasi di gpedit.msc

2. a. instal driver untuk laptop dengan driverpack
b. jika manufaktur laptop menyediakan cd dari box laptop, itu adalah drivernya *biasanya sih -_- coba instal lewat cd yang tersedia itu
