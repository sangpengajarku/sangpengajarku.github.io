Jawaban:

jadi, harga untuk satu baju adalah Rp. 80.000,00 dan harga untuk satu celana adalah Rp. 120.000,00

Penjelasan dengan langkah-langkah:

perumpamaan :

baju= x

celana = y

3x + 5y = 840.000 | x2| 6x + 10y = 1.680.000

2x + 3y = 520.000 | x3| 6x + 9y = 1.560.000

------------------------------- -

y = 120.000

2x + 3y = 520.000

2x + 3(120.000) = 520.000

2x + 360.000 = 520.000

2x = 520.000 - 360.000

2x = 160.000

x = 160.000 / 2

x = 80.000

25. Harga 2 celana panjang dan 5 kemeja adalah Rp 410.000 sedangkan harga 3 celana panjang dan 2 kemeja adalah Rp 340.000. Harga sebuah celana panjang dan dua kemeja adalah jawab dengan caranya yaa​


Jawaban:

Rp. 180.000

Penjelasan dengan langkah-langkah:

kita pakai permisalan

celana panjang = x

kemeja = y

maka didapat persamaan

2x + 5y = 410.000 | 3 | 6x + 15y = 1.230.000

3x + 2y = 340.000 | 2 | 6x + 4y = 680.000

11y = 550.000

y = 50.0000

jadi harga kemeja adalah Rp. 50.000

sekarang cari harga celana panjang. tinggal di substitusi kan nilai y ke persamaan

3x + 2y = 340.000

3x + 2(50.000) = 340.000

3x + 100.000 = 340.000

3x = 340.000 - 100.000

3x = 240.000

x = 80.000

dan kita dapat harga celana panjang adalah Rp. 80.000

jadi harga sebuah celana panjang dan 2 kemeja

=80.000 + 2(50.000)

=80.000 + 100.000

=Rp. 180.000

semoga membantuuuu

SEMOGA BENER YA

Semisal su celana itu X dan kemeja itu b

2x + 5b | ×3 | 6x + 15b = 1.230.000

3x + 2b | ×2 | 6x + 4b = 680.000

11b = 550.000

b = 55.000

b = 50.000

3x + 2b = 340.000

3x + 100.000 = 340.000

3x = 340.000 - 100.000

3x = 240.000

x = 80.000

Harga sebuah celana dan 2 kemeja

80.000 + 2 (50.000)

80.000 + 100.000

180.000
