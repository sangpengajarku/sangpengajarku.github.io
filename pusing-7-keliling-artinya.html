6. bantuin aku yang lagi pusing 77 keliling yaa...


Bab Permutasi Kombinasi
Matematika SMA Kelas X

2a] nP5 = 20 . nP3
      n ! / (n - 5) ! = 20 . n ! / (n - 3) !

**coret n !, lalu dikali silang **

(n - 3) ! = 20 . (n - 5) !
(n - 3) x (n - 4) x (n - 5) ! = 20 x (n - 5) !

** coret (n - 5) ! **

(n - 3) x (n - 4) = 20
n² - 7n + 12 - 20 = 0
n² - 7n - 8 = 0
(n + 1) (n - 8) = 0
n + 1 = 0
n = -1 (tidak memenuhi karena negatif)

n - 8 = 0
n = 8 (memenuhi karena positif)

jadi, n = 8

*******************************************************************

2b] P (n, 2) = 56
      nP2 = 56
      n ! / (n - 2) ! = 56
      n  x (n - 1) x (n - 2) ! / (n - 2) ! = 56

** coret (n - 2) ! **

     n x (n - 1) = 56
     n² - n - 56 = 0
     (n + 7) (n - 8) = 0
     n + 7 = 0
     n = -7 (tidak memenuhi karena negatif)

     n - 8 = 0
     n = 8 (memenuhi karena positif)

jadi, n = 8

23. Pertanyaan Ini Pasti Membuatmu Pusing 7 Keliling ( Kecuali Yang Pintar ) Ubahlah Angka Di Bawah Ini Menjadi Berpangkat -15.625 = ???


Jawab:

[tex]-25^{3}[/tex]

Penjelasan dengan langkah-langkah:

Cara mudahnya, ingat pangkat 2 dari bilangan 1-25 (yang paling sering dipakai)

Dari bilangan 1-25 yang paling memungkinkan adalah bilangan [tex]25^{2}[/tex] yang hasilnya 625, angka pada soal dan hasil perpangkatan tersebut sama, maka bisa dikalikan lagi hasil tersebut dengan bilangan awal.

= 25 x 25 x 25

= 15.625

Karena pada soal bilangannya negatif, maka hasilnya [tex]-25^{3}[/tex]

bila dibuktikan,

(-25) x (-25) x (-25) = -15.625

Jadi, -15.625 = [tex]-25^{3}[/tex]
