4. Kelebihan dan kelemahan instal aplikasi melalui media USB dan CD


Untuk kelebihan atau kelemahannya ITU tergantung, Dari Media yang kita Gunakan(CD/Flashdisk).

Kelebihan Install via Flashdisk:
1. I/O(Input dan Output) pada saat menginstall via Flashdisk lebih cepat daripada menggunakan CD. Kecepatan I/O data di Flashdisk bisa s/d 12MbPs.
2. Ukuran file bisa di-isikan lebih besar(di Flashdisk), daripada menggunakan CD yang hanya kapasitasnya sampai ±700MB.

Kelemahan Install via Flashdisk:
1. Jika kecepatan I/O datanya kecil/lemah, akan sangat lama menginstall Software-nya melalui Flashdisk. Lebih baik menggunakan CD
2. Beberapa Aplikasi/Software mungkin tidak mendukung Instalasi via Flashdisk, dikarenakan Software itu membutuhkan Ruang yang lebih lega(dari Flashdisk)


Kelebihan Install via CD(Compact Disc)
1. Kecepatan-nya saat meng-install, tidak jauh berbeda I/O data dari CD(tergantung Spec. pada CD-nya). Jika Spec. pada CD-nya lebih bagus daripada Flashdisk, bisa jadi Kecepatan menginstall via CD lebih cepat daripada menggunakan Flashdisk
2. Bisa memuat file software/aplikasi s/d ±700MB(itupun hanya memuat Software2 yang Size-nya kecil



Kelemahan Install via CD(Compact Disc)
1. Jika CD-nya lecet pada lapisan penyimpanan datanya, bisa jadi CD itu tidak dapat terbaca oleh "Eye" pada CD-ROM

2. Penyimpanannya yang hanya sampai ±700MB. Tidak seperti Flashdisk, yang penyimpanannya sampai GigaByte
