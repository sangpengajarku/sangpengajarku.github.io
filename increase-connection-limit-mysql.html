1. menjelaskan teori tentang apa itu MySQL database connection


Jawaban:

MySQL adalah sebuah database management system (manajemen basis data) menggunakan perintah dasar SQL (Structured Query Language) yang cukup terkenal. Database management system (DBMS) MySQL multi pengguna dan multi alur ini sudah dipakai lebih dari 6 juta pengguna di seluruh dunia.

MySQL adalah DBMS yang open source dengan dua bentuk lisensi, yaitu Free Software (perangkat lunak bebas) dan Shareware (perangkat lunak berpemilik yang penggunaannya terbatas). Jadi MySQL adalah database server yang gratis dengan lisensi GNU General Public License (GPL) sehingga dapat Anda pakai untuk keperluan pribadi atau komersil tanpa harus membayar lisensi yang ada.

Seperti yang sudah disinggung di atas, MySQL masuk ke dalam jenis RDBMS (Relational Database Management System). Maka dari itu, istilah semacam baris, kolom, tabel, dipakai pada MySQL. Contohnya di dalam MySQL sebuah database terdapat satu atau beberapa tabel.

SQL sendiri merupakan suatu bahasa yang dipakai di dalam pengambilan data pada relational database atau database yang terstruktur. Jadi MySQL adalah database management system yang menggunakan bahasa SQL sebagai bahasa penghubung antara perangkat lunak aplikasi dengan database server.

4. Bagaimana user melakukan pengaksesan dalam MySQL?


Dengan melalui Command promt untuk membuka mysql dan juga dapat melalui Browser untuk membuka mysql yaitu dengan mengetikkan localhost/phpmyadmin pada address Bar

Pembhasan:

MySQL adalah sebuah DBMS (database management system) yang menggunakan perintah SQL (Structured Query Language) yang banyak digunakan saat ini untuk membangun aplikasi berbasis web. MySQL dibagi menjadi dua lisensi, yang pertama adalah perangkat lunak bebas yang memungkinkan siapa saja untuk mengakses perangkat lunak. Yang kedua adalah shareware, perangkat lunak berpemilik dengan batasan penggunaan.

MySQL adalah bagian dari RDBMS (Relational Database Management System). Jadi gunakan tabel, kolom, dan baris dalam struktur database Anda. Jadi kami sedang dalam proses pengumpulan data menggunakan metode database relasional. Juga, koneksi antara perangkat lunak dan server database.

Fungsi

Pada tingkat tinggi, fungsi MySQL adalah menggunakan bahasa SQL untuk membuat dan mengelola database yang menyimpan berbagai informasi di sisi server. Fungsi lainnya adalah memberikan kemudahan bagi pengguna untuk mengakses data yang berisi informasi dalam bentuk string (teks) yang dapat diakses secara pribadi atau publik di Internet.

Hampir semua vendor web server atau web host menawarkan fungsionalitas MySQL untuk mengembangkan aplikasi berbasis web yang dikelola oleh pengembang web. Kedua, antarmuka MySQL adalah PHPMyAdmin. Hal ini untuk menghubungkan bahasa pemrograman PHP ke MySQL untuk proses manajemen database di web.

21. Connection oriented protocolnya apa ? dan connection less protocolnya apa ?


Connection-Oriented Protocol (COP) adalah protokol jaringan yang digunakan untuk membuat sesi komunikasi data di mana perangkat endpoint menggunakan protokol awal untuk membangun koneksi ujung ke ujung dan kemudian aliran data berikutnya disampaikan dalam mode transfer sekuensial.

Protokol tanpa koneksi (connectionless protocol) adalah bentuk transmisi data di mana sinyal IT keluar secara otomatis tanpa menentukan apakah penerima siap, atau bahkan apakah ada penerima.
