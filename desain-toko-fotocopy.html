18. Rata-rata gaji karyawan di Tokofotocopy "Sinar Karya" adalahRp3.225.000. Terdapat 2 bagianpekerjaan di toko tersebut, yaitubagian foto studio dan bagianpenjualan alat tulis sertafotocopy. Sebanyak 3 karyawanbagian foto studio digajiRp3.600.000, sedangkankaryawan lain yang bekerja dibagian penjualan alat tulis sertafotocopy digaji Rp3.000.000.Jumlah karyawan yang bekerja dibagian penjualan alat tulis sertafotocopy ada...orang tolong kasih tahu cara sama jawaban​


R = Rp 3.225.000,00

nFS = 3 karyawan

gFS = Rp 3.600.000,00

gPF = Rp 3.000.000,00

R = [(nFS × gFS) + (nPF × gPF)]/(nFS + nPF)

3.225.000 = [(3 × 3.600.000) + (nPF × 3.000.000)]/(3 + nPF)

3.225.000 = (10.800.000 + 3.000.000nPF)/(3 + nPF)

3.225.000 × (3 + nPF) = 10.800.000 + 3.000.000nPF

9.675.000 + 3.225.000nPF = 10.800.000 + 3.000.000nPF

3.225.000nPF - 3.000.000nPF = 10.800.000 - 9.675.000

225.000nPF = 1.125.000

nPF = 1.125.000/225.000

nPF = 5

jadi jumlah karyawan yang bekerja di bagian penjualan alat tulis serta fotocopy ada 5 orang

keterangan :

R = rata-rata gaji karyawan

nFS = banyak karyawan bagian foto studio

gFS = gaji karyawan bagian foto studio

gPF = gaji karyawan bagian penjualan alat tulis serta fotocopy

nPF = banyak karyawan bagian penjualan alat tulis serta fotocopy
