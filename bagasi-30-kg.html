1. Suatu pesawat udara dapa mengangkut penumpang tidak lebih dari 40 orang. Setiap penumpang kelas utama boleh membawa bagasi 30 kg Dan kelas ekononi boleh membawa bagasi 10 kg. 30x + 10y = 1200

2. Suatu pesawat mempunyai tempat duduk tidak lebih dari 180 penumpang. Setiap penumpang kelas bisnis boleh membawa bagasi 30 kg dan kelas ekonomi 20 kg. Kapasitas bagasi pesawat hanya 5400 kg, maka model matematika dari persoalan di atas adalah TOLONG BANTU YA / MAKASIH :)


Jawab:

x + y ≤ 180

3x + 2y ≤ 540

x ≥ 0

y ≥ 0

Penjelasan dengan langkah-langkah:

misal x = penumpang kelas bisnis

misal y = penumpang kelas ekonomi

total penumpang tidak lebih dari 180

maka, x + y ≤ 180

maks bagasi penumpang bisnis =30 kg

maks bagasi penumpang ekonomi = 20kg

maks bagasi pesawat = 5400 kg

maka, 30x + 20y ≤ 5400 = 3x + 2y ≤ 540 (disederhanakan)

syarat, penumpang dalam nilai positif

x ≥ 0

y ≥ 0

*x dapat ditukar menjadi y menyesuaikan permisalan


3. suatu pesawat udara mempunyai tempat duduk tidak lebih dari 50 penumpang. penumpang kelas utama boleh membawa bagasi paling banyak 60 kg, sedangkan penumpang kelas ekonomi dibatasi 30 kg. pesawat tersebut dapat membawa bagasi tidak lebih dari 1.800 kg.buatlah model matematikanya!


penumpang maksimal = 50 orang
kelas I = 60 kg/org, & kelas II = 30 kg/org
beban maksimal = 1800 kg

misal: a = penumpang kls I, & b = penumpang kls II
a x 60 + b x 30 = 1800 --> 60a + 30b = 1800
a + b = 50 --> (dikali 30) : 30a + 30b = 1500

60a + 30b = 1800
30a + 30b = 1500
------------------------------ - (eliminasi)
30a = 300
a = 300/30
a = 10 ---> a + b = 50 --> b = 50-10 = 40

jadi, penumpang kelas I sebanyak 10 org, & penumpang kelas II sebanyak 40 org

Jawaban:

Penumpang   Bagasi       tiket

Utama                x                60     1.600.000

Ekonomi            y                20       850.000

Jumlah             52            1.560   

Persamaan Linier

x + y ≤ 52

60x + 20y ≤ 1.560

Ditanya, pendapatan maksimum

1.600.000x + 850.000y =

Titik potong A ; kondisi cuma mengangkut penumpang utama (y=0)

x + y    ≤ 52

x + (0) ≤ 52

     x     ≤ 52

60x + 20y    ≤ 1.560

60x + 20(0) ≤ 1.560

       x          ≤ 1.560/60 

           x      ≤ 26

Jadi, titik potong A = (26, 0)

Titik potong B ; kondisi cuma mengangkut penumpang ekonomi (x=0)

x + y    ≤ 52

(0) + y ≤ 52

     y     ≤ 52

60x + 20y    ≤ 1.560

60(0) + 20y ≤ 1.560

             y    ≤ 1.560/20 

          y       ≤ 78

Jadi, titik potong B = (0, 52)

Titik potong C ; perpotongan garis A dan B

    x + y     ≤ 52      |x20| 20x + 20y ≤ 1.040

60x + 20y ≤ 1.560 |x1  | 60x + 20y ≤ 1.560

                                    ------------------------ ---

                                       40x         ≤ 520

                                               x     ≤ 520/40

                                                   x ≤ 13

     x + y ≤ 52

(13) + y ≤ 52

          y ≤ 52 - 13

          y ≤ 39

Jadi, titik potong C = (13, 39)

Masukkan fungsi 1.600.000x + 850.000y =

Titik A = (26, 0)   => 1.600.000(26) + 850.000(0)   = 41.600.000

Titik B = (0, 52)   => 1.600.000(0) + 850.000(52)   = 44.200.000

Titik C = (13, 39) => 1.600.000(13) + 850.000(39) =

                           (=) 20.800.000 + 33.150.000       = 53.950.000

---------------------------------------------------------------------------------

Jadi, pendapatan maksimun yang bisa didapat adalah Rp. 53.950.000,-

dengan mengangkut 13 penumpang utama dan 39 penumpang ekonomi
