7. bagaimana langkah langkah untuk menampilkan nomor halaman.


di cari di daftar halamannya - Buka dokumen yang akan ditampilkan nomor halamannya
- Kemudian klik menu/ribbon Insert, lalu klik ikon Page Number pada group Header and Footer
- Selanjutnya, pilih posisi nomor halaman tersebut ditampilkan, bisa atas halaman (Top of Page), dibawah halaman (Bottom of Page), di margin/batas kertas (Page Margin), atau di posisi kursor saat ini (Current Position), lalu pilih desain/bentuk nomor halamannya
- Kalau ingin diatur lagi (misal nomor halamannya dimulai dari angka 2 atau memakai angka romawi), bisa ke Insert-Page Number-Format Page Number

28. sebutkan 5 langkah mencetak laporan neraca dalam aplikasi myob accounting​


Langkah-langkah untuk mencetak/print pada program MYOB yaitu :

1. Pada Command Panel pilih Report

2. Disini misalkan kita akan mengeprint Standart Balance Sheet

3. Pilih bulan Desember (jika akhir periode adalah Desember)

4. Klik Print, jika sudah terhubung dengan printer.

Pembahasan

Atau dengan cara lain yaitu menyimpan file dulu menjadi format PDF atau excel, dengan cara :

1. Pada Command Panel pilih Report

2. Disini misalkan kita akan mengeprint Standart Balance Sheet

3. Pilih bulan Desember (jika akhir periode adalah Desember)

4. Klik Send to, pilih PDF (jika menginginkan format ecxel, maka excel yang diprint), masukkan ke folder yang kita kehendaki.

5. Kemudian buka folder file tersebut

6. Lali klik print.

Cara ini tidak hanya digunakan untuk mengeprint saja, tetapi juga merupakan cara untuk menyimpan MYOB menjadi format PDF atau excel.
