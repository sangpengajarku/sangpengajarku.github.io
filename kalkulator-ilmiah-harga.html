21. Harga sebuah kalkulator dan sebuah flashdisk Rp135.000,00. Sedangkan, harga sebuah kalkulator dan dua buah flashdisk Rp200.000,00. Tentukan harga dari 3 kalkulator dan 2 flashdisk Jawab:jawab Plss mau d kumpulin​


Misalkan :

x = harga kalkulator

y = harga flashdisk

Model Matematika :

x + y = 135.000

x + 2y = 200.000

.................................

Metode Substitusi :

x + y = 135.000

x = 135.000 - y

Substitusikan x = 135.000 - y ke persamaan x + 2y = 200.000

x + 2y = 200.000

135.000 - y + 2y = 200.000

135.000 + y = 200.000

y = 65.000

Nilai y (flashdisk) = Rp 65.000,00

Substitusikan y = 65.000 ke x + y = 135.000

x + y = 135.000

x + 65.000 = 135.000

x = 70.000

Nilai x (kalkulator) = Rp 70.000,00

Harga 3 kalkulator + 2 flashdisk :

(70.000 x 3) + (65.000 x 2)210.000 + 130.000340.000

Jadi, harga 3 kalkulator dan 2 flashdisk adalah Rp 340.000,00
