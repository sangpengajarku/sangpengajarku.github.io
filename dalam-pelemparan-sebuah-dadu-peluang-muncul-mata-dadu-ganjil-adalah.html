5. Sebuah dadu dilempar maka peluang munculnya mata dadu genap atau mata dadu ganjil adalah.


Penjelasan dengan langkah-langkah:

Diketahui ruang sampel pelemparan sebuah dadu S = {1, 2, 3, 4, 5, 6} sehingga n(S) = 6.

A.kejadian muncul mata dadu berangka ganjil, maka

= A = {1, 3, 5}

= n(A) = 3

= P(A) = n(A)/n(S)

= P(A) = 3/6 = 1/2

B.Genap

= b = {2, 4, 6}

= n(B) = 3

= P(B) = n(B)/n(S)

= P(B) = 3/6 = 1/2

23. Sebuah dadu dilempar sekali, peluang muncul mata dadu ganjil adalah ......


Sebuah dadu dilempar sekali, peluang muncul mata dadu ganjil adalah [tex]\boxed {\frac{1}{2} }[/tex].

Hasil tersebut diperoleh dengan menggunakan rumus peluang yaitu [tex]\boxed {P(A) = \frac{n(A)}{n(S)} }[/tex].

Penjelasan dengan langkah-langkah

Diketahui :

Sebuah dadu dilempar sekali.

Ditanya :

Peluang muncul mata dadu ganjil.

Penyelesaian :

Menentukan peluang

Ruang sampel (S) = {1, 2, 3, 4, 5, 6}

                     n (S) = 6

Kejadian muncul mata dadu ganjil (A) = {1, 3, 5}

                                                      n (A) = 3

Peluang

P (A) = [tex]\displaystyle \frac{n(A)}{n(S)}[/tex]

        = [tex]\displaystyle \frac{3}{6}[/tex]

        = [tex]\displaystyle \frac{1}{2}[/tex]

Jadi peluang muncul mata dadu ganjil adalah [tex]\displaystyle \frac{1}{2}[/tex].
