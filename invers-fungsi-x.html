12. Diketahui fungsi g(x)=x(2-x). Jika g^-1(x) merupakan invers dari fungsi g(x), grafik invers g adalah..


Fungsi invers adalah pemetaan yang mempunyai arah berlawanan dengan fungsinya. Misal, suatu fungsi memetakan dari himpunan A ke himpunan B. Maka, yang dimaksud fungsi invers adalah fungsi yang memetakan dari himpunan B ke himpunan A.

Pembahasan

g(x) = x(2 - x)

y = x(2 - x)

2x - x² = y

x² - 2x = -y

(x² - 2x + 1) - 1 = -y

(x - 1)² = -y + 1

x - 1 = [tex]\sqrt{-y+1}[/tex]

x = [tex]\sqrt{-y+1}+1[/tex], atau

x - 1 = [tex]-\sqrt{-y+1}[/tex]

x = [tex]-\sqrt{-y+1}+1[/tex]

Inversnya, [tex]f^{-1}(x)=\sqrt{-y+1}+1[/tex] atau [tex]f^{-1}(x)=-\sqrt{-y+1}+1[/tex]

Jadi, gambarnya grafiknya adalah pilihan B (terlampir).

13. Tentukan Domain fungsi f(x) = 4 − ^2 agar inversnya juga merupakan fungsi, serta tentukan fungsi inversnya.


Invers f(x) adalah f-¹(x) = 2 ± √(4 - x) dengan domain x ≤ 4.

Penjelasan dengan Langkah-Langkah

Diketahui:

f(x) = 4x - x²

Ditanyakan:

Invers dan domain invers.

Jawab:

f(x) = 4x - x²

y = 4x - x²

-y = x² - 4x

-y + 4 = x² - 4x + 4

-y + 4 = (x - 2)²

±√(-y + 4) = x - 2

2 ± √(4 - y) = x

f-¹(x) = 2 ± √(4 - x)

Domain

4 - x ≥ 0

4 ≥ x

Jadi, invers f(x) adalah f-¹(x) = 2 ± √(4 - x) dengan domain x ≤ 4.

Langkah-langkah mencari fungsi dari invers fungsi :

1. Misalkan f-¹(x) adalah y

2. Nyatakan x dalam y

3. Pindahkan y ke ruas kiri dan x keruas kanan.

4. Ganti x menjadi f(x)

• || Penyelesaian ||

• Dik :

f-¹(x) = (x + 2)/x

• Dit :

Nilai f(3) = .....?

• Jawaban.

Pertama kita cari rumus dari fungsi x.

f-¹(x) = (x + 2)/x

y = (x + 2) / x

xy = x + 2

xy - x = 2

x (y - 1) = 2

x = 2 / (y - 1)

f(x) = 2/(x - 1) ........ rumus f(x)

Setelah dapat rumus f(x), maka kita cari nilai f(3).

f(x) = 2 / (x - 1)

f(3) = 2 / (3 - 1)

f(3) = 2/2

f(3) = 1 ✔️

Jadi, nilai dari fungsi f(3) adalah 1.
