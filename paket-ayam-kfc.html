28. seorang peternakan mampu menghasilkan ayam di jual dengan harga Rp.25jt. ayam tersebut di proses menjadi KFC di jual dengan harga Rp.36 jt. kemudian KFC ini di beli oleh pengusaha makanan dengan merk tertentu dengan harga Rp. 40jt. hitung nilai tambah dari sektor produksi ayam tersebut?​


Diketahui :

Ayam ternak = Rp25.000.000

Ayam KFC = Rp36.000.000

Ayam pengusaha = Rp40.000.000

Ditanya :

Nilai tambah = ?

Nilai Tambah adalah total kenaikan harga dari suatu produk barang/jasa. Namun, untuk harga produk pertama, tetap ditulis apa adanya. Maka nilai tambah masing-masingnya ....

Ayam ternak = 25.000.000

Ternak ke KFC = (36-25) = 11.000.000

KFC ke pengusaha = (40-36) = 4.000.000

Maka, total nilai tambahnya ....

= 25.000.000 + 11.000.000 + 4.000.000

= 36.000.000 + 4.000.000

= Rp40.000.000

Jadi, nilai tambah dari sektor produksi ayam tersebut adalah Rp40.000.000

Semoga Bermanfaat

Jika fungsi permintaan dianggap mengikuti garis lurus maka persamaan fungsi permintaan dapat dicari menggunakan persamaan garis lurus yang dua titiknya diketahui, sebagai berikut:

24.000                   100

30.000                    50


DITANYA

Berapa paket terjual jika harganya 27.000?


PENYELESAIAN

Mencari persamaan fungsi permintaan yang melalui titik (24.000, 100) dan (30.000, 50)

[tex]\frac{P-24.000}{30.000-24.000} =\frac{Q-100}{50-100}[/tex]

[tex]\frac{P-24.000}{6.000} =\frac{Q-100}{-50}[/tex]

[tex]\frac{P-24.000}{-120}[/tex] = Q - 100

P - 24.000 = -120(Q - 100)

P - 24.000 = -120Q + 12.000

P = -120Q + 12.000 + 24.000

P = -120Q + 36.000


Mencari jumlah paket yang terjual per harinya jika harga = 27.000

P = -120Q + 36.000

27.000 = -120Q + 36.000

27.000 - 36.000 = -120Q

-9.000 = -120Q

Q = 75


KESIMPULAN

Pada sebuah restoran cepat saji, ketika harga paket ayam 24.000 maka restoran tersebut menjual sebanyak 100 paket tiap hari. Tetapi ketika harga paket ayam naik menjadi 30.000 maka restoran tersebut dapat menjual paket ayam tersebut sebanyak 50 paket per hari. Jika harga paket ayam 27.000, jumlah penjualan ayam tersebut per harinya adalah 75 paket.
