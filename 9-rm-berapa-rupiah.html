8. Jika [tex]\rm{ 4^{x} - 4^{x \: - \: 1} = 6}[/tex] maka, [tex]\rm{ (2x)^{x}}[/tex]A. [tex]\rm{ 3}[/tex]B. [tex]\rm{27 }[/tex]C. [tex]\rm{ 3\sqrt{3} }[/tex]D. [tex]\rm{9 }[/tex]E. [tex]\rm{ 9\sqrt{3} }[/tex]​


Jawaban:

[tex] \sf{jika \: {4}^{x} - {4}^{x - 1} } = 6 \: maka \: (2x) {}^{x} \: adalah \: \boxed{3 \sqrt{3} } [/tex]

.

PENDAHULUAN :

Eksponen atau sering disebut Bilangan Berpangkat adalah sebuah bentuk aturan dalam operasi hitung bilangan yang berbentuk pangkat perkalian yang dimana jika dikali harus bernilai berulang.

.

PEMBAHASAN :

Bilangan berpangkat pada dasar nya ditulis dengan notasi sebagai berikut

[tex] \boxed{ \sf{ {a}^{n} } } [/tex]

Dimana,

[tex] \sf{a \: merupakan \: bilangan \: pokok} [/tex]

[tex]\sf{n} \: merupakan \: bilangan \: pangkat \: [/tex]

sehingga,

[tex]\sf{ {a}^{n} } = a \times a \times a \times ... \times n \: (sebanyak \: n).[/tex]

Sifat - sifat Eksponen bilangan berpangkat sebagai berikut

[tex] {a}^{0} = 1[/tex]

[tex] {a}^{ - n} = \frac{1}{ {a}^{n} } [/tex]

[tex]( {a}^{x} ) {}^{y} = {a}^{xy} [/tex]

[tex] {a}^{x} . {a}^{y} = {a}^{x + y} [/tex]

[tex] \frac{ {a}^{x} }{ {a}^{y} } = {a}^{x - y} [/tex]

[tex]( \frac{a}{b} ) {}^{n} = \frac{ {a}^{n} }{ {b}^{n} } [/tex]

[tex](ab) {}^{n} = {a}^{n} . {b}^{n} [/tex]

[tex] {a}^{ \frac{x}{y} } = \sqrt[y]{ {a}^{x} } [/tex]

[tex] {a}^{n} \times {b}^{n} = (a × b) {}^{n} [/tex]

.

PENYELESAIAN :

Diketahui :

[tex]\sf{ {4}^{x} - {4}^{x - 1} } = 6[/tex]

Ditanyakan :

[tex]\sf{}nilai \: dari \: (2x) {}^{x} [/tex]

Jawab :

[tex]\sf{} {4}^{x} - {4}^{x - 1} = 6[/tex]

[tex]\sf{} {4}^{x} - {4}^{x} . {4}^{ - 1} = 6[/tex]

[tex]\sf{} {4}^{x} - \frac{ {4}^{x} }{4} = 6[/tex]

misalkan :

[tex]\sf{} nilai \: dari \: ( {4}^{x} ) = a[/tex]

maka,

[tex]\sf{} {4}^{x} - \frac{ {4}^{x} }{4} = 6[/tex]

[tex] \sf{} a - \frac{a}{4} = 6[/tex]

[tex]\sf{} 4a - a = 24[/tex]

[tex]\sf{} 3a = 24[/tex]

[tex]\sf{} a = \frac{24}{3} [/tex]

[tex]\sf{} a = 8[/tex]

Mencari nilai x

[tex]\sf{} ( {4}^{x} ) = a[/tex]

[tex]\sf{} {4}^{x} = 8 \\ \sf{} ( {2})^{2x} = {2}^{3} \\ \sf{} 2x = 3 \\ \sf{} x = \frac{3}{2} [/tex]

maka, nilai dari

[tex]\sf{} ( {2x})^{x} = (2( \frac{3}{2} )) {}^{ \frac{3}{2} } = ( \frac{6}{2} ) {}^{ \frac{3}{2} } = ( {3})^{ \frac{3}{2} } = \sqrt{ {3}^{3} } [/tex]

[tex]\sf{} \sqrt{27} = \sqrt{(9 \times 3)} = 3 \sqrt{3} [/tex]

.

KESIMPULAN :

Jadi,

[tex]\sf{} jika \: {4}^{x} - {4}^{x - 1} = 6 \: maka \: (2x) {}^{x} \: adalah \: \boxed{3 \sqrt{3} }[/tex]

.
