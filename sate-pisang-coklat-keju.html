Kelas         : X
Pelajaran   : Matematika
Kategori     : Sistim Persamaan Linear Tiga Variabel
Kata Kunci : toko roti Bu Indah, roti, pisang, coklat, keju, harga

Kode : 10.2.3 [Kelas 10 Matematika Bab 3 - Sistem Persamaan Linear]

Penyelesaian

Naskah soal aslinya:
Toko roti Bu Indah menjual roti pisang, roti coklat, dan roti keju. Harga 2 roti pisang, 1 roti coklat, dan 1 roti keju Rp15.300,00. Harga 3 roti pisang, 2 roti coklat, dan 1 roti keju Rp22.800,00. Harga 1 roti pisang, 3 roti coklat, dan 2 roti keju Rp22.100,00. Jika Nia membeli 1 roti pisang dan 5 roti coklat, ia harus membayar sebesar ...
a. Rp7.500,00
b. Rp11.500,00
c. Rp15.000,00
d. Rp21.500,00
e. Rp23.500,00

Dimisalkan
x = harga roti pisang 
y = harga roti coklat
z = harga roti keju
semua dalam rupiah

Persamaan-persamaan yang diperoleh dari soal di atas adalah sebagai berikut:
2x + y + z = 15.300 ........ [persamaan-1]
3x + 2y + z = 22.800 ...... [persamaan-2]
x + 3y + 2z = 22.100 ...... [persamaan-3]

Eliminasikan variabel z dari persamaan-1 dan persamaan-2
3x + 2y + z = 22.800
2x + y + z = 15.300
------------------------- ( - )
x + y = 7.500 ..... [persamaan-4]

Eliminasikan variabel z dari persamaan-1 (terlebih dahulu dikalikan 2) dan persamaan-3 
4x + 2y + 2z = 30.600
 x + 3y + 2z = 22.100
------------------------- ( - )
3x - y = 8.500 ..... [persamaan-5]

Eliminasikan variabel y dari persamaan-4 dan persamaan-5
3x - y = 8.500
x + y = 7.500
----------------- (+)
4x = 16.000
Diperoleh x = 4.000

Substitusikan nilai x ke (pilih) persamaan-4
4.000 + y = 7.500
Diperoleh y = 3.500

Substitusikan nilai x dan y ke (pilih ) persamaan-1
2(4.000) + 3.500 + z = 15.300
11.500 + z = 15.300
z = 15.300 - 11.500
Diperoleh z = 3.800

Jadi harga satuan dari roti pisang sebesar Rp4.000, harga satuan roti coklat sebesar Rp3.500, dan harga satuan roti keju sebesar Rp3.800.

Nia membeli 1 roti pisang dan 5 roti coklat.
Berarti, persamaan yang tersusun adalah
4.000(1) + 3.500(5) = 21.500.

Jadi, Nia harus membayar sebesar Rp21.500.

Jawaban : D

18. Toko roti bu indah menyediakan roti pisang, roti coklat, dan roti keju. harga 2 roti pisang, 1roti coklat, dan 1 roti keju Rp15.300,00. harga 3 roti pisang, 2 roti coklat, dan 1 roti keju Rp22.800,00. harga 1 roti pisang, 3 roti coklat, dan 2 roti keju Rp22.100,00. Jika nia membeli 1 roti pisang dan 5 roti coklat, ia harus membayar sebesar? Pake jalannya yaa kak.


Kelas         : 10
Mapel        : Matematika
Kategori    : Bab 3  Persamaan Linear Tiga Variabel
Kata kunci : SPLTV, soal cerita, roti toko Bu Indah

Kode : 10.2. 3 [Kelas 8 Matematika Bab Persamaan Linear Tiga Variabel]

Penjelasan : 

Metode penyelesaian SPLTV

1.  Metode subtitusi yaitu menyatakan suatu variabel dalam variabel lainnya yang selanjutnya digunakan untuk mengganti variabel yg sama dalam persamaan lainnya.
2.  Metode eliminasi yaitu mengeliminasi atau menghilangkan salah satu variabel untuk mendapatkan nilai variabel lainnya.
3.  Metode gabungan eliminasi dan subtitusi yaitu menentukan nilai salah satu variabel dengan metode eliminasi, selanjutnya nilai variabelnya.
-----------------------------------------------------

Soal : 

Toko roti bu indah menyediakan roti pisang, roti coklat, dan roti keju. harga 2 roti pisang, 1roti coklat, dan 1 roti keju Rp15.300,00. harga 3 roti pisang, 2 roti coklat, dan 1 roti keju Rp22.800,00. harga 1 roti pisang, 3 roti coklat, dan 2 roti keju Rp22.100,00. Jika nia membeli 1 roti pisang dan 5 roti coklat, ia harus membayar sebesar?
a. Rp7.500,00
b. Rp11.500,00
c. Rp15.000,00
d. Rp21.500,00
e. Rp23.500,00

Pembahasan : 

misalkan
roti pisang = p
roti coklat = c
roti keju = k

Penulisan persamaan dari pernyataan soal

2p + c + k = 15.300  ... pers I
3p + 2c + k = 22.800  .... pers II
p + 3c + 2k = 22.100  .... pers III

eliminasikan k dari persamaan I dan II

2p +   c + k = 15.300
3p + 2c + k = 22.800
--------------------------- -
-p - c           = -7500     (dikali -1 kesemua ruas)
p + c = 7500    .... pers IV

eliminasikan k dari persamaan I dan III

2p +   c + k = 15.300  |×2|    4p + 2c + 2k = 30.600
p + 3c + 2k = 22.100  |×1|      p + 3c + 2k = 22.100
                                            ------------------------------ -
                                             3p - c           = 8500    .... pers V

eliminasikan c dari persamaan IV dan V

p  + c = 7500 
3p - c = 8500
------------------ + 
4p      = 16.000
       p = 16.000 / 4
       p = 4.000

subtitusikan p = 4.000 ke dalam pers IV

p  + c = 7500
4.000 + c = 7.500
             c = 7.500 - 4.000
             c = 3.500

subtitusikan p = 4.000 dan c = 3.500 ke dalam pers I

2p + c + k = 15.300
2 (4.000) + 3.500 + k = 15.300
8.000 + 3.500 + k = 15.300
11.500 + k = 15.300
               k = 15.300 - 11.500
               k = 3.800

Harga : 1 roti pisang = Rp 4.000
             1 roti coklat = Rp 3.500
             1 roti keju = Rp 3.800

Nia membeli 1 roti pisang dan 5 roti coklat
p + 5c = 4.000 + 5 (3.500)
           = 4.000 + 17.500
           = 21.500

Jadi Nia harus membayar 1 roti pisang dan 5 roti coklat adalah Rp 21.500


Semoga bermanfaat

Pemodelan matematika
x ≥ 0, y ≥ 0, 2x + 3y ≤ 100, x + y ≤ 40

Untuk 2x + 3y = 100, jika x = 0 maka y = 100/3 dan jika y = 0 maka x = 50

Grafik terlampir

                                 --------------------- -

27. Seorang ibu memproduksi dua jenis keripik pisang, yaitu rasa coklat dan rasa keju. Setiap kilogram keripik pisang rasa coklat membutuhkan modal Rp 10.000,00 sedangkan keripik rasa keju membutuhkan modal Rp 15.000,00 per kilogram. Tiap hari ia hanya bisa memproduksi paling banyak 45 kilogram. Modal yang dimiliki Rp 600.000,00 Keuntungan tiap kilogram keripik pisang rasa coklat Rp 2.500,00 dan kripik rasa keju Rp 3.000,00 per kilogram. Keuntungan maksimum yang dapat diperoleh adalah​


Pembahasan :

Misal

x = keripik pisang rasa coklat

y = keripik pisang rasa keju

Modal :

keripik rasa coklat = Rp10.000,00

keripik rasa keju = Rp15.000,00

Modal ibu yang dimiliki = Rp500.000,00

10.000x + 15.000y ≤ 500.000

10x + 15y ≤ 500

2x + 3y ≤ 100 ........... (1)

(≤ : daerah diarsir ke bawah)

x = 0

=> 2(0) + 3y = 100

=> y = 100/3

=> (0, 100/3)

y = 0

=> 2x + 3(0) = 100

=> x = 50

=> (50, 0)

Tiap hari dapat memproduksi paling banyak 40 kg

x + y ≤ 40 ................ (2)

(≤ : daerah diarsir ke bawah)

x = 0

=> 0 + y = 40

=> y = 40

=> (0, 40)

y = 0

=> x + 0 = 40

=> x = 40

=> (40, 0)

x ≥ 0, y ≥ 0 .............. (3)

Fungsi Sasaran => Keuntungan

Keripik rasa coklat = Rp2.500,00

Keripik rasa keju = Rp3.000,00

f(x, y) = 2.500x + 3.000y

Ditanyakan :

Keuntungan terbesar = ... ?

Titik potong garis (1) dan (2)

2x + 3y = 100 |×1|

x + y = 40 |×2|

2x + 3y = 100

2x + 2y = 80

-------------------- -

....... y = 20

x + y = 40

x + 20 = 40

x = 20

(20, 20)

Berdasarkan gambar (lihat pada lampiran)

(0, 100/3), (20, 20) dan (40, 0)

Kita substitusikan ke fungsi sasaran

f(x, y) = 2.500x + 3.000y

f(0, 100/3) = 2.500(0) + 3.000(100/3) = 100.000

f(20, 20) = 2.500(20) + 3.000(20) = 110.000

f(40, 0) = 2.500(40) + 3.000(0) = 100.000

Jadi keuntungan maksimumnya adalah Rp110.000,00
