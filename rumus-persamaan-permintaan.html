6. Minta tolong ya..... 1.Tuliskan Rumus persamaan pada cerming cekung 2. Tuliskan rumus persamaan pada cermin cembung


s = jarak benda, s’ = jarak bayangan, h = P P’ = tinggi benda, h’ = Q Q’ = tinggi bayangan, F = titik fokus cermin cekung.Pada gambar di atas digambarkan dua berkas cahaya yakni P’BFQ’ dan P’AQ’. Berkas cahaya P’AQ’ memenuhi hukum pemantulan cahaya karenanya segitiga P’AP serupa dengan Q’AQ. Dengan demikian :Pada berkas cahaya P’BFQ’, segitiga BFA serupa dengan QFQ’ di mana jarak AB = tinggi benda (h) dan jarak FA = panjang fokus (f) cermin cekung. Dengan demikian :Ruas kiri dan ruas kanan persamaan 1 dan 2 sama, karenanya ruas kanan disamakan :Kalikan kedua ruas persamaan dengan s’ :Keterangan rumus :
s = jarak benda (positif jika benda berada di depan permukaan cermin cekung yang memantulkan cahaya)
s’ = jarak bayangan (positif jika bayangan benda berada di depan permukaan cermin cekung yang memantulkan cahaya atau bayangan bersifat nyata)
f = panjang fokus (positif jika titik fokus cermin cekung terletak di depan permukaan cermin cekung yang memantulkan cahaya)
Ingat selalu aturan tanda cermin cekung ketika menggunakan rumus ini untuk menyelesaikan soal-soal cermin cekung.Perbesaran Bayangan (m)
Amati gambar pembentukan bayangan benda di atas. Segitiga PAP’ dan QAQ’ serupa maka kita dapat menurunkan hubungan antara jarak benda dan jarak bayangan dengan tinggi benda dan tinggi bayangan :Keterangan :
h = tinggi benda (positif jika benda berada di atas sumbu utama cermin cekung atau benda tegak. Negatif apabila benda terbalik)
h’ = tinggi bayangan (positif jika bayangan berada di atas sumbu utama cermin cekung atau bayangan tegak. Negatif apabila bayangan terbalik)
s = jarak benda (positif jika benda dilalui berkas cahaya datang)
s’ = jarak bayangan (positif jika bayangan dilalui berkas cahaya atau bayangan nyata. Negatif apabila bayangan tidak dilalui berkas cahaya atau bayangan maya)
Rumus di atas dapat ditulis lagi menjadi seperti di bawah dengan menambahkan simbol m :Keterangan :
m = perbesaran bayangan

Rumus Kuadratik Yang Dimaksud:

[tex]X _{1.2} =\frac{ - b \: ± \sqrt{ {b}^{2} - 4ac } }{ \: \: 2a} [/tex]

[tex]X _{1} = \frac{ - 2 + 8}{ \: \: \: 2} \: \: \: atau \: \: \: X _{2} = \frac{ - 2 - 8}{ - 2} [/tex]

[tex]X _{1} = \frac{6}{2} \: \: \: atau \: \: X _{2} = \frac{ - 10} { \: \: 2} [/tex]

18. Fungsi permintaan dinyatakan dengan persamaan Qd = 40 – P dan fungsi penawaran dinyatakan dengan rumus Qs = – 5+ 2P , maka keseimbangan adalah adalah​


keseimbangan antara fungsi permintaan Qd = 40 - P dan fungsi penawaran Qs = - 5 + 2P adalah saat harga Rp 15 dan saat jumlah permintaan 25 unit

Pembahasan

Bentuk fungsi permintaan dan juga penawaran :

Qd = a - bP atau Pd = a - bQ

Qs = a - bP atau Ps = a - bQ

Keterangan :

Qd = jumlah permintaan fungsi permintaanQs = jumlah penawaran fungsi penawarana = konstantab = gradienPd = harga pada fungsi permintaanPs = harga pada fungsi penawaran

Persamaan yang digunakan untuk menentukan keseimbangan permintaan dan penawaran

Qd = Qs

Diketahui

Qd = 40 - P

Qs = - 5 + 2P

Ditanyakan

keseimbangannya = ...?

Penyelesaian

Qd = Qs

40 - P = - 5 + 2P

- P - 2P = - 5 - 40

- 3P = - 45

P = - 45/- 3

P = 15

Qd = 40 - P

Qd = 40 - 15

Qd = 25

Jadi, keseimbangan antara fungsi permintaan Qd = 40 - P dan fungsi penawaran Qs = - 5 + 2P adalah saat harga Rp 15 dan saat jumlah permintaan 25 unit

20. kak minta tolong dong kerjain yg a-b aja disuruh buktikan knpa rumus trigonometri nya begitu disuruh buktikan persamaan  makasih  tolong dibantu ya 


Buat segitiga ABC siku2 di B ---> dengan AB = alas, BC = tinggi, sudut A = α
sisi a ---> depan sudut α ( = BC = tinggi )
sisi b ---> depan sudut B ( sisi miring )
sisi c ---> sisi depan sudut C ( = AB = alas )

Soal (a) :
sin α = a/b, cos α = c/b, tan α = a/c, cot α = c/a
sin α / cos α = (a/b) / (c/b) = a/c = tan α ---> terbukti
cos α / sin α = (c/b) / (a/b) = c/a = cot α ---> terbukti

Soal (b) :
Pythagoras ---> a^2 + c^2 = b^2 ... (1)
sin^2 α = ( a^2 ) / ( b^2 ), cos^2 α = ( c^2 ) / ( b^2 )

sin^2 α + cos^2 α = ( a^2 )/( b^2 ) + ( c^2 )/( b^2 ) ---> penyebut sdh sama
= ( a^2 + c^2 ) / ( b^2 ) ---> a^2 + c^2 = b^2 [ pers (1) ] 
= ( b^2 ) / ( b^2 )
= 1 ---> terbukti

Pendahuluan

Apa Itu Persamaan Kuadrat?

Persmaan kuadrat adalah suatu persamaan yang pangkat tertingginya dua. Bentuk umum persamaan kuadrat adalah sebagai berikut

ax²+bx+c=0

a≠0 dan a,b,c anggota bilangan real. Di mana a merupakan koefisien dari x², b merupakan koefisien daei x, sedangkan c merupakan konstanta

Pembahasan

Menentukan Akar-Akar Persamaan Kuadrat

Ada tiga metode yang paling sering digunakan untuk menentukan akar-akar persamaan kuadrat yaitu dengan memfakrorkan, melengkapkan kuadrat sempurna, dan menggunakan rumus kuadratik atau "rumus ABC"

Tetapi karena yang diminta hanya menggunakan rumus kuadratik/ABC, Maka kali ini kita hanya akan membahas cara menentukan akar-akar persamaan kuadrat dengan menggunakan rumus ABC saja
