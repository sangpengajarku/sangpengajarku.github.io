10. Penggunaan Conditional If dilakukan jika?


Conditional if merupakan kalimat pengandaian. Conditional if(Conditional sentence) dibagi menjadi 3type
1.type 1 (Menghayal yg pasti akan terjadi)
Rumus= if+subject+verb1, subject+will+verb1
Contoh: if she studies hard tonight,she will pass the exams. (Jika dia prempuan belajar dengan keras,dia akan lulus ujian)

2. type2 (Menghayal yg lebih sulit terjadi)
Rumus= if+subject+v2, subject+would+verb1
Contoh: if I had money, i would travel around the world. (Jika saya punya uang,saya akan berkeliling keselutuh dunia)

3. type3 (Kalimat penyesalan)
Rumus= if+subject+had+verb3, subject+would+have+verb3
Contoh: if I had not stayed up late, I would have not been late.(Jika saya tidak begadang,saya tidak akan terlambat)

      }else{

          std::cout<<"Tahun "<<Tahun<<" bukan tahun kabisat";

      }

  }else{

      if(Tahun % 4 == 0){

          std::cout<<"Tahun "<<Tahun<<" merupakan tahun kabisat";

      }

  }

}

  int Tahun;

  std::cin>>Tahun;

  tahunKabisat(Tahun);

  return 0;

}

30. Kapan if clause bisa digunakan? Tense apa yg digunakan?


saat menggunaka kata kerja

Klausa “if” atau if-clause digunakan pada conditional tenses, yaitu kalimat untuk mengungkapkan pengandaian. Dalam struktur kalimatnya, conditional tenses terdiri atas 2 clause, yaitu subordinate clause dan main clause. Kata “if” itu sendiri letaknya adalah di bagian subordinate clause.



PEMBAHASAN


Secara umum conditional tenses terdiri atas subordinate clause (berupa kata “if” dan condition/syarat) dan main clause (berupa result/hasil). Sebagai contoh:


If you burn a wood, it becomes ash


Dalam kalimat ini, “burn a wood” adalah syarat (condition) untuk terjadinya hasil (result) berupa “it becomes ash.


Conditional tenses itu sendiri terdiri dari dari 4 tipe, yaitu type zero, type 1, type 2, dan type 3.



1. Type Zero

Type zero menjelaskan fakta ilmiah atau kebenaran secara umum. Baik main clause maupun condition clause menggunakan simple present tense.

(Subordinat) + (Main)

(If + simple present), (simple present)

Contoh = If you step on an ant, it dies.


2. Type 1

Type 1 digunakan bila result memiliki kemungkinan untuk terwujud di masa depan karena condition-nya bisa dipenuhi. Di sini, subordinatnya tetap menggunakan simple present namun main clause menggunakan simple future tense.

(Subordinat) + (Main)

(If + simple present), (simple future)

Contoh = if you push the button, water will flow.


3. Type 2

Type 2 digunakan untuk mengungkapkan sesuatu yang tidak nyata atau hampir mustahil. Karena itu, subordinatnya menggunakan simple past tense dan result menggunakan present conditional atau present continous conditional.

(Subordinat) + (Main)

(If + simple past), (present conditional)

Contoh = If I was in your position, I would tell her to stop.



4. Type 3

Type 3 digunakan untuk mengungkapkan hal yang berlawanan dengan kenyataan, atau sudah terlanjur kejadian hal sebaliknya.

(Subordinat) + (Main)

(If + simple past), (present conditional)

Contoh = if you had woken up earlier this morning, you would have arrived here on time.
