14. Cara memfaktorkan X2 + 3x-28= 0 Dengan rumusan abc


Jawaban:

Persamaan Kuadrat

Cara Pemfaktoran

x² + 3x - 28 = 0

bilangan jika dikali hasilnya -28 dan jika ditambah hasilnya 3

7 × -4 = -28

7 + -4 = 7 - 4 = 3

bilangannya adalah 7 dan -4lalu masukkan kedalam rumus

[tex](x....)(x...) = 0[/tex]

[tex](x + 7)(x - 4) = [/tex]

[tex]x1 + 7 = 0[/tex]

[tex]x1 = 7[/tex]

[tex]x - 4 = 0[/tex]

[tex]x = 4[/tex]

Dengan Rumus ABC

[tex]x = \frac{ - b± \sqrt{ {b}^{2} - 4ac } }{2a} [/tex]

[tex]x = \frac{ - 3± \sqrt{ {3}^{2} - (4 \times 1 \times - 28) } }{2 \times 1} [/tex]

[tex]x = \frac{ - 3± \sqrt{9 - - 112} }{2} [/tex]

[tex] x = \frac{ - 3± \sqrt{121} }{2} [/tex]

[tex]x = \frac{ - 3±11}{2} [/tex]

[tex]x1 = \frac{ - 3 + 11}{2} = \frac{8}{2} = 4[/tex]

[tex]x2 = \frac{ - 3 - 11}{2} = \frac{ - 14}{2} = - 7[/tex]
