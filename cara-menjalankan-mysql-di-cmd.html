2. Perintah perintah mysql di cmd


pertama buka mysql dlu dari cmd, haru sesuai lokasi pada hdd
ex: cd C:\xampp\mysql\bin

kemudian jalankan mysql dengan perintah "mysql -u root"

1. Membuat Database: create database 'namadatabase';
   ex: create database sekolah;

2. Menghapus Database: drop database 'namadatabase';
   ex: drop database sekolah;

3. Menampilkan Daftar Database: show databases;

4. Menggunakan/Mengaktifkan Database: use 'namadatabase';
     ex: use sekolah;

5. Membuat Table :
CREATE TABLE nama_tbl(
kolom1 tipekolom1(ukuran),
kolom2 tipekolom2(ukuran),
...
);
ex:
CREATE TABLE Siswa(
NIS Char(10) NOT NULL PRIMARY KEY,
NAMA Varchar(40),
Umur Int Not Null Default 0,
Kelamin Enum('Pria','Wanita')
);
*Jangan Lupa Tanda "," Koma Pada Akhir

6. Melihat Daftar Tabel: show tables;

7. Melihat Deskripsi Tabel: desc 'namatabel';
   ex: desc siswa;

8.Memodifikasi Struktur Tabel
ALTER TABLE nama_tbl action sedangkan perintah untuk menambah kolom baru dengan tipe dan posisi tertentu adalah:
ALTER TABLE nama_tbl
ADD kolom_baru type() definisi

*Selalu Gunakan Tanda ";" Pada Akhiran Perintah

11. Bagaimana cara mengkoneksikan PHP dengan MYSQL?


Instruksi Buka koding PHP yang anda buat di dalam text editor seperti Windows Notepad ataupun yang lainnya.Simpan alamat, username, dan password untuk dua database MySQL dalam variable PHP, sebagai contoh "$myhost1 = "example1.com"; $myusername1 = "user1"; $mypassword1 = "password1"; $myhost2 = "example2.com"; $myusername2 = "user2"; $mypassword2 = "password2";".Buat koneksi ke database MySQL yang pertama dengan menggunakan fungsi "mysql_connect(server_address, username, password)". Fungsi "mysql_connect” akan mengembalikan link identifier MySQL jika berhasil. Misalnya "$link1 = mysql_connect ($ myhost1, $ myusername1, $ mypassword1);", dengan "$link1" menjadi MySQL identifier link untuk database pertama.Buat koneksi ke database MySQL kedua menggunakan fungsi "mysql_connect (server_address, username, password)" , tetapi dengan MySQL identifier link  yang berbeda daripada koneksi pertama. Misalnya, "$link2 = mysql_connect ($myhost2, $myusername2, $mypassword2);", dengan "$link2" menjadi MySQL identifier link untuk database kedua.Simpan lah menjadi file PHP, dan jangan lupa untuk menyertakan kodingan tag php yakni “<?php” dan “?>”

20. Cara mengetahui nama localhost mysql server


Jawaban:

Jika sudah masuk ke klien baris perintah cobalah ini:

mysql> select user();

Ini akan menampilkan sesuatu yang mirip dengan ini:

+----------------+

| user() |

+----------------+

| root@localhost |

+----------------+

1 row in set (0.41 sec)

contoh diatas login sebagai rootdari localhost.

Jika ingin mengetahui nomor port host lokal di mana Mysql berjalan,gunakan kueri ini pada klien baris perintah:

MySQL -

SHOW VARIABLES WHERE Variable_name = 'port';

mysql> SHOW VARIABLES WHERE Variable_name = 'port';

+---------------+-------+

| Variable_name | Value |

+---------------+-------+

| port | 3306 |

+---------------+-------+

1 row in set (0.00 sec)

kueri diatas akan memberi nomor port tempat MySQL dijalankan.

Jika ingin mengetahui nama host Mysql Anda, gunakan kueri ini pada klien baris perintah:

MySQL -

SHOW VARIABLES WHERE Variable_name = 'hostname';

mysql> SHOW VARIABLES WHERE Variable_name = 'hostname';

+-------------------+-------+

| Variable_name | Value |

+-------------------+-------+

| hostname | Dell |

+-------------------+-------+

1 row in set (0.00 sec)

Kueri diatasakan memberi Anda nama host untuk mysql.

Jika ingin mengetahui nama pengguna Mysql, gunakan kueri ini pada klien baris perintah:

MySQL -

select user();

mysql> select user();

+----------------+

| user() |

+----------------+

| root@localhost |

+----------------+

1 row in set (0.00 sec)

Kueri diatas akan memberi Anda nama pengguna untuk mysql.

22. cara menghapus database dan tabel (pake MYSQL)


Menghapus Tabel
DROP TABLE [IF EXIST]  table_name [, table_name, ...]
keterangan : parameter [IF_EXIST] berupa opsional yang berguna menghilangkan pesan error jika tabel tidak ada.
cara pakai : mysql> Drop Table [IF EXIST]  tabel_mahasiswa;

Menghapus Database
DROP DATABASE [IF_EXIST] Database_name;
Keterangan : parameter [IF_EXIST] berupa opsional yang berguna menghilangkan pesan error jika database tidak ada.
cara pakai : mysql> DROP DATABASE [IF_EXIST] DB_Penjualan;
