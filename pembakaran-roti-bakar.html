9. sebuah kafe, harga 3 cangkir kopi sama dengan harga 4 roti bakar. Jika harga 6 cangkir kopi dan 5 roti bakar adalah Rp78.000,00, maka selisih harga secangkir kopi dan sebuah roti bakar di kafe itu adalah


Diketahui :

3 cangkir kopi = 4 roti bakar

6 cangkir kopi dan 5 roti bakar = Rp78.000,00

Ditanyakan :

Selisih harga secangkir kopi dan sebuah roti bakar.

Penyelesaian : Misal

1 cangkir kopi = x

1 roti bakar = y

Persamaan

3x = 4y

6x + 5y = Rp78.000,00

Mencari harga roti bakar = y

→ 6x + 5y = Rp78.000,00

→ 6( [tex] \tt \frac{4}{3} [/tex]y ) + 5y = Rp78.000,00

→ 8y + 5y = Rp78.000,00

→ 13y = Rp78.000,00

→ y = Rp6.000,00

Mencari harga secangkir kopi = x

→ 3x = 4y

→ 3x = 4( Rp6.000,00 )

→ 3x = Rp24.000,00

→ x = Rp8.000,00

Selisih secangkir kopi dan sebuah roti bakar

→ x - y

→ Rp8.000,00 - Rp6.000,00

→ Rp2.000,00

-

Maka selisih harga secangkir kopi dan sebuah roti bakar di kafe itu sebesar Rp2.000,00

-

[tex]\purple{\boxed{\blue{\boxed{\green{\star{\orange{\ \: \: JK \: \: \: {\green{\star}}}}}}}}} [/tex]

Maka selisih harga secangkir kopi dan sebuah roti bakar di kafe itu adalah Rp. 2.000,00

=> Pembahasan:

-

SPLDV Adalah Sistem Persamaan Linear Yang Memiliki Dua Variabel Yang Saling Berhubungan Antara Satu Dengan Yang Lainnya.

-

=> Penyelesaian:

Kita Anggap =

Cangkir Kopi = x

Roti Bakar = y

Diketahui =

3x = 4y

6x + 5y = Rp. 78.000,00

Ditanya =

Selisih x Dan y ?

Jawab:

3x = 4y

6x + 5y = 78.000

x = 4/3y

6x + 5y = 78.000

Substitusikan Nilai x Kedalam Persamaan

6x + 5y = 78.000

6x + 5y = 78.000

6( 4/3y ) + 5y = 78.000

6 × 4/3y + 5y = 78.000

2 × 4y + 5y = 78.000

8y + 5y = 78.000

13y = 78.000

y = 78.000 ÷ 13

y = 6.000

Substitusikan Nilai y Kedalam Persamaan

x = 3/4y

x = 4/3y

x = 4/3 × 6.000

x = 8.000

Selisih x Dan y =

= 8.000 - 6.000

= Rp. 2.000,00

Jadi, selisih harga secangkir kopi dan sebuah roti bakar di kafe itu adalah Rp. 2.000,00

19. [tex] \huge \tt{Roti \: Bakar }[/tex]1). 1000^2 = (...)2). 5! + 2! = (...)UwU Roti Bakar ​


Pertanyaan:

1) 1.000² = .......

2) 5! + 2! = .......

Penyelesaian :

1. 1.000² =

= 1.000 × 1.000

= 1.000.000

2. 5! + 2! =

5! = ( 5 × 4 ) × ( 3 × 2 ) × 1

= 120

2! = 2 × 1

= 2

= 120 + 2

= 122

[tex] \red{ - andreas02juli - }[/tex]

Soalannya :

1). 1000^2 = (...)

2). 5! + 2! = (...)

Jawabannya :1). 1.000.000 ✔️2). 122 ✔️Langkah - langkahnya :No. 1) :Diketahui :

1000² = 1000 × 1000

Jawab :

= 1.000² ( 1000 × 1000 )

= 1.000.000 ✍️

Jadi, jawabannya = 1.000.000 ✔️

_______________________No. 2) : Diketahui :

5! = 5×4×3×2×1

2! = 2×1

Jawab :

= 5! + 2!

= ( 5×4×3×2×1 ) + ( 2×1 )

= 120 + 2

= 122 ✍️

Jadi, jawabannya = 122 ✔️
