15. apa perbedaan yang mendasar antara aplikasi microsoft office Excel 2007 dengan versi sebelumnya?


dari tahun rilisnya
dari tampilannya excel 2007 lebih modern
dari fiturnya excel 2007 lebih banyak dr pd v.sebelumnya- Dari tahun rilisnya (sudah jelas, yang satu rilis tahun 2007, yang satunya rilis tahun 2003) 
- Dari tampilan: tampilan Excel 2007 lebih rapih & modern karena memakai sistem Ribbon/tab untuk menu barnya, sementara tampilan Excel 2003 masih memakai sistem Drop Down untuk menu barnya 
- Dari ekstensi/format filenya: file format Excel 2007 berekstensi/berakhiran .docx dan walaupun format filenya baru tapi masih bisa membuka dan membuat format file Excel 2003 (.doc). Sementara format file Excel 2003 berekstensi .doc dan tidak bisa membuka dan membuat format file .docx (format file Excel 2007) 
- Dari fitur: ada penambahan fitur pada Excel 2007 yang tidak ada pada Excel 2003 seperti fitur SmartArt untuk membuat diagram, AutoSum untuk menjumlahkan data secara otomatis, formula/rumus-rumusnya makin banyak, template/temanya makin beragam, dsb

17. Bagaimana penggolongan fungsi pada aplikasi microsoft office exel 2007


Fungsi di Ms. office excel 2007 digolongkan menjadi:
1. Fungsi matematika
umumnya memakai tanda operator seperti +, -, *, /, ^

2. Fungsi statistika

4. Fungsi string
