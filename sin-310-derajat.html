16. Diketahui cos 40 derajat= p Nyatakan nilai trigonometri berikut dalam p! a. Cos (-50derajat) b. Sin 220 derajat c. cotan 310 derajat d. Sec 850 derajat


Kategori Soal : Matematika - Trigonometri
Kelas : X (1 SMA)
Pembahasan :
Pembagian sudut dalam trigonometri.
Dalam trigonometri, besar sudut α dibagi menjadi 4 kelompok, yaitu :
Kuadran I : 0° < α < 90° (0 < α < π/2)
sin (90° - α) = cos α
cos (90° - α) = sin α
tan (90° - α) = cotan α
cotan (90° - α) = tan α
secan (90° - α) = sec α
sec (90° - α) = cosec α

Kuadran II : 90° < α < 180° (π/2 < α < π)
sin (180° - α) = sin α
cos (180° - α) = -cos α
tan (180° - α) = -tan α
cotan (180° - α) = -cotan α
secan (180° - α) = secan α
sec (180° - α) = -sec α

Kuadran III : 180° < α < 270° (π < α < 3π/2)
sin (270° - α) = -cos α
cos (270° - α) = -sin α
tan (270° - α) = cotan α
cotan (270° - α) = tan α
secan (270° - α) = -sec α
sec (270° - α) = -cosec α

Kuadran IV : 270° < α < 360° (3π/2 < α < 2π)
sin (360° - α) =sin (-α) = -sin α
cos (360° - α) = cos (-α) = cos α
tan (360° - α) = tan (-α) = -tan α
cotan (360° - α) = cotan (-α) = -cotan α
secan (360° - α) = secan (-α) = -secan α
sec (360° - α) sec (-α) = sec α

Mari kita lihat soal berikut.
Diketahui cos 40° = p dan perhatikan gambar terlampir.
AB² = AC² + BC²
⇔ BC² = AB² - AC²
⇔ BC² =  1² - p²
⇔ BC² = 1 - p²
⇔ BC = √(1 - p²)
cos 40° = p/1 = p
sin 40° = √(1 - p²)/1 = √(1 - p²)
tan 40° = √(1 - p²)/p = 1/p √(1 - p²)
sec 40° = 1/cos 40° = 1/p
cosec 40° = 1/sin 40° = 1/(√(1 - p²)) = √(1 - p²)/(1 - p²) = 1/(1 - p²) √(1 - p²)
cotan 40° = 1/tan 40° = 1/[√(1 - p²)/p] = p/(√(1 - p²) = p(√(1 - p²))/(1 - p²) = p/(1 - p²) √(1 - p²)

Nilai dari bentuk trigonometri berikut, yaitu :
cos (-50)° = cos 50° = sin (90 - 40)° = sin 40° = √(1 - p²)
sin 220° = -cos (3 x 90 - 50)° = -cos 50° = -sin (90 - 40)° = -sin 40° = -√(1 - p²)
cotan 310° = -cotan (4 x 90 - 50)° = -cotan 50° = -tan (90 - 40)° = -tan 40° = -√(1 - p²)/p = -1/p √(1 - p²)
sec 850° = -sec (10 x 90 - 50)° = -sec 50° = -cosec (90 - 40)° = -cosec 40° = -1/√(1 - p²) = -√(1 - p²)/(1 - p²) = -1/(1 - p²) √(1 - p²)

Soal sebenarnya keliru, seharusnya yang diketahui sin 40° = p

Nilai dari bentuk trigonometri berikut, yaitu :
cos (-50)° = cos 50° = sin (90 - 40)° = sin 40° = p
sin 220° = -cos (3 x 90 - 50)° = -cos 50° = -sin (90 - 40)° = -sin 40° = -p
sec 310° = sec (4 x 90 - 50)° = sec 50° = cosec (90 - 40)° = cosec 40° = 1/sin 40° = 1/p
sec 850° = -sec (10 x 90 - 50)° = -sec 50° = -cosec (90 - 40)° = -cosec 40° = -1/sin 40° = -1/p

[tex]\rm\circ~L=\frac{1}{2}\times a\times b\times sin~C[/tex]

dimana :

•a = sisi di depan sudut A

•b = sisi di depan sudut B

•c = sisi di depan sudut C

Pembahasan :

Diketahui :

sin 310°

Ditanya :

Terletak di kuadran berapa sin 310° ?

Jawab :

sin 310° terletak di kuadran IV

Kuadran IV (270° ≤ α ≤ 360°)

Nilai sin 310° di kuadran IV nantinya akan bernilai negatif, karena kuadran IV yang positif hanya cos

Kesimpulan :

Jadi, nilai sin 310° berada di kuadran IV.
