jadi 180.000.000 : 2 = 90.000.000.
1 mobil : 90.000.0005 motor + 1 mobil = 180.000.000
harga 5 motor = harga 1 mobil
artinya, harga 5 motor + 1 mobil = harga 2 mobil.
Sehingga, harga 1 mobil = 180.000.000 / 2

5 motor + 3 mobil = Rp19.000
7 motor + 2 mobil = Rp20.000

20 motor + 5 mobil = ......?


Penyelesaian:
Menggunakan metode eliminasi dan substitusi
Pemisalan: motor = x // mobil = y

5x + 3y = Rp19.000  | x7 | 35x + 21y = Rp133.000
7x + 2y = Rp20.000  | x5 | 35x + 10y = Rp100.000
                                                    11y = Rp33.000
                                                        y = Rp3.000

#Subtitusi
5x +         3y       = Rp19.000
5x + 3(Rp3.000) = Rp19.000
5x +   Rp9.000   = Rp19.000
5x = Rp19.000 - Rp9000
5x = Rp10.000
  x = Rp2.000

HP = ({x,y | Rp2.000, Rp3.000)

20x + 5y =  ...?
20(Rp2.000 + 5(Rp3.000) = ... ?
Rp40.000    +  Rp15.000  = Rp55.000

Jadi, uang yang didapat tukang parkir jika ada 20 motor dan 5 mobil yang parkir adalah sejumlah Rp55.000
