2. Cara menghitung rata rata nilai utbk


Cara menghitung rata-rata nilai UTBK adalah dengan membagi jumlah nilai UTBK dengan banyaknya nilai yang diperoleh

Rata-rata dalam statistika adalah suatu bilangan yang mewakili sekelompok data

Rata-rata dalam bahasa Inggris adalah Mean

Rumus :

Nilai rata-rata = [tex]\frac{jumlah\:nilai}{banyak\:data}[/tex]

Contoh soal :

Arif mendapatkan nilai sebagai berikut :

Ulangan ke-1  75

Ulangan ke-2  80

Ulangan ke-3  95

Ulangan ke-4  86

Ulangan ke-5  76

Nilai rata-rata ulangan Arif adalah ?

Jumlah nilai ulangan Arif = 75 + 80 + 95 + 86 + 76 = 412

Banyak data = 5

Nilai rata-rata = [tex]\frac{jumlah\:nilai}{banyak\:data}[/tex]

Nilai rata-rata = [tex]\frac{412}{5}[/tex] = 82,4

Pembahasan :

Cara menghitung rata-rata nilai UTBK adalah dengan menjumlahkan dahulu seluruh nilai UTBK lalu dibagi dengan banyaknya data nilai tersebut

☑ Pertidaksamaan Nilai Mutlak

-

Nilai x yang memenuhi |x + 1|+ 2x < 2 adalah x < ⅔

PEMBAHASAN :

|x + 1| + 2x < 3

|x + 1| < 3 - 2x

Dengan menerapkan sifat :

|x| < r maka -r < x < r

Maka :

x + 1 > -(3 - 2x)

x + 1 > -3 + 2x

x - 2x > -3 - 1

-x > -4

x < 4 { TM }

x + 1 < 3 - 2x

x + 2x < 3 - 1

3x < 2

x < ⅔

Maka nilai x yang memenuhi pertidaksamaan adalah x < ⅔

Penjelasan:

Asam Laktat adalah salah satu produk dari respirasi seluler otot hewan dalam kondisi anaerob (fermentasi). Asam laktat dihasilkan ketika otot hewan harus menghasilkan banyak energi dalam waktu singkat, namun oksigen untuk respirasi tidak terkompensasi dalam jumlah yang dibutuhkan. Mekanisme repirasi ini digunakan ketika respirasi aerob tidak menghasilkan ATP yang cukup dalam waktu "sekejap". Asam laktat ikut dihasilkan dan terakumulasi dalam sel otot, merupakan hasil pembongkaran piruvat hasil glikolisis.

Penjelasan:

Diketahui:

ABC siku-siku di C

CD merupakan garis tinggi

Panjang CD

Jika tambahan diketahui: (1) BC = 5 dan (2) BD = 3

Jawab:

Untuk pernyataan (1) BC = 5

Panjang AB = [tex]\sqrt{AC^2+BC^2}[/tex]

                    = [tex]\sqrt{7^2+5^2}[/tex]

                    = [tex]\sqrt{49+25}[/tex]

                    = [tex]\sqrt{74}[/tex]

Panjang CD = [tex]\frac{AC*BC}{AB}[/tex]

                    = [tex]\frac{7*5}{\sqrt{74} }[/tex]

Untuk pernyataan (2) BD = 3

AD² + CD² = AC²    ⇒  AD² + CD² = 49

BD² + CD² = BC²    ⇒     9   + CD² = BC²

                                    ______________ -

                                    AD² - 9 = 49 - BC²

                                         BC² = 49 + 9 - AD²

                                         BC² = 58 - AD²

Pada segitiga ABC berlaku

A C ²  +  B C ²  =  A B ²

7 ²  + 58 - AD²   = ( A D  + 3)²

107 - AD²   = AD² + 6AD + 9

2AD² + 6AD - 98 = 0

Faktorkan, maka akan ketemu panjang AD

Jika Panjang AD sudah diketahui, Maka panjang CD juga dapat kita cari dengan pythagoras.
