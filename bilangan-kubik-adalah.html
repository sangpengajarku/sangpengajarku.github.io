2. Bilangan kubik satu angka yaitu? ... dan ...Bilangan kubik dua angka yaitu?  ... dan ...Bilangan kubik tiga angka yaitu? ... , ... , ... , ... , ...Bilangan kubik empat angka yaitu? ... , ... , ... , ... , ... , ... , ... , ... , ... , ... , ... , ...


Kelas : VII (1 SMP)
Materi : Bilangan Bulat
Kata Kunci : bilangan, bulat, kubik, pangkat, tiga

Pembahasan :
Pengertian bilangan pangkat n dapat kita definisikan sebagai berikut.
Jika a suatu bilangan real dan n suatu bilangan bulat positif, maka
aⁿ = a x a x ... x a
       ____v_____
           n faktor
dengan
n dinamakan eksponen atau pangkat.
a dinamakan bilangan pokok (atau basis atau bilangan dasar).
aⁿ dinamakan bilangan berpangkat.
a x a x ... x a (sampai dengan n suku) dinamakan hasil perpangkatan.

Bilangan kuadrat atau pangkat dua merupakan perkalian berulang sebanyak dua faktor.
a² = a x a
       __\/__
       2 faktor

Bilangan kubik atau pangkat tiga merupakan perkalian berulang sebanyak tiga faktor.
a³ = a x a x a
       ___\/___
        3 faktor

Mari kita lihat soal tersebut.
Tentukan bilangan-bilangan kubik satu angka, dua angka, tiga angka, dan empat angka!

Jawab :
Bilangan kubik satu angka, yaitu :
1³ = 1 dan 2³ = 8.
Bilangan kubik dua angka, yaitu :
3³ = 27 dan 4³ = 64.
Bilangan kubik tiga angka, yaitu :
5³ = 125, 6³ = 216, 7³ = 343, 8³ = 512, dan 9³ = 729
Bilangan kubik empat angka, yaitu :
10³ = 1000, 11³ = 1331, 12³ = 1728, 13³ = 2197, 14³ = 2744, 15³ = 3375, 16³ = 4096, 17³ = 4913, 18³ = 5832, 19³ = 6859, 20³ = 8000, dan 21³ = 9261.

Semangat!
