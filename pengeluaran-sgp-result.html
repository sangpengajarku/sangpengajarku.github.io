7. Fanny mengeluarkan dua buah kabel dengan gaya masing masing 100 N, dan keduanya membentuk sudut 120°, berapa resultan gaya dari kabel fanny?​


Jawaban:

FR = 100 N

Langkah Pengerjaan:

FR = √(F₁²+F₂²+2.F₁.F₂.cosθ)

FR = √(100²+100²+2.100.100.cos120)

FR = √(10000+10000+20000.(-1/2))

FR = √(20000-10000)

FR = √10000

FR = 100 N

Jawaban:

5 N ke Grup B

Penjelasan:

Grup A = 20 + 25 = 45 N

Grup B = 20 + 30 = 50 N

Resultan gaya = Grup A - Grup B

= 45 N - 50 N

= - 5 N

(-) disini memiliki arti mengarah ke Grup B

Grup A <--------•------->Grup B

45 N <--------•-------->50 N

---------> 5 N ke Grup B

kanan

a+b =15+15=30N

kiri

c=20N

30-20=10N ke kanan

F kanan = Fa + Fb = 15 + 15 = 30N

F kiri = Fc = 20N

Resultan(R) = F besar - F kecil

= F kanan - F kiri

= (30 - 20) N

= 10 N

jadi resultannya adalah 10 N....

29. 2 orang mengeluarkan gaya masing-masing 100 n ke kanan dan 50 N ke kiri, besarnya resultan gaya tersebut adalah.... Pakai penjelasan


Diketahui:

F1 = 100 N (ke kanan)

F2 = 50 N (ke kiri)

Ditanya:

R =___?

Jawab:

R = F1 - F2

R = 100 N - 50 N

R = 50 N → ke kanan

Kesimpulan;

Maka, besar resultan gaya tersebut adalah 50 N, karena positif maka arahnya ke kanan.
