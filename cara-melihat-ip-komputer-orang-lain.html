1. Bagaimana cara melihat konfigurasi IP address pada komputer melalui CMD?


Cara Check/Mengetahui IP Address Komputer Melalui Terminal / Command Prompt

1. Klik Start dan lanjutkan dengan klik menu Run.

2. Kemudian ketikan pertintah CMD lalu tekan tombol Enter.

3. Setelah jendela Command Prompt muncul, ketik ipconfig -all dan lanjutkan dengan menekan tombol Enter.

12. Bagaimana caranya agar ip kita tidak sama dengan ip orang lain


Jawaban:

dengan mengubah Alamat IP secara Manual

Penjelasan:

Sebenarnya alamat IP kita ketika terhubung ke suatu jaringan akan automatis terisi (default) dan itu tidak mungkin sama dengan milik orang lain.

tapi kita bisa mengubah alamat IP kita akan tetapi memiliki resiko yaitu tabrakan IP yang terjadi karna ada 2 alamat IP dengan alamat yg sama dalam 1 jaringan.

cara mengubah alamat IP secara manual:

1. masuk ke Control Panel

2. klik Network And Internet

3. klik Open Network and Sharing Center, dan klik device atau modem pada bagian Connections (yg ada logo sinyalnya)

4. di wifi status klik Properties

5. kemudian cari internet Protocol versi 4 (IPv4) dan klik Properties

6. pada internet Protocol versi 4 (IPv4) Properties bagian general, ganti centang menjadi "use the following IP address" kemudian isi IP address, subnetmask, dan default gateway. cara mengetahui Alamat IP kita pada jaringan yaitu melalui CMD ketik ipconfig disitu keliatan Alamat IP, Subnetmask, dan default gateway kita.

23. Apa perintah untuk melihat konfigurasi TCP atau IP komputer


Perintah yang digunakan untuk melihat IP  pada komputer

1. Ipconfig

2. PING

3. Tracert

4. Nslookup

5. Netstat

Pembahasan

TCP/IP (Transmission Control Protocol/Internet Protocol) adalah gabunag protokol yang digunakan untuk mengatur komunikasi data dalam proses tukar menukar data dari komputer satu ke komputer yang lain.

Perintah yang digunakan untuk melihat IP  pada komputer, sebagai berikut:

1. Ipconfig

Ipconfig biasanya dugunakan untuk menampilkan informasi IPaddress pada network adapter. Ipconfig biasanya dijalankan pada CMD (Command Prompt) pada komputer.

2. PING

Ping merupakan dalah satu perintah yang digunakan untuk mengetahui sebuah komputer sudah terhubung dengan komputer yang lain.  

3. Tracert

Tracert merupakan tool yang digunakan untuk mengetahui rute dari sebuah packet dalam jaringan komputer.

4. Nslookup

Nsloopup biasanya digunakan untuk mengetahui IP address dari sebuah domain. Selain itu Nslookup bisa digunakan untuk mendeteksi masalah pada jaringan yang berkaitan dengan Query DNS.

5. Netstat

Netstat digunakan untuk mengetahui dan menampilkan statistik dari komputer dan ke komputer yang sedang digunakan. BIasanya perintah ini dijalankan pada CMD (Command Prompt).
