3. She is a ... supermodel.a. beautiful slim Brazilianb. Brazilian beautiful slimc. slim Brazilian beautifuld. slim beautiful Braziliane. Slim Brazilian beautiful141​


THE ORDER OF ADJECTIVE

She is a beautiful slim Brazilian supermodel ( A )

EXPLANATION

Terdapat aturan dalam penyusunan kata sifat / pendeskripsian secara detail suatu benda. Akronim dari pelafalan nya adalah OSASCOMP. Berikut susunan kata sifat yang harus ditempatkan dalam memodifikasi kata benda :

1. Determiner

=> Awalan, berfungsi sebagai informasi penjelas apakah benda yang dijabarkan termasuk tunggal atau jamak.

Exp  : a, an, the, several, that

2. Opinion

=> Berfungsi sebagai penilaian terhadap benda yang dijabarkan

Exp  : beautiful, silly, hard, difficult, easy

3. Size

=> Berfungsi sebagai pengukuran seberapa besar-kecil benda yang dijabarkan.

Exp : small, large, big, long, short

4. Age

=> Berfungsi sebagai penjabaran seberapa tua-muda benda yang dijabarkan

Exp : old, young, ancient

5. Shape

=> Berfungsi sebagai penjabaran bentuk suatu benda

Exp  : round, triangle, square

6. Color

=> Berfungsi sebagai penjabaran warna suatu benda

Exp : purple, yellow, black

7. Origin/Nationality

=> Berfungsi sebagai penjabaran letak geografis, dimana itu berasal.

Exp : Indonesian, French, Spanish, American

8. Material

=> Berfungsi sebagai penjabaran bahan-bahan benda itu terbuat

Exp  : cotton, paper, metal, rock, sand, cloth

9. Purpose

=> Berfungsi sebagai penjabaran benda tersebut difungsikan/digunakan. Pada umumnya menggunakan present participle verb –ing.

Exp : cooking, roasting

10. Noun/Root

=> Kata benda sebelum  dimodifikasi

Exp : table, chair, book, cat, car, girl, hair

ELABORATION

She is a beautiful slim Brazilian supermodel.

a => determinerbeautiful => opinionslim => sizeBrazilian => Origin/NationalitySupermodel => noun

Maka, berdasarkan penjelasan di atas, jawaban yang tepat sesuai struktur the order of adjective adalah a. beautiful slim Brazilian

17. seorang pedagang kaos membeli 60 kaos dari grosir dengan harga rp40.000 per kaos jika dia berhasil menjual semua kaos tersebut dengan mendapat untung 25% maka harga jual tiap kaos adalah​


Jawab:

Harga awal = 40.000 x 60 kaos

= 2.400.000‬

Jika mendapat keuntungan 25%, maka uang yg didapat:

= 125% x 2.400.000‬

= 125/100 x 2.400.000‬

= 3.000.000‬

Maka harga jual tiap kaos adalah:

= 3.000.000‬ / 60 kaos

= 50.000

Maka harga jual tiap kaos adalah Rp. 50.000.
