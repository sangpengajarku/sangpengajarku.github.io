1. Cara menyimpan pattern di photoshop


1. Buka lembar kerja baru Photoshop.
2. Ukurannya terserah.
3. Tekan CRTl + J (untuk menduplikat layer background).
4. Klik icon Add Layer Style > Pattern Overlay.
5. Klik panah kecil seperti gambar nomor 1.
6. Setelah itu maka akan keluar tampilan default dari pattern Photoshop dan ada panah kecil. Sekarang klik panah kecil tersebut dan pilih save patterns. (Lihat Gambar 2)
7. Tentukan ditempat mana agan ingin menyimpan pattern Photoshop tersebut.

5. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

15. cara mengganti warna background foto menggunakan photoshop, itu gmn ya..


1. Buka dulu photoshopnya. Setelah itu buka pas foto yang akan di ubah background nya, 2.Tekan Ctrl_J pada keyboard sebanyak 2 (dua) kali, sehingga sekarang kita mempunyai 3 layer yaitu layer background, layer 1 dan layer 1 copy., 3.Aktifkan / Klik layer 1.. dan seleksi rambut , 4.Aktifkan Magic Wand Tool (W) lalu klik pada bagian background yang akan diganti, 5.Klik select pada menu - modify - expand lalu masukkan angka 1 (ini akan memperluas bidang seleksi sebesar 1 pix), setelah itu feather (masih pada menu yang sama/klik kanan-feather) yang berfungsi untuk memperhalus tepian bidang seleksi. Masukkan lagi angka 1. Setelah itu tekan delete pada keyboard. Sekarang background nya sudah hilang, 6.Selanjutnya kita akan memasukkan background baru ke pas foto. Buat layer baru dengan menekan create new layer pada pallette layers. Posisikan layer baru tersebut tepat berada diatas layer background dengan menekan ctrl_[ pada keyboard, 7.Masih layer 2 aktif, tekan G pada keyboard (gradien tool) dan pilih warna background yang diinginkan (lebih cepatnya pilih di pallete swatches/window - swatches) lalu klik pada foto, 8.Gabung semua layer dengan mengklik pada sudut atas pallette layers - flatten image. Kemudian save atau pindahkan ke dokumen baru untuk di cetak.

22. Bagaimanakah cara memunculkan tampilan penyimpanan hasil scanning dengan Adobe Photoshop?


Cara memunculkan tampilan penyimpanan hasil scanning dengan Adobe Photoshop adalah menggunakan menu Open pada Adobe Photoshop. . Hasil scanning merupakan file berupa gambar. Jadi kita bisa menggunakan menu open pada Adobe Photoshop.

Pembahasan

Adobe Photoshop merupakan salah satu aplikasi yang berguna untuk memodifikasi suatu file gambar atau foto. Beberapa format gambar yang bisa dimodifikasi menggunakan aplikasi Adobe Photoshop adalah sebagai berikut

Bitmap (BMP, RLE, DIB)Camera RAW (TIF, CRW, NEF, RAF, ORF, MRW, DCR, RAW, dll)Cineon (CIN, SDPX, DPC, FIDO)Campuserve GIF (GIF)Photoshop Format (EPS, PDF, PDP, RAW, PSDX, PSD, dll)JPEG (JPG, JPEG, JPE)Generic EPS (AI3, AI4, AI5, dll)Portable Bitmap (PBM, PGM, PPM, dll)IFF Format (IFF, TDI)TIFFJPEG Stereo (JPS)JPEG 2000 (JPF, JPX, J2C, J2K, dll)Multi Picture Format (MPO)PICT File (PICT)OpenEXR (EXR)PCXPixar (PXR)Radiance (HDR, RGBE, XYZE)Scitex CT (SCT)Targa (TGA, VDA, ICB, VST

Dengan melihat format gambar yang sudah disupport oleh Adobe Photoshop, maka kita akan semakin mudah dalam melakukan pengolahan gambar atau foto yang kita miliki.

25. cara cepat menghapus foto yang tidak kita inginkan di photoshop??​


Jawaban:

1. Buka dan buat duplikat gambar untuk mencegah kerusakan permanen pada foto. Anda bisa membuka gambar di Photoshop dan memilih "File" (berkas) → "Save as Copy" (simpan sebagai salinan), atau mengeklik "Layer" (lapisan) → "Duplicate Layer" (duplikasi lapisan) untuk membuat versi cadangan dari foto original seandainya Anda membuat kesalahan

2. Gunakan alat Quick Selection (seleksi cepat) untuk memilih objek yang ingin disingkirkan. Alat ini memiliki ikon kuas dengan garis bertitik-titik di dekat bulunya, dan seharusnya berada di alat keempat dari atas bilah alat (toolbar). Kalau Anda tidak menemukannya, klik dan tahan tombol Magic Wand (tongkat ajaib) dan alat ini seharusnya muncul. Kalau Anda ingin menghapus objek tanpa menutupinya, tekan Delete untuk menghapusnya.

3. Gunakan alat seleksi lain yang lebih akurat jika objek terlalu kompleks bagi Quick Selection. Anda memiliki banyak opsi yang tersedia untuk memilih objek, bergantung tingkat akurasi yang diinginkan. Jika latar belakang di belakang objek cukup sederhana, dan objek terkait lumayan mencolok (warnanya berbeda, garis-garisnya mudah dilihat, dll.)

27. ada berapa cara proses penyimpanan file pada photoshop?sebutkan dan jelaskan!


3cara Sobat Yaitu:

1.Save 
   Kalau yang satu ini sih, saya yakin semuanya sudah pada tau ya.Yap,cara yang pertama ini kita menggunakan menu Save yang  berada di menu File->Save atau bisa juga menggunakan shortcut Ctrl+S. Sayangnya, jika sobat menggunakan menu ini sobat hanya bisa melakukan proses penyimpanan dengan format yang sama (contoh : jika sobat membuka PSD, maka akan disimpan menjadi PSD juga).
  
2.Save As
Lanjut ke cara yang kedua yaitu melalui menu Save As yang berada di File->Save As atau bisa menggunakan shortcut Shift+Ctrl+S. Perintah save as ini digunakan untuk menyimpan file dengan format, nama, dan lainnya berbeda dengan format aslinya. (contoh : jika sobat membuka PSD, maka sobat bisa mengubah formatnya menjadi JPEG dengan cara ini).
 
3.Export
Nah, mungkin sobat hanya tahu dengan kedua cara diatas ya. Menu export ini bisa sobat gunakan juga loh untuk melakukan perintah menyimpan. Caranya yaitu melalui menu File->Export dan pilih ingin export yang apa. Jika untuk situs web atau blog pilih yang Save For Web. Keunggulan dalam menggunakan menu Export ini, sobat bisa dengan bebas mengatur warna, kompresi, dan lainnya lengkap.
