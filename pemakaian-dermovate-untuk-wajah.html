19. Untuk mengruangi pengeluaran, sebuah perusahaam memutuskan untuk mengurangi pemakaian bensin sebanyak 15%. Jika pemakaian bensin saat ini sebanyak 580 liter per bulan, hitunglah pengurangan pemakaian bensin per bulannya. Tentukan juga pemakaian bensin bulanan perusahaan itu sekarang


pengurangan = 15%

pemakaian = 580 L/bln

pengurangan bensin perbulan = 15/100 × 580 L = 87 L

Pemakaian Bensin sekarang = pemakaian bensin awal - pengurangan = 580 L - 87 L = 493 L
