5. Ajeng ingin membeli buku keuangan negara, bahasa inggris,manajemen, statistika, dan ekonomi untuk semster baru ini. Diantara buku buku tersebut, harga termurah adalah Rp 10.000, informasi lengkap harga buku buku tersebut sebagai berikut : -harga buku bahasa inggris setengah harga buku statistika -Harga buku ekonomi dikurangi harga buku statistika sama dengan harga buku manajemen -harga buku ekonomi lebih mahal tiga kali lipat dari buku statistika, tetapi masih lebih murah dibandingkan dengan buku keuangan negara. urutkan buku dari yang paling mahal ke yang paling murah


Pembahasan.
A : harga buku bahasa inggris
B : harga buku statistik
C : harga buku ekonomi
D : harga buku manajemen
E : harga buku keuangan negara


1. A = 1/2 B
2. C - B = D
3. C = 3B
4. C < E

Kesimpulan :
D = 2B
C = 3B
A = 1/2B
E > 3B

Paling kecil adalah A
Jika A = 10.000
Maka
B = 2 x 10.000 = 20.000
C = 3 x 20.000 = 60.000
D = 2 x 20.000 = 40.000
E > 60.000

Urutannya
E > 60.000
C = 60.000
D = 40.000
B = 20.000
A = 10.000

12. Teknik layout yang menerapkan teknik layaknya lingkaran yang terbagi menjadi empat yaknia. frame layoutb. bleed layoutc. quadran layoutd. brace layoute. copy heavy layout​


Jawaban:

C. Quadran Layout

Penjelasan:

a. frame layout

Suatu tampilan dimana border/bingkai/framenya membentuk suatu naratif (suatu cerita)

b. bleed layout

Sajian iklan di mana sekeliling bidang menggunakan frame (seolah-olah dipotong pinggirnya).

c. quadran layout

Bentuk tampilan iklan yang gambarnya dibagi menjadi empat bagian dengan volume/isi yang berbeda.

d. brace layout

Unsur-unsur dalam tata letak iklan membentuk letter L (L-Shape). Posisi bentuk L nya bisa tebalik, dan dimuka bentuk L tersebut dibiarkan kosong.

e. copy heavy layout

Tata letaknya mengutamakan pada bentuk copy writing (naskah iklan) atau dengan kata lain komposisi lay out nya didominasi oleh penyajian teks (copy)
