28. Sebuah galeri seni akan membuat dua jenis lukisan yaitu natural dan abstrak. Persediaan cat sebanyak 80 set cat minyak dan 60 set cat jenis lain. Seorang pelukis menghitung jenis lukisan natural memerlukan 2 set cat minyak dan 1 set cat jenis lain, sedangkan lukisan abstrak memerlukan masing-masing 1 set cat .Banyak lukisan abstrak maksimum yang dapat dibuat adalah ….


Sebuah galeri seni akan membuat dua jenis lukisan yaitu natural dan abstrak. Persediaan cat sebanyak 80 set cat minyak dan 60 set cat jenis lain. Seorang pelukis menghitung jenis lukisan natural memerlukan 2 set cat minyak dan 1 set cat jenis lain, sedangkan lukisan abstrak memerlukan masing-masing 1 set cat. Banyak lukisan abstrak maksimum yang dapat dibuat adalah 60 lukisan.

Penjelasan dengan langkah-langkah:

Soal diatas bisa diselesaikan dengan sistem persamaan linear dua variabel dengan lukisan natural sebagai variabel x dan lukisan abstrak sebagai variabel y. Untuk membuat sebuah lukisan natural diperlukan 2 set cat minyak dan untuk membuat sebuah lukisan abstrak diperlukan 1 set cat minyak, jumlah cat minyak yang tersedia adalah 80 set, maka persamaannya adalah:

2x + y = 80

Untuk membuat sebuah lukisan natural diperlukan 1 set cat jenis lain dan untuk membuat sebuah lukisan abstrak diperlukan 1 set cat jenis lain, jumlah cat jenis lain yang tersedia adalah 60 set, maka persamaannya adalah:

x + y = 60

Model matematika untuk soal di atas adalah:

2x + y = 80

x + y = 60

x , y ≥ 0

Setelah dibuat model matematikanya kemudian dibuat grafik dari model matematika tersebut dan dicari titik potong grafik tersebut dengan sumbu x dan sumbu y. Untuk persamaan pertama titik potong dengan sumbu x dan sumbu y adalah:

2x + y = 80 → x = 0 → 0 + y = 80 → y = 80

Titik potong = ( 0 , 80 )

2x + y = 80 → y = 0 → 2x + 0 = 80 → x = 40

Titik potong = ( 40 , 0 )

Untuk persamaan kedua titik potong dengan sumbu x dan sumbu y adalah:

x + y = 60 → x = 0 → 0 + y = 60 → y = 60

Titik potong = ( 0 , 60 )

x + y = 60 → y = 0 → x + 0 = 60 → x = 60

Titik potong = ( 60 , 0 )

Daerah penyelesaian dari soal diatas adalah daerah antara titik ( 0 , 0 ), ( 40 , 0 ), ( 0 , 60 ) dan titik potong kedua garis. Untuk mencari titik potong kedua garis digunakan cara eliminasi dan substitusi dari nilai x dan y dari kedua persamaan diatas. Langkah pertama eliminasi variabel y dari kedua persamaan di atas.

2x + y = 80

 x + y = 60    -

x = 20

Setelah diketahui nilai x, substitusikan nilai x ke persamaan x + y = 60

20 + y = 60

y = 60 - 20

y = 40

Jadi titik potong dari kedua garis tersebut adalah ( 20 , 40 ). Setelah itu dilakukan pengujian dari ketiga titik potong tersebut

( 40 , 0 ) → 40 lukisan natural

( 0 , 60 ) → 60 lukisan abstrak

( 20 , 40 ) → 20 buah lukisan natural dan 40 buah lukisan abstrak

Jadi jika ingin membuat lukisan abstrak paling banyak, persediaan cat yang ada di galeri seni tersebut dapat digunakan untuk membuat 60 buah lukisan abstrak.
