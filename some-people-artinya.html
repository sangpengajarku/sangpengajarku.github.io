Jawaban:

it's very dirt ma'am some people don't care artinya adalah Itu sangat kotor, Bu. Beberapa orang tidak peduli.

PEMBAHASAN

Present tense adalah tenses untuk menyatakan peristiwa di masa sekarang (Present) dan kegiatan berulang/rutinitas, kebiasaan/habitual serta Kebenaran umum/fakta.

• Present Tense bentuk Nominal karena tidak ada kata kerja

(+) S + To be (is/am/are) + O

(-) S + To be (is/am/are) + not + O

(+) to be (is/am/are) + S + O?

• Present Tense bentuk Verbal karena ada kata kerja yaitu care

(+) S + V1 (s/es)

(-) S + do/does + not + V1

(?) do/does + S + V1

Kesimpulan

Berdasarkan rumus:

1. Nominal Sentence

(+) S + ToBe (is/am/are) + O

= it + is + very dirty, ma'am

(itu sangat kotor, Bu.)

(-) S + do/does + not + V1

= Some people + don't + care

(Beberapa orang tidak peduli)
