9. Pada sebuah restoran cepat saji, ketika harga paket ayam 24.000 maka restoran tersebut menjual sebanyak 100 paket tiap hari. Tetapi ketika harga paket ayam naik menjadi 30.000 maka restoran tersebut dapat menjual paket ayam tersebut sebanyak 50 paket Perhari. Jika harga paket ayam 27.000 tentukan berapa jumlah penjualan paket ayam ter sebut perharinya!


Materi ini termasuk aplikasi dari persamaan garis lurus yaitu fungsi permintaan.

Fungsi Permintaan adalah fungsi yang menunjukkan hubungan antara jumlah barang yang diminta oleh konsumen (Q) dengan harganya (P).

Jika fungsi permintaan dianggap mengikuti garis lurus maka persamaan fungsi permintaan dapat dicari menggunakan persamaan garis lurus yang dua titiknya diketahui, sebagai berikut:

[tex]\boxed {\frac{P-P_1}{P_2-P_1} =\frac{Q-Q_1}{Q_2-Q_1}}[/tex]


PEMBAHASAN

DIKETAHUI

HARGA         PAKET TERJUAL

24.000                   100

30.000                    50


DITANYA

Berapa paket terjual jika harganya 27.000?


PENYELESAIAN

Mencari persamaan fungsi permintaan yang melalui titik (24.000, 100) dan (30.000, 50)

[tex]\frac{P-24.000}{30.000-24.000} =\frac{Q-100}{50-100}[/tex]

[tex]\frac{P-24.000}{6.000} =\frac{Q-100}{-50}[/tex]

[tex]\frac{P-24.000}{-120}[/tex] = Q - 100

P - 24.000 = -120(Q - 100)

P - 24.000 = -120Q + 12.000

P = -120Q + 12.000 + 24.000

P = -120Q + 36.000


Mencari jumlah paket yang terjual per harinya jika harga = 27.000

P = -120Q + 36.000

27.000 = -120Q + 36.000

27.000 - 36.000 = -120Q

-9.000 = -120Q

Q = 75


KESIMPULAN

Pada sebuah restoran cepat saji, ketika harga paket ayam 24.000 maka restoran tersebut menjual sebanyak 100 paket tiap hari. Tetapi ketika harga paket ayam naik menjadi 30.000 maka restoran tersebut dapat menjual paket ayam tersebut sebanyak 50 paket per hari. Jika harga paket ayam 27.000, jumlah penjualan ayam tersebut per harinya adalah 75 paket.
