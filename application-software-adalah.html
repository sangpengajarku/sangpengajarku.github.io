18. Yang termasuk application software yaitu … .


Yang termasuk application software yakni :

Microsoft Office atau software pengolah kata .  Adobe Photoshop .  Adobe Dreamweaver .  Adobe Acrobat .  Google Chrome .  WinRAR .  

Application Software dapat disebut sebagai jenis software yang dirancang untuk menyelesaikan tugas-tugas spesifik dalam operating system. Beberapa dari tugas ini adalah termasuk menangani dokumen, menangani perhitungan, mengedit gambar, mengedit video, dan yang lainnya.  

Pembahasan

Pemakaian dari Application software yakni melakukan tugas tertentu tergantung pada bagaimana ia dirancang, jadi tidak dapat melakukan tugas lain yang berada di luar cakupannya.

Application software diinstal hanya pada saat pengguna membutuhkannya. Contohnya, seseorang hanya akan menginstal software edit foto ketika ingin mengedit beberapa foto.

Waktu eksekusi Application software hanya akan berjalan jika diminta, Jadi seseorang akan memulai pengolah kata ketika perlu membuka atau menyiapkan beberapa dokumen.

Untuk application software, pengguna akan mempunyai interaksi langsung dengan software. Bahkan, pengguna dapat mengontrol cara software beroperasi.

Application software ini tidak dapat berjalan secara independen, Jadi mereka tergantung pada kondisi yang disediakan oleh operating system.

Pemrograman application software sendiri tidak rumit. Seorang programmer hanya perlu mempunyai pengetahuan tentang system software yang mendasarinya serta bahasa tingkat tinggi.

27. Berdasarkan fungsinya, software dikategorikan sebagai .... a. Application software dan System software b. System software dan Compilers c. System software dan Compilers d. System software dan assemblers


Berdasarkan fungsinya, software yang digunakan dalam komputer dikategorikan menjadi Application software dan System software

Pembahasan:

Dalam penggunaan komputer, kita membutuhkan software atau perangkat lunak agar bisa mengoperasikan komputer. Tanpa adanya perangkat lunak, maka suatu komputer tidak bisa digunakan secara maksimal. Dilihat dari fungsinya, software secara garis besar dibedakan menjadi dua macam yaitu application software dan system software.

Application software merupakan perangkat lunak yang memiliki fungsi dan tujuan khusus. Kekhususannya ini membuat perangkat lunak ini efektif dalam mengerjakan suatu tugas secara spesifik, namun secara general tidak bisa. Fungsi spesifik antara lain untuk pengolahan kata bisa menggunakan microsoft word, untuk pengolahan gambar bisa menggunaka Adobe photoshop, dan pengolah angka bisa menggunakan Microsoft exel.

System software merupakan perangkat lunak yang membuat suatu komputer bisa bekerja secara umum seperti menyimpan file, mengorganisit file, hingga menggandakan suatu file. System software juga menjadi tempat untuk instalasi dari application software. Contoh dari system software antara lain Microsoft Windows 10, Ubuntu 19.04, Machintos.
