1)  diketahui :

   p = 20 m

   l = 10 cm = 0,1 m

   t = 15 cm = 0,15 m

   P = 300 N/m²

   ditanya :

   F

   jawab :

   F  =  P . A

       =  P . ( p x l )

       =  300 x ( 20 x 0,1 )

       =  600 N


    F  =  P . ( l x t )

        =  300 x ( 0,1 x 0,15 )

        =  4,5 N


     F  =  P . ( p x t )

         =  300 x ( 20 x 0,15 )

          = 900 N


2)  diketahui :

    F = 800 N

    A = 4 cm²  = 4.10⁻⁴ m²

    ditanya :

    P

    jawab :

    P  =  F / A

        =  800 / 4.10⁻⁴

        =  2.000.000 Pa

        =  2.000 kPa
