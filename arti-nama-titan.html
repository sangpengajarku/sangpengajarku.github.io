22. barisan dan deret bilanganTitan berkembang biak dengan cara membelah diri. Mula-mula terdapat 2 titan dan membelah menjadi 5 kali setiap 2 menit. Banyak titan selama 1,5 jam adalah ... titan.​


~Barisan dan Deret

___________________

DiketahuiTitan berkembang biak dengan cara membelah diri. Mula-mula terdapat 2 titan ( a = 2 ), dan membelah menjadi 5 kali setiap 2 menit ( r = 5 )

Ditanya

Banyak titan selama 1,5 jam

» Penyelesaian

• Menentukan banyak pembelahan ( n )

n = 1,5 jam ÷ 2 menit

n = (1,5 × 60) menit ÷ 2 menit

n = 90 menit ÷ 2 menit

n = 45 menit

• Menentukan banyak titan selama 1,5 jam ( Un )

Gunakan rumus pertumbuhan

Maka:

Un = arⁿ

U45 = 2 × 5⁴⁵ titan

Penjelasan dengan langkah-langkah:

Barisan dan Deret Geometry

Un = arⁿ-¹

Sn = a(rⁿ - 1)/(r - 1) , r ≥ 0

S∞ = a / (1 - r)

Diketahui :

Mula - mula terdapat 2 Titan → membelah menjadi 5 kali setiap 2 menit.

Ditanyakan :

Banyak Titan selama 1,5 jam ?

Jawab :

• Waktu Titan membelah setiap menit

misalkan n (waktu) :

n = 1,5 jam / 2 menit

n = 90 menit / 2 menit

n = 45 menit

a = suku pertama = 2

r = rasio = 5

sehingga, banyak Titan :

B = a . rⁿ

B = 2 . 5⁴⁵

B = 2 . 5⁴⁵ ...

Jadi, banyak nya Titan adalah 5 .2⁴⁵
