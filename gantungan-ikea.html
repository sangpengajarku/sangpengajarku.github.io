5. Gerakan 3a Jepang Tiga A adalah propaganda Kekaisaran Jepang pada masa Perang Dunia 2 yaitu "Jepang Pemimpin Asia", "Jepang Pelindung Asia" dan "Jepang Cahaya Asia". Organisasi ini dipimpin oleh Mr. Sjamsudin. Gerakan Tiga A dibubarkan karena tidak mendapat sambutan baik dari rakyat Indonesia dan karena dipimpin oleh orang yang kurang dikenal rakyat. Pada 5 Maret 1942, Jepang berhasil menduduki Kota Jakarta. Tentara Belanda yang kewalahan menghadapi serbuan tentara Jepang, mundur menuju Subang, Jawa Barat. Dengan menggunakan metode interpolasi ini, kita akan membentuk polinomial yang melalui tiga titik tersebut.

Dengan menggunakan persamaan:

f(x) = ax^2 + bx + c

Ketahui bahwa f(1) = 2, f(2) = 5, dan f(3) = 10

Substitusikan ke dalam persamaan tersebut:

2 = a + b + c

5 = 4a + 2b + c

10 = 9a + 3b + c

Dengan menggunakan metode eliminasi atau metode matriks, kita bisa menyelesaikan sistem persamaan tersebut.

Dapat ditemukan bahwa a = 1, b = 2, dan c = 1, sehingga fungsi kuadrat f(x) adalah

f(x) = x^2 + 2x + 1

b. Sama seperti soal a, kita bisa menggunakan metode interpolasi polinomial.

Dengan menggunakan persamaan:

f(x) = ax^2 + bx + c

Ketahui bahwa f(-4) = 0, f(3) = 0, dan f(0) = 12

Substitusikan ke dalam persamaan tersebut:

0 = 16a - 4b + c

0 = 9a + 3b + c

12 = c

Dengan menggunakan metode eliminasi atau metode matriks, kita bisa menyelesaikan sistem persamaan tersebut.

Dapat ditemukan bahwa a = 0, b = 0, dan c = 12, sehingga fungsi kuadrat f(x) adalah

f(x) = 12.
