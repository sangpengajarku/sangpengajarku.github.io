6. 7. Gambarkanlah grafik fungsi trigonometri :a. Y = sin xb. Y = cos xc. Y = tan x#tolong​


Gambarkanlah grafik fungsi trigonometri

a. y = sin x b. y = cos x c. y = tan x

Untuk menggambar grafik dari trigonometri kita masukkan untuk x sudut istimewa seperti 0⁰, 30⁰, 45⁰, 60⁰, 90⁰, …, 360⁰ (kita buat x kelipatan dari 30⁰). Lalu kita buat tabelnya dan hubungkan titik-titik tersebut dalam koordinat kartesius agar terbentuk grafiknya

Pembahasan

a. y = sin x

x = 0⁰ ⇒ y = sin 0⁰ = 0 x = 30⁰ ⇒ y = sin 30⁰ = ½  x = 45⁰ ⇒ y = sin 45⁰ = ½ √2 x = 60⁰ ⇒ y = sin 60⁰ = ½ √3 x = 90⁰ ⇒ y = sin 90⁰ = 1 x = 120⁰ ⇒ y = sin 120⁰ = ½ √3 x = 135⁰ ⇒ y = sin 135⁰ = ½ √2  x = 150⁰ ⇒ y = sin 150⁰ = ½  x = 180⁰ ⇒ y = sin 180⁰ = 0 x = 210⁰ ⇒ y = sin 210⁰ = – ½  x = 225⁰ ⇒ y = sin 225⁰ = – ½ √2 x = 240⁰ ⇒ y = sin 240⁰ = – ½ √3 x = 270⁰ ⇒ y = sin 270⁰ = –1 x = 300⁰ ⇒ y = sin 300⁰ = – ½ √3 x = 315⁰ ⇒ y = sin 315⁰ = – ½ √2 x = 330⁰ ⇒ y = sin 330⁰ = – ½  x = 360⁰ ⇒ y = sin 360⁰ = 0

b. y = cos x

x = 0⁰ ⇒ y = cos 0⁰ = 1 x = 30⁰ ⇒ y = cos 30⁰ = ½ √3 x = 45⁰ ⇒ y = cos 45⁰ = ½ √2 x = 60⁰ ⇒ y = cos 60⁰ = ½  x = 90⁰ ⇒ y = cos 90⁰ = 0 x = 120⁰ ⇒ y = cos 120⁰ = – ½  x = 135⁰ ⇒ y = cos 135⁰ = – ½ √2 x = 150⁰ ⇒ y = cos 150⁰ = – ½ √3 x = 180⁰ ⇒ y = cos 180⁰ = –1 x = 210⁰ ⇒ y = cos 210⁰ = – ½ √3 x = 225⁰ ⇒ y = cos 225⁰ = – ½ √2 x = 240⁰ ⇒ y = cos 240⁰ = – ½  x = 270⁰ ⇒ y = cos 270⁰ = 0 x = 300⁰ ⇒ y = cos 300⁰ = ½  x = 315⁰ ⇒ y = cos 315⁰ = ½ √2 x = 330⁰ ⇒ y = sin 330⁰ = ½ √3 x = 360⁰ ⇒ y = cos 360⁰ = 1

c. y = tan x

x = 0⁰ ⇒ y = tan 0⁰ = 0 x = 30⁰ ⇒ y = tan 30⁰ = ⅓ √3 x = 45⁰ ⇒ y = tan 45⁰ = 1 x = 60⁰ ⇒ y = tan 60⁰ = √3 x = 90⁰ ⇒ y = tan 90⁰ = tak terdefinisi x = 120⁰ ⇒ y = tan 120⁰ = –√3 x = 135⁰ ⇒ y = tan 135⁰ = –1 x = 150⁰ ⇒ y = tan 150⁰ = –⅓ √3 x = 180⁰ ⇒ y = tan 180⁰ = 0 x = 210⁰ ⇒ y = tan 210⁰ = ⅓ √3 x = 225⁰ ⇒ y = tan 225⁰ = 1 x = 240⁰ ⇒ y = tan 240⁰ = √3 x = 270⁰ ⇒ y = tan 270⁰ = tak terdefinsi x = 300⁰ ⇒ y = tan 300⁰ = –√3 x = 315⁰ ⇒ y = tan 315⁰ = –1 x = 330⁰ ⇒ y = tan 330⁰ = – ⅓ √3 x = 360⁰ ⇒ y = tan 360⁰ = 0

Hubungkan titik-titik yang diperoleh, maka dapatlah grafik dari y = sin x, y = cos x dan y = tan x

Tabel dan grafiknya dapat dilihat di lampiran

18. gambarlah grafik fungsi trigonometri y= tan (x-15°)​


Grafik fungsi trigonometri y = tan ( x - 15° ) bisa di lihat pada gambar di lampiran​

PEMBAHASAN

Sebelumnya kita mesti mengetahui prinsip dasar dari trigonometri.

Jikalau di ketahui segitiga siku-siku maka bisa di buat :

sinus sebuah sudut = sisi di depan sudut / sisi miring

kosinus sebuah sudut = sisi di samping sudut / sisi miring

tangen sebuah sudut = sisi di depan sudut / sisi di samping sudut

Beberapa identitas trigonometri yang sering kita jumpai adalah sebagai berikut ini :

cosec A = 1 / sin A

sec A = 1 / cos A

cot A = 1 / tan A

tan A = sin A /  cos A

cos²A + sin² A = 1

Marilah kita sekarang mencoba menyelesaikan soalnya.

Grafik fungsi Trigonometri y = tan x dapat digambarkan dengan mencari beberapa titik bantu tergantung daerah asal yang ingin di gambarkan :

x = 0° → y = tan 0° = 0

x = 30° → y = tan 30° = 1/√3

x = 45° → y = tan 45° = 1

x = 60° → y = tan 60° = √3

x = 90° → y = tan 90° = ∞

Setelah fungsi trigonometri y = tan x° , maka fungsi trigonometri y = tan ( x° - 15° ) dapat di gambarkan dengan cara menggeser ke kanan grafik y = tan x sebesar 15°

Untuk lebih jelasnya bisa di lihat pada gambar di lampiran.
