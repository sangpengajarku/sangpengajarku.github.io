_Semoga Membantu_

Operasi Hitung Aljabar

Soal: Jumlah dari x - 4y + 5 dan x + 5y - 3 adalah...

Penyelesaian:

(x - 4y + 5) + (x + 5y - 3)

= (x + x) + (-4y + 5y) + (5 + (-3))

= 2x + y + 2

Maka, jumlah dari x - 4y + 5 dan x + 5y - 3 adalah 2x + y + 2 (D)
