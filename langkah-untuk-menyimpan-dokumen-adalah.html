6. tuliskan langkah langkah untuk menyimpan dokumen pada Microsoft word 2007 ​


Jawaban:

Cara Menyimpan Dokumen Baru

1. Klik Office Button

2. Klik Save As

3. Buka folder tempat menyimpan dokumen sesuai keinginan Anda

4. Ketik nama file dokumen di kotak File name kemudian klik Save

Cara Menyimpan Dokumen Lama

1. Klik Office Button

2. Klik Save

Menyimpan Dokumen Menggunakan Kombinas Tombol Keyboard

Rekan pembaca dapat menggunakan tombol kombinasi untuk menyimpan file Microsoft Word 2007.

1. Tekan dan tahan CTRL

2. Tanpa melepas tombol CTRL, tekan huruf S

Bila dokumen baru akan tampil jendela explorer kecil untuk membuka folder lokasi anda menyimpan file dan beri nama dokumen.

Jika dokumen lama yang telah anda save as sebelumnya, tidak terlihat prosesnya. Tetapi akan terasa bila anda menutup program Microsoft Word 2007. Tidak akan muncul peringatan untuk menyimpan hasil ubahan

7. jelaskan langkah-langkah untuk menyimpan dokumen microsoft excel!


1. Klik Office Button, lalu pilih menu Save atau Save As.
– Save : menyimpan dokumen baru atau dokumen lama dengan nama yang sama.
– Save As : menyimpan dokumen baru yang belum pernah disimpan atau dokumen lama dengan nama baru.
2. Jika kamu menyorot ikon Save As maka akan muncul kotak pilihan format penyimpanan data. Pilih Excel Workbook atau kamu dapat langsung mengklik ikon Save As, maka akan muncul kotak dialog Save As.
3. Tentukan folder tempat penyimpanan dokumen kamu pada kotak isian Save In dengan acara klik tombol panah kecil di ujung lalu pilih folder yang diinginkan.
4. Ketikkan nama dokumen kamu pada kotak File Name.
5. Isilah kotak isian Save as type dengan Excel Workbook.
6. Klik tombol Save

23. Langkah untuk menyimpan data atau dokumen, kita gunakan

Klik File > Pilih Save > Pilih Letak penyimpanannya dmana > Klik Save

Jawaban:

2.Klik Simpan Sebagai.

3.Pilih lokasi file, seperti OneDrive atau PC Ini untuk menyimpan file Anda.
