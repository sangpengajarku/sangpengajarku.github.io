Harga beli sandal pria adalah Rp. 20.000,00 dan sandal wanita harga belinya Rp. 16.000,00 perpasang. Mengingat kapasitas kiosnya, ia hanya dapat membeli sebanyak-banyaknya 450 pasang sandal. Jika harga jual dari sandal pria dan wanita berturut-turut adalah Rp. 26.000,00 dan Rp. 21.000,00 per pasang. banyaknya keuntungan maksimum yang bisa diraih pedagang tersebut adalah...​


Jawab: dari ilustrasi soal dapat kita rangkum

yang diketahui:

modal = 8.000.000

harga sepatu pria = 20.000

harga sepatu wanita = 16.000

keuntungan sepatu pria = 6.000

keuntungan sepatu wanita = 5.000

kapasitas kios = 450 pasang

pertanyaan :

a) model matematika

b) keuntungan maksimum

penyelesaian

jika sepatu pria = x dan sepatu wanita = y

model matematika yang dapat dibuat

1) 20.000x + 16.000y ≤ 8.000.000 kita kecilkan dengan membagi semua ruas dengan angka 4.000, sehingga model matematikanya

5x + 4y ≤ 2.000

2) x + y ≤ 450

3) x ≥ 0

4) y ≥ 0

5) nilai obyektif → f(x,y) = 6.000x + 5.000y

penyelesaian agar mengetahui keuntungan maksimum

untuk pertidaksamaan 1

5x + 4y ≤ 2000

kita ambil tanda = untuk mengetahui titik potong terhadap sumbu x dan sumbu y nya

5x + 4y = 2000

memotong sumbu x, jika y = 0

5x + 4y = 2000

5x + 4(0) = 2000

5x + 0 = 2000

5x = 2000

x = 2000/5

x = 400

memotong sumbu y, jika x = 0

5x + 4y = 2000

5(0) + 4y = 2000

0 + 4y = 2000

4y = 2000

y = 2000/4

y = 500

untuk pertidaksamaan 2

x + y ≤ 450

kita ambil tanda = untuk mengetahui titik potong terhadap masing" sumbu

x + y = 450

memotong sumbu x, jika y = 0

x + y = 450

x + 0 = 450

x = 450

memotong sumbu y, jika x = 0

x + y = 450

0 + y = 450

y = 450

untuk mengetahui titik potong kedua persamaan kita eliminasi kedua persamaan

5x + 4y = 2000 → kali 1 → 5x + 4y = 2000

x + y = 450        → kali 4 → 4x + 4y = 1800  -

                                           x  = 200

subsitusikan ke persamaan x + y = 450

200 + y = 450

y = 450 - 200

y = 250

dari hasil hitungan tadi kita buat gambarnya , perhatikan lampiran

dari gambar kita ketajui ada 3 titik uji,

kita subsitusikan 3 titik uji pada nilai obyektif f(x,y)

f(x,y) = 6.000x + 5.000y

f(0,450) → 6000(0) + 5000(450) = 2.250.000

f(200,250) → 6000(200) + 5000(250) = 2.450.000

f(400,0) → 6000(400) + 5000(0) = 2.400.000

jadi keuntungan maksimum didapat ketika pedagang membeli 200 sepatu pria dan 250 sepatu wanita, dengan besar keuntungan 2.450.000

26. 1.seorang pedagang sandal membeli selusin sandal dengan harga Rp120.000,00. Jika sandal-sandal itu dijual dengan harga Rp12.500,00 per pasang,keuntungan yang didapat pedagang tersebut jika semua sandal terjual adalah...​


Jawab:

satu lusin = 12

12 pasang sendal

120.000 : 12 = 10.000

dijual 12.500

kejual semua =

12.500 x 12 = 150.000

untung = 150.000-120.000

            = 30.000

PENDAHULUAN

Aritmatika sosial adalah materi matematika yang berhubungan dengan kehidupan sehari", seperti menghitung harga jual, harga beli, untung, rugi, diskon, rabat, bruto, netto, tara, tangungan awal, tabungan ahir, bunga dll.

PENYELESAIANLangkah pertama

Menentukan harga jual 1 lusin sendal

HJ = HJ/buah × 12

HJ = Rp. 12.500,00 × 12

HJ = Rp. 150.000,00

Langkah kedua

Menentukan jumlah untung pedagang tersebut

Jumlah untung = HJ - HB

Jumlah untung = Rp. 150.000,00 - Rp. 120.000,00

Jumlah untung = Rp. 30.000,00.

KESIMPULAN

Jadi, pedagang tersebut mendapatkan keuntungan sebesar Rp. 30.000,00.

4/7/2018For British English,  day followed by month followed by year, the 13th day of the month April, year 2014, might be written in full (in order of complexity):
Example :

13 April
13 April 2014
13th April 2014
the 13th of April 2014
the 13th of April, 2014,

In American English, the month comes before the day, which means you cannot use of and rarely use ordinal numbers (adding st, nd, rd, th). Commas should also be used to separate the day and year, and again the name of the day should come at the beginning. The date should therefore be written:
Example :

April 13
April 13, 2014
Sunday, April 13, 2014
