7. Gimana klo saya lupa password laptop saya,dan bagaimana cara mereset ulang laptop tanpa menggunakan software dan tidak di install ulang, laptop saya Windows 10?


Jika laptopmu terhubung dengan akun Microsoft, bisa direset password akun Microsoftnya di komputer/gadget lain dengan cara:
- Buka https://account.live.com/password/reset di komputer/gadget lain,
- Kemudian pilih pilihan "I've forgotten my password", lalu masukkan email akun Microsoftmu tersebut, dan ikuti langkah2 selanjutnya (bisa berupa verifikasi email cadangan, verifikasi nomor hp, dsb) sampai bisa mengganti password
- Kalau sudah, hubungkan laptop yang lupa password tersebut ke internet dan masukkan password yang baru. Kalau cara diatas tidak bisa, terpaksa harus pakai cara dibawah.

Jika laptopmu tidak terhubung dengan akun Microsoft/memakai akun lokal, laptopmu harus direset (dengan syarat, data yg tersimpan di data C akan terhapus semua), caranya:
- Hidupkan laptopmu dan tunggu sampai lockscreen muncul
- Jika sudah sampai lockscreen, klik ikon Power di pojok kiri layar, lalu tekan & tahan terus tombol Shift dan klik pilihan Restart (jangan dilepas tombol Shiftnya), dan tunggu sampai laptop restart
- Kalau sudah direstart (boleh dilepas tombol Shiftnya), pilih Boot Option, lalu pilih Troubleshoot, selanjutnya pilih Reset This PC, lalu pilih Remove Everything, dan tunggu hingga laptopmu restart sendiri, dan Windows yang ada di laptopmu akan seperti baru lagi

12. Bisa bantu?Bagaimana cara menyambungkan wifi yang menggunakan password ke laptop windows 7?​


Jawaban:

Windows 7 Buka Start Menu dan pilih Control Panel . Klik kategori Jaringan dan Internet kemudian pilih Pusat Jaringan dan Berbagi.Dari opsi di sisi kiri, pilih Ubah pengaturan adaptor . Klik kanan pada ikon untuk Koneksi Nirkabel dan klik aktifkan .

Penjelasan:

Bukanlah perkara pelik menemukan cara terbaik dan tercepat menyambung wifi ataupun hotspot ke laptop windows anda. Ini berlaku untuk Sistem operasi termasuk windows 7, 8 atau windows 10. Rasanya sudah hampir setiap perangkat komputer ataupun laptop sudah dibenamkan adapter Wi-fi didalamnya. Sama sekali tidak ribet dan hal ini bisa dilakukan dengan hanya beberapa step saja.

Setidaknya ada beberapa catatan yang perlu anda ketahui saat akan menyambungkan hotspot wifi dari perangkat lain ke laptop yang anda miliki. Jika sumber hotspotnya adalah dari perangkat anda sendiri (entah itu via modem atau Hp), sebaiknya gunakan password yang tidak mudah ditebak. Tujuannya apa? Hal ini bertujuan untuk menjaga jaringan koneksi internet anda bersifat pribadi. Jika anda membiarkannya terbuka alias tidak memakai password, maka setiap orang yang ter-cover dengan jaringan hotspot anda akan bisa dengan mudah memakai hotspot milik anda tsb. Efeknya apa? Bisa jadi paket data anda akan cepat habis dan koneksi yang anda gunakan akan mengalami penurunan dari segi kecepatan, karena telah dibagi ataupun dipakai secara bersama-sama.

Gunakanlah Wifi anda dengan bijak. Jangan biarkan wifi menyala secara terus menerus jika tidak sedang digunakan. Jika Wifi anda menyala secara kontinue, tentu saja baterai laptop anda akan cepat habis.

Apapun merk laptop anda, entah itu Acer, Dell, Toshiba, Axio atau merk lainnya, langkah yang diperlukan sama saja. Tutorial ini bisa pula diaplikasikan pada OS Windows mulai dari Windows 7, 8 atau versi Windows 10.

Perlu di ingat, tidak semua merk laptop menyediakan tombol khusus WiFi. Anda cukup beruntung jika laptop yang anda pakai ternyata menyediakan tombol tersebut, karena hal ini akan memudahkan anda dalam mengaktifkan wifi untuk hotspot di laptop anda. Cara melakukannya? Gampang, cukup menekan tombol yang ada lambang WiFinya tsb dalam beberapa detik, nanti akan ada indikator yang terlihat (biasanya berupa lampu yang menyala).

Via Tombol Fn + Function (Tombol F)

Coba perhatikan keyboard di laptop anda. Di barisan paling atas akan nampak deretan Function mulai dari F1 sampai yang terakhir F12, biasanya di salah satu tombol function tsb ada lambang Wifi-nya (berbentuk gambar signal). Letaknya memang berbeda-beda disetiap laptop. Khusus di laptop saya merek Acer (Aspire) ini, tombol yang ada lambang WiFi tersebut ada di F3.  Sedangkan tombol Wi-Fi yang ada di Laptop saya yang satunya merk Lenovo Thinkpad ada di F8.

29. [Quiz Rimuru] Rimuru ingin membuat sebuah password untuk laptopnya, password itu adalah bilangan 5 angka abcde yang dibentuk dari angka angka 0 sampai 9. Jika diketahui a < b dan a tidak sama dengan 0. Berapa banyak password laptop yang bisa dibuat oleh Rimuru ? Note : Jawab dengan sungguh sungguh, jika jawabannya main main akan segera Rimuru report


Banyak susunan password yang bisa dibuat Rimuru ada 36.000 kombinasi

Pembahasan:

Kaidah pencacahan adalah metode untuk menghitung banyak susunan yang memungkinkan. Salah satu metode kaidah pencacahan adalah filling slots/pengisian tempat. Prinsipnya mengalikan banyak kejadian yang mungkin dari tiap penyusunnya.

Diketahui:

Password terdiri dari 5 angka abcde

Angka yang digunakan 0-9

Angka boleh digunakan berulang

a < b dengan a ≠ 0

Ditanya:

Banyak kombinasi password yang dapat dibentuk ...

Penyelesaian:

Pertama-tama hitung ada berapa cara menyusun ab dengan a < b dan a ≠ 0.

Jika b = 1, banyak cara a = 0 (tidak ada nilai a yang memenuhi karena a tidak boleh 0)

b = 2, banyak cara a = 1 (a = 1)

b = 3, banyak cara b = 2 (a = 0, 1)

...

b = 8, banyak cara a = 7 (a = 1, 2, 3, 4, 5, 6, 7)

b = 9, banyak cara a = 8 (a = 1, 2, 3, 4, 5, 6, 7, 8)

Banyak cara a dan b yang memenuhi:

[tex]1 + 2 + 3 + ... + 7 + 8 [/tex]

[tex] = (1 + 8)( \frac{8}{2} )[/tex]

[tex] = (9)(4)[/tex]

[tex] \boxed{= 36 \: susunan}[/tex]

Sekarang kita tinggal menghitung banyak susunan pada cde.

Angka yang digunakan 0 - 9 ===> ada 10 angka

Banyak cara c = 10

Banyak cara d = 10

Banyak cara e = 10

Banyak cara menyusun cde:

[tex]10 \times 10 \times 10[/tex]

[tex] \boxed{= 1000 \: susunan}[/tex]

Banyak susunan abcde:

[tex]36 \times 1000[/tex]

[tex] \boxed{ \boxed{= 36.000 \: susunan}}[/tex]

Kesimpulan:

Jadi, banyak susunan password yang dapat dibuat Rimuru ada 36.000 susunan
