16. Berat badan paman kurang 5kg dari 3 kali berat badan indra. jumlah berat badan mereka kurang dari 96 kg. tentukan model matematikanya!


Jawaban:

Misal : berat badan Indra = x

BB Paman = 3x - 5

Jumlah BB Paman + Indra kurang dari 96

Maka, model matematika :

(3x-5) + x < 96

4x - 5 < 96 (B)

Jawab:

Misal berat badan indra = x kg, maka:

berat badan paman = ( 3x - 5) kg.

Jumlah berat badan indra dan paman < 96

x + (3x - 5) < 96

4x - 5 < 96

jadi, model matematikanya adalah 4x - 5 < 96
