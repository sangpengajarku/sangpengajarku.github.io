3. bahasa jepangarti dari baka?arti dari ohayou?​


Baka = Bσdoh, Sangat/BangetOhayou = Selamat pagi

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Penjelasan :

BAKA「馬鹿・バカ・ばか」

Merupakan semacam kata umpatan yang biasa diartikan "Bσdoh". Sebenarnya tidak hanya sebagai kata umpatan, tetapi juga sebagai kata umum untuk mengartikan "Bσdoh" namun secara kasar. Kata ini tidak terlalu berkesan menghina jika digunakan di daerah Kantou. Namun jika digunakan di daerah Kansai, kata ini akan berkesan sangat menghina.

Selain diartikan sebagai "Bσdoh", kata ini juga bisa berarti "Sangat" atau "Banget". Digunakan untuk menyatakan sesuatu yang bangetnya nggak ngotak atau tidak masuk akal. Contoh penggunaannya seperti 'Baka tsuyoi', artinya "Kuat banget" atau "Kuatnya ngga ngotak".

- - - - - - -

OHAYOU「お早う・おはよう」

Merupakan salah satu contoh Aisatsu (salam/sapaan), yang memiliki arti "Selamat pagi". Ungkapan ini adalah ungkapan yang kasual, jadi biasa digunakan di situasi informal saja. Kalau dalam Bahasa Indonesia, ungkapan "Selamat pagi" itu terkesan sopan, dan untuk yang kasual nya biasa hanya dengan mengatakan "Pagi" saja. Sama hal nya dengan Ohayou, yang bentuk sopannya 'Ohayou gozaimasu'. Jadi Ohayou biasa juga diartikan "Pagi", tapi hanya sebagai makna konotatif (kiasan) saja.

5. Apa artinya Baka yarou?​


Arti bakayarou jika diartikan ke dalam bahasa Indonesia adalah dasar bodoh.

Pembahasan

Kata bakayarou ditulis dengan bentuk katakana dan juga hiragana バカやろう

バカやろう (bakayarou)

Dimana terdapat kata バカ (baka) ditulis dengan bentuk katakana yang dalam bahasa Indonesianya adalah bodoh atau bisa diartikan beg0.

Sedangkan diakhir kata setelah kata バカ (baka), terdapat kata やろう (yarou) yang ditulis dengan bentuk hiragana, biasanya salah satu kata ini bertujuan sebagai penekanan sekaligus menyatakan rasa kesal dan juga bisa digunakan untuk mengejek atau mengatai orang lain. Kata やろう (yarou) ini bisa diartikan 'dasar'.

------

Perlu diketahui ya, bahwa salah satu kata ini yaitu バカやろう (bakayarou) bukanlah termasuk jenis kata yang baik melainkan jenis kata dengan tingkatan kasar.

Umumnya kata bakayarou digunakan untuk mengejek orang lain dengan sebutan bodoh/beg0. Bisa juga untuk menyatakan rasa kekesalan atas apa yang telah diperbuat oleh orang tersebut.

(*seharusnya kata arigato yang benar ditulis arigatou ya, bukan arigato).

Pembahasan

ありがとうバカ

Arigatou baka

------

Berikut akan saya jelaskan lengkapnya dengan mengartikan satu-persatu katanya dan menyertakan penjelasannya :

› ありがとう (arigatou)

Merupakan salah satu bentuk ungkapan untuk menyatakan bentuk terima kasih namun dengan kesan informal. Sehingga kata ありがとう (arigatou) ini lebih baik digunakan kepada teman sebaya, saudara, kerabat, rekan dekat atau dalam bentuk situasi akrab.

› バカ (baka)

Merupakan salah satu jenis bentuk kata dengan tingkatan yang kasar alias tidak sopan, karena dalam bahasa Indonesia diartikan bod*h atau beg*.

Karena salah satu kata tersebut memiliki makna yang kasar sekaligus merupakan jenis kata yang mengandung ejekan, sehingga lebih baik jangan kita ucapkan secara sembarangan.

21. apa arti baka, dan sayonara dalam bahasa jepang​


Baka : bodoh, idi*t (terkadang bisa diartikan sangat yang bukan umpatan, penggunaan sesuai dengan konteks kalimatnya).Sayounara : selamat tinggal.

Pembahasan

Baka「莫迦・ばか・バカ」

Di Jepang, kata ini tidak akan terdengar sebagai kata yang kasar jika diucapkan di daerah Kantou, karena kata baka merupakan kata yang berasal dari dialek Kantou.

Sebaliknya jika mengucapkan kata ini di daerah Kansai, maka akan terdengar lebih kasar.

Kemudian, kata ini tidak sepenuhnya berfungsi sebagai kata kasar. Karena kata baka juga dapat diartikan sebagai sangat, amat, ataupun banget.

Sayounara「さようなら」

Untuk artinya ke dalam bahasa Indonesia memiliki arti yaitu selamat tinggal. Merupakan salam penutup yang formal.

Penggunaannya tidak boleh diucapkan sembarangan, karena kata sayounara hanya digunakan ketika terjadinya waktu perpisahan yang dimana kedua pihak tidak akan bertemu lagi atau akan bertemu namun dengan waktu yang sangat lama.
