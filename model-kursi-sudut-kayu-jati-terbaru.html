15. Pabrik A memproduksi dua jenis kursi, yaitu kursi rotan dan kursi jati, biaya produksi untuk dua set kursi rotan dan tiga set kursi jati adalah RP.18.0000.000,00. Pabrik B yang merupakan cabang dari pabrik A, memproduksi tiga set kursi rotan dan dua set kursi jati, dengan biaya produksi RP. 20.000.000,00 buatlah model matematika untuk persoalan tersebut


Saya bantu jawab ya
Diketahui :
Kursi Rotan = x
Kursi Jati = y

Pabrik A : 2x + 3y = 18.000.000
Untuk mencari x dan y maka terlebih dahulu dijumlahkan 2+3 = 5
2x = 2/5 × 18.000.000 = 7.200.000 ⇒ x = 7.200.000 / 2 = 3.600.000
3y = 3/5 × 18.000.000 = 10.800.000 ⇒ y = 10.800.000 / 3 = 3.600.000

Pabrik B : 3x + 2y = 20.000.000
Untuk mencari x dan y maka terlebih dahulu dijumlahkan 3+2 = 5
3x = 3/5 × 20.000.000 = 12.000.000 ⇒ x = 12.000.000 / 3 = 4.000.000
2y = 2/5 × 20.000.000 = 8.000.000 ⇒ y = 8.000.000 / 2 = 4.000.000

Dari nilai kedua Pabrik diatas dapat disimpulkan bahwa :
x = y
Dimana kedua nilai bernilai sama pada masing2 pabrik tersebut.
