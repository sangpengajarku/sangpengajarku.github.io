Jika besar vektor A = 4 satuan, membentuk sudut 30o dengan sumbu x positip, maka besar vektor tersebut dalam sumbu x dan sumbu y adalah … Pembahasan Diketahui : A = 4 satuan, Sudut = 30o Ditanya : Ax dan Ay ?



[1].
Diketahui:
F₁ = 3 N
F₂ = 4 N
Kedua vektor saling tegak lurus 

Ditanya:
Resultan kedua vektor

Penyelesaian:

Kedua vektor membentuk sudut 90° sehingga resultan dihitung dengan dalil Phytagoras
Gaya resultan = √ [F₁² + F₂²]
F resultan = √ [3² + 4²]
∴ F resultan = 10 N

[2].
Diketahui:
Vektor A = 4 satuan
Membentuk sudut 30° dengan sumbu x positip

Ditanya:
Besar komponen vektor tersebut dalam sumbu x dan sumbu y

Penyelesaian:

Besar komponen horizontal (terhadap sumbu x) adalah
⇔ Ax = A.cos α
⇔ Ax = (4)(cos 30°)
⇔ Ax = (4)(√3/2)
∴ Ax = 2√3 satuan

Besar komponen vertikal (terhadap sumbu y) adalah
⇔ Ay = A.sin α
⇔ Ay = (4)(sin 30°)
⇔ Ay = (4)(1/2)
∴ Ay = 2 satuan

8. Diketahui dua buah vektor dengan nilai secara berturut turut adalah 8N dan 15Nserta membentuk sudut 90°. Tentukan RESULTAN KEDUA VEKTOR tersebut!!!( menggunakan rumus Resultan Cosinus)​


Jawaban:

Rumus resultan vektor dengan cosinus yaitu :

v = [tex]\sqrt{v_{1} ^{2} +v_{2} ^{2}-2v_{1} v_{2} cos\alpha }[/tex]

v1 = 8N

v2 = 15N

Sudut = 90°

Substitusikan nilai ke rumus, sehingga didapat :

[tex]v = \sqrt{8^{2}+15^{2}-2.8.15.cos90 }[/tex]

[tex]v = \sqrt{64 + 225 -240.0}[/tex]

[tex]v=\sqrt{289}[/tex]

[tex]v = 17 N[/tex]

Maka, resultan vektor tersebut yaitu 17 N.

Note : Sa-chan kasih satu rumus cepat ya :)

Kalau membentuk sudut 90, pake pythagoras aja, jadi :

[tex]v=\sqrt{8^{2} +15^{2} } \\v=\sqrt{64+225} \\v=\sqrt{289} \\v= 17 N[/tex]

10. Dik.vektor segitiga ABC dengan titik titik sudut A (2,-3,2) B(-1,0,2) dan C (0,1,4).Dengan menggunakan rumus koainus antara dua vektor,tentukan sudut sudut berikut a.Sudut BAC b.Sudut ABC c.Sudut ACB


PERTANYAAN

Diketahui vektor segitiga ABC dengan titik-titik sudut A (2,-3,2), B(-1,0,2), dan C (0,1,4). Dengan menggunakan rumus kosinus antara dua vektor, tentukan sudut-sudut berikut:

a. Sudut BAC

b. Sudut ABC

c. Sudut ACB


PENYELESAIAN

Diketahui: A (2,-3,2) B(-1,0,2) C (0,1,4)


Ingat rumus jarak antara titik:

[tex] d = \sqrt{(x2-x1)^{2}+(y2-y1)^{2}+(z2-z1)^{2}} [/tex]


AB = [tex] \sqrt{(-1-2)^{2}+(0+3)^{2}+(2-2)^{2}} = \sqrt{9+9+0} = 3\sqrt{2} [/tex]

AC = [tex] \sqrt{(0-2)^{2}+(1+3)^{2}+(4-2)^{2}} = \sqrt{4+16+4} = 2\sqrt{6} [/tex]

BC = [tex] \sqrt{(0+1)^{2}+(1-0)^{2}+(4-2)^{2}} = \sqrt{1+1+4} = \sqrt{6} [/tex]


Gunakan aturan cosinus untuk mencari titik sudut.

a. Sudut BAC

BC² = AB² + AC² - 2.AB.AC.cos∠BAC

(√6)²=(3√2)²+(2√6)²-2(3√2)(2√6) cos∠BAC

6 = 18 + 24 - 24√3 cos∠BAC

36 = 24√3 cos∠BAC

cos∠BAC = 1/2 √3

∠BAC = 30°


b. Sudut ABC

AC² = AB² + BC² - 2.AB.BC.cos∠ABC

(2√6)²=(3√2)²+(√6)²-2(3√2)(√6) cos∠ABC

24 = 18 + 6 - 12√3 cos∠ABC

0 = 12√3 cos∠ABC

cos∠ABC = 0

∠ABC = 90°


c. Sudut ACB

AB² = AC² + BC² - 2.AC.BC.cos∠ACB

(3√2)²=(2√6)²+(√6)²-2(2√6)(√6) cos∠ACB

18 = 24 + 6 - 24 cos∠ACB

12 = 24 cos∠ACB

cos∠ACB = 1/2

∠ACB = 60°


Cek jumlah sudut segitiga,

⇒ ∠BAC + ∠ABC + ∠ACB = 180°

⇒ 30° + 90° + 60° = 180° (OK)

dua buah vektor setitik tangkap masing masing besarnya 6n dan 8n. jika kedua vektor saling mengapit sudut 90° , maka resultan kedua vektor tersebut adalah 10 N

Pembahasan

[tex]{\boxed { R = \sqrt{{F_1}^{2} + {F_2}^{2} + 2 F_1F_2 \: \: Cos \: α} }} [/tex]

Keterangan :

R = resultan vektorF₁ = vektor pertamaF₂ = vektor keduaα = sudut apit antara kedua vektor

________________________________

Diketahui :

[tex]F_1 = 6 \: N[/tex][tex]F_2 = 8 \: N [/tex]α = 90 °

Ditanya :

R ... ?

Dijawab :

[tex]R =\sqrt{{F_1}^{2} + {F_2}^{2} + 2 F_1F_2 \: \: Cos \: α} [/tex]

[tex]R = \sqrt{{6}^{2} + {8}^{2} + 2 \: \: (6 \: . \: 8)\: \: Cos \:90° } [/tex]

[tex]R = \sqrt{36 + 64 + 2 \: \: (48)\: \: \: 0 } [/tex]

[tex]R = \sqrt{100+ 0 \: } [/tex]

[tex]R = \sqrt{100} [/tex]

[tex]{\boxed {R = 10\: N}}[/tex]

jadi, maka resultan kedua vektor tersebut adalah 10 N
