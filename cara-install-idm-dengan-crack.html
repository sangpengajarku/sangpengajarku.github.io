Yang perlu di siapkan adalah : 

1. Master program IDM.

2. Patch IDM

Langkah untuk mengatasi masalah fake serial number : 

1. Uninstall terlebih dahulu IDM yang sekarang digunakan. 

2. Setelah selesai di uninstal masuk ke menu regedir dengan cara klik Start –> Run –> ketik “regedit” (tanpa tanda petik) –> klik OK 

3. Akan muncul menu editor, kemudian cari di HKEY_LOCAL_MACHINE –> SOFTWARE –> Internet Download Manager. 

4. Kemudian hapus folder Internet Download Manager tersebut. 

5. Tutup editor tersebut kemudian restart komputer 

Install kembali IDM dengan master yang telah di download tadi, langkahnya sebagai berikut : 

1. Ekstrak yang file yang telah di download tadi 

2. Install master IDM tersebut. 

3. setelah program terinstall copy patch.exe pada folder instalan. biasanya ada di C:\Program Files\Internet Download Manager. 

4. setelah tercopy jalankan patch.exe tersebut

28. apa yang dimaksud IDM ?


IDM (Internet Download Manager) adalah sebuah program untuk memudahkan kita untuk men”Download” data dari internet. Download tidak sama dengan Upload. Upload adalah suatu pengunggahan dari komputer atau laptop kita masuk kedalam internet. Contohnya seperti mengUpload foto di facebook atau yang ain-lainnya. Beberapa kegunaan yang terdapat didalam IDM ini adalah bisa menDownload data dari internet yang berupa file seperti video, musik dan lain-lain. Dengan menggunakan IDM, kita bisa mengambil data-data seperti file yang tadi dengan mudah dan cepat. Jika kita menDownload video-video yang ada diIDM adalah sebuah aplikasi yang digunakan  untuk mendownload .
