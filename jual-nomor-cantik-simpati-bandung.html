17. Samba seorang pedagang kosmetik menjual produk kecantikan merk A dan merk B. Merk A dibeli seharga Rp 20.000,00 per buah dijual dengan harga Rp 35.000,00 per buah. Merk B dibeli seharga Rp 40.000,00 per buah dijual dengan harga Rp 60.000,00 per buah. Jika ia mempunyai modal Rp 520.000,00 dan tokonya mampu menampung paling banyak 20 buah produk kecantikan, hitunglah keuntungan terbesar yang diperoleh dari penjualan produk kecantikan tersebut!


Merk A = x
Merk B = y

Merk A dibeli seharga Rp 20.000,00 per buah dan Merk B dibeli seharga Rp 40.000,00 per buah dengan modal Rp 520.000,00
20.000x + 40.000y ≤ 520.000
x + 2y ≤ 26

Tokonya mampu menampung paling banyak 20 buah produk kecantikan
x + y ≤ 20

Keuntungan Merk A = Rp35.000,00 - Rp20.000,00 = Rp15.000,00/buah
Keuntungan Merk B = Rp60.000,00 - Rp40.000,00 = Rp20.000,00/buah
f(x,y) = 15.000x + 20.000y

Pemodelan matematika
x ≥ 0, y ≥ 0, x + 2y ≤ 26, x + y ≤ 20

Untuk x + 2y = 26, jika x = 0 maka y = 13 dan jika y = 0 maka x = 26
Untuk x + y = 20, jika x = 0 maka y = 20 dan jika y = 0 maka x = 20

Gambar terlampir

Titik potong garis
x + 2y = 26
x +   y = 20
---------------- -
        y = 6
x = 20 - y = 20 - 6 = 14

Dari grafik terdapat titik pojok (0,13), (20,0), (14,6)
  x   y   f(x,y) = 15.000x + 20.000y
  0  13                260.000
 20  0                 300.000
 14  6                 330.000         <<< nilai maksimum

Keuntungan terbesar yang diperoleh adalah Rp330.000,00

Baik dalam promosi, sistem bisnis, dan lainnya.Buat blog. Blog penting sebagai sebagai salah satu cara anda berkomunikasi secara intensif kepada para anggota maupun calon anggota.
