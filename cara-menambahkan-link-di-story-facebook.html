9. Jelaskan langkah-langkah untuk menambahkan link antar kotak teks


Menyalin kotak teksKlik batas kotak teks yang ingin Anda salin.Tekan Ctrl+C.CATATAN: Pastikan penunjuk berada di batas kotak teks, tidak di dalamnya. Jika penunjuk berada di dalam kotak, menekan Ctrl+C akan menyalin teks, bukan kotak teks.Pilih salah satu lokasi dan tekan Ctrl+V untuk menempelkan kotak teks.Menghapus kotak teksKlik batas kotak teks yang ingin Anda hapus, lalu tekan Delete.CATATAN: Pastikan penunjuk berada di batas kotak teks dan tidak di dalamnya. Jika penunjuk berada di dalam kotak, menekan Delete akan menghapus teks di dalam kotak teks, bukan kotaknya.

29. Tolong Sebutkan bentuk coding web untukk membuat link ke facebook dan yahooo


Jawaban:

Untuk menyisipkan tautan link dalam website kita menuju Facebook atau Yahoo, kita bisa gunakan tag <a> dalam HTML.

Berikut saya contohkan pengimplementasian kodenya dalam file HTML.

note : karena rules dalam brainly tidak boleh mencantumkan suatu website, maka saya samarkan saja ya kak.

<html>

<body>

<a href="facebook dot com">Klik untuk menuju ke FB</a>

<a href="yahoo dot com">Klik untuk menuju ke Yahoo</a>

</body>

</html>

Jadi, untuk membuat link dalam web, kita perlu menambahkan tag <a> pada dokumen HTML dalam web yang kita buat.

Semoga membantu.
