3. ukuran baju XL sama L besaran mana?


Ukuran baju XL lebih besar daripada ukuran baju L. XL singkatan dari Xtra Large sedangkan L singkatan dari Large. Jadi ukuran XL lebih besar dari ukuran L. Urutan ukuran baju dari terkecil ke terbesar adalah S, M, L, XL, XXL.  

Pembahasan

Ukuran baju terdiri dari

S (Small) artinya ukuran bajunya kecil M (Medium) artinya ukuran bajunya sedang L (Large) artinya ukuran bajunya besar XL (Xtra Large) artinya ukuran bajunya lebih besar lagi dari L XXL (Xtra-Xtra Large) artinya ukuran bajunya lebih besar lagi dari XL

Jadi jika ditanya ukuran baju XL sama L besaran mana?

Jawabannya lebih besar ukuran baju XL

Ukuran xl = 102

Ukuran xxl = 106

Ukuran 3xl = 110

Untuk menyelesaikan soal ini kita gunakan formula Un = a + (n-1)b.

Keterangan:

Un = suku ke-n

a = suku pertama

n = banyak suku

b = beda (selisih tiap suku).

Penjelasan dengan langkah-langkah:

Diketahui:

Bu rani memproduksi baju perempuan dengan variasi ukuran s m l xl xxl 3xl.

Ada beberapa standar untuk tiap ukuran ukuran barisan berbentuk barisan aritmetika.

Ukuran terbesar 110. dan terkecil 90.

Ditanya:

Sebutkan ukuran-ukuran baju sesuai variasi ukuran.

Jawab:

Suku pertama = s = 90

Suku keenam = 3xl = 110

Un = a + (n-1)b

U₆ = a + (6-1)b

U₆ = a + 5b

110 = 90 + 5b

5b = 110 - 90

5b = 20

b = 20/5

b = 4

Ukuran s = 90

Ukuran m = 94

Ukuran l = 98

Ukuran xl = 102

Ukuran xxl = 106

Ukuran 3xl = 110

24. Ibu rani memproduksi baju perempuan dengan variasi ukuran s m l xl xxl. Ada beberapa standar ukuran untuk tiap ukuran. Meliputi lebar baju lebar pundak. Lingkar pinggul. Tiap ukuran membentuk barisan aritmetika Ukuran paling kecil 90 dan paling besar 110. a. UN adalah lingkar dada ukuran ke n


Standar ukuran baju yang dibuat Bu Rani adalah:

Ukuran S = 90Ukuran M = 95Ukuran L = 100Ukuran XL = 105Ukuran XXL = 110

Penjelasan dan langkah-langkah

Diketahui:

Bu Rani memproduksi pakaian dalam berbagai ukuran, antara lain S, M, L, XL dan XXL. Ukuran terbesar adalah 110, dan yang terkecil adalah 90.Ada beberapa standar untuk tiap ukuran ukuran barisan berbentuk barisan aritmetika.

Ditanya:

Sebutkan ukuran baju sesuai variasi ukuran.?

Jawab:

Suku pertama: S = 90

Suku kelima: XXL = 110

Untuk menyelesaikan soal ini kita gunakan formula Un = a + (n-1)b.

Un = a + (n-1)b

U5 = a + (5-1)b

U5 = a + 4b

110 = 90 + 4b

4b = 110 - 90

4b = 20

b = 20/4

b = 5

Ukuran barisan berbentuk barisan aritmetika adalah 5, maka:

Ukuran S = 90Ukuran M = 95Ukuran L = 100Ukuran XL = 105Ukuran XXL = 110
