10. Tuliskan cara membuat teks agar terdapat di kanan kolom pada aplikasi Ms.Excel​


Jawaban:

Dalam bentuk atau kotak teks, Anda dapat mengubah arah baca teks. Anda dapat memutar teks 90 atau 270 derajat sehingga teks tampak miring.

Klik kanan tepi bentuk atau kotak teks.

Dalam menu pintasan, pilih Format Bentuk.

Dalam panel Format Bentuk, klik Ukuran/Tata Letak & Properti  .

Di bawah Kotak Teks, pilih opsi yang diinginkan dalam daftar Arah teks.

Penjelasan:

Kotak teks dan sebagian besar bentuk dapat memuat teks di dalamnya. Teks dapat diatur ke posisi horizontal atau vertikal, dan dapat ditampilkan dalam satu baris atau dibungkus menjadi beberapa baris.

Anda dapat mengubah arah teks serta mengubah margin kotak teks dan bentuk untuk penspasian yang optimal, atau mengubah ukuran bentuk agar lebih pas dengan teks.

Garis dan konektor adalah bentuk yang tidak dapat memuat teks

14. 1. Bagaimanakah cara membuat dan menyimpan ketikan tulisan pada aplikasi word2. Bagaimanah cara membuat tabel pada aplikasi word3. Bagaimanakah cara menyimpan dokumen file dalam folder​


1. Untuk membuat tinggal membuka aplikasi word dan pilih "New Document" lalu kita sudah mulai bisa mengetik seusai keinginan kita, apabila ingin menyimpan bisa mengklik file, lalu pilih opsi save, atau bisa menggunakan shortcut (ctrl + s) dan tulisan akan disimpan ke direktori default

2. Jika ingin membuat tabel bisa dengan mengklik kolom insert, lalu akan ada pilihan untuk membuat table yang kemudian bisa dipilih ingin membuat berapa row dan column, jika sudah maka tabel akan otomatis terbuat dan tinggal disesuaikan ukurannya

3. Menyimpan dokumen pada folder yang diinginkan hampir sama dengan cara pertama namun pilih opsi "save as" lalu kita akan diminta untuk menentukan direktor yang akan dituju, jika sudah tinggal klik save

Semoga membantu yaa

1. Jika ingin membuat dokumen baru tinggal pakai shortcut (Ctrl +N) atau klik “file—>New Document” dan jika ingin save tinggal klik tombol shortcut (Ctrl + S) atau klik “File—>Save”

2. Dengan klik menu insert —> Table. Tinggal pilih mau berapa Row ( tabel horizontal ) dan Column ( Tabel Vertikal )

3. Dengan Klik menu “File —> Save As —> terus pilih akan disimpan di direktori mana file tersebut.

*Saran = untuk menyimpan file jangan di folder atau partisi C:, simpan dalam partisi lain selain C:
