2. Pada rangkaian gerbong kereta api perbandingan tempat duduk ekonomi dan bisnis adalah 1:3 banyak tempat duduk rangkaian kereta api tersebut 480 tempat duduk. Jika harga tiket ekonomi rp 80.000,00 dan harga tiket bisnis rp 120.000,00. Tentukan perolehan penjualan tiket penumpang jika tempat duduk kereta api terisi penuh penumpang!​


Jawaban:

Rp 52.800.000

Penjelasan dengan langkah-langkah:

Tiket ekonomi = 1/4 × 480

= 120 kursi

= 120 × Rp 80.000

= Rp 9.600.000

Tiket bisnis = 3/4 × 480

= 360 kursi

= 360 × Rp 120.000

= Rp 43.200.000

Perolehan penjualan = tiket ekonomi + tiket bisnis

= Rp 9.600.000 + Rp 43.200.000

= Rp 52.800.000

Jadi perolehan penjualan tiket Rp 52.800.000

semoga membantu
