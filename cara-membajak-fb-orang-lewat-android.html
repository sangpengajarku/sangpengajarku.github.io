Cara Transfer File dari Android ke Laptop Melalui WiFi

- Pertama-tama download dulu aplikasi WiFi FTP Server langsung di Google Play Store.

- Sambungkan android dan laptop ke jaringan WiFi yang sama. Jika di tempat anda tidak tersedia jaringan WiFi, nyalakan WiFi Hotspot android anda dan sambungkan laptop ke hotspot.

- Selanjutnya buka aplikasi WiFi FTP Server yang sudah didownload tadi. Aktifkan FTP Server dengan menekan tombol Awal dan izinkan aplikasi untuk mengakses file anda.

- Setelah diaktifkan, akan muncul informasi berupa URL server ftp://192.168.43.136:2221, username, dan password.

- Langkah selanjutnya yaitu mengakses file android di komputer menggunakan Windows Explorer.

- Buka File Explorer di Windows, lalu ketikkan URL server beserta port-nya pada address bar seperti gambar di bawah ini.

- Kemudian tekan Enter,

- Untuk mentransfer file dari android ke laptop cukup dengan copy dan paste file seperti biasanya.

- Selain menyalin file dari android ke laptop, anda juga bisa menyalin file dari laptop ke android. Caranya kurang lebih sama, copy file di laptop dan akses direktori android menggunakan URL server, lalu paste di dalam direktori android yang anda inginkan.

12. Cara melewati human verification di android


gak bisa kayak nya karena human verification itu adalah keamanan androidKatagori : Aplikasi Android
Pelajaran : Teknik informatika
Kelas : SMP
Kata kunci : melewati hutan verification di Android

Penjelasan :

biasanya orang ingin menghindari human verification di Android karena aplikasi yang digunakan ada berbayar.
Kebanyakan orang menyukai gratisan dengan cara mendownload aplikasi bajakan untuk di instal, tapi akan ada pemberitahuan 'lisensi tidak valid'.

Cara menghindari human verification di Android:

1. download dan instal aplikasi Lucky pacther
2. buka aplikasi lalu klik 'toolbox'
3. kemudian klik 'patch to Android'
4. ceklis semua menu : i. signature status always true
                                      ii. disable apk signature verification
                                      iii. disable signature verification in the package manager
5. klik terapkan tunggu hingga selesai dan mereboot setelah menyala kembali masuk ke Lucky pacther lagi dan klik tool box lagi
6. klik instal Modded google playstore.

15. bagaimana cara menginstal cwm android lewat folder system


BAHAN
1. Download dulu bahannya
2. Andoroid harus sudah di Root Devide dahulu
3. Download dan install apps Root Explorer v3.1
4. Download dan install apps terminal emulator (dari playstore)

INSTALLASI
1. Extract file instalasi CWM setelah di download dan taruh recovery.img di sd card (jangan di dalam folder)
2. Buka root explorer v3.1 yg telah di install dan masuk ke folder "system/" , cari file "recovery-from-boot.p" lalu move ke folder manapun, lbh baik di sdcard dan di luar agar lebih mudah diingat
3. buka terminal emulator dan ketik "su" tekan enter, kalau ada notifikasi, grant saja
4. Ketik di terminal: dd if=/sdcard/recovery.imgof=dev/block/mmcblkOp13 lalu enter
5. muncul tulisan 3 baris
6. ketik reboot di terminal emulator untuk me-restart android
7. saat terjadi reboot system, tekan tombol volume up dan volume down secara bersamaan untuk menguji apakah CWM (Clock World Mod) sudah terinstal dgn baik atau belum
