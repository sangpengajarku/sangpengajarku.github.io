8. langkah langkah mengedit profil edmodo dan mengganti nama pada edmodo


Langkah langkah mengedit profil edmodo dan mengganti nama pada edmodo:

Masuk profil siswa anda Arahkan kursor ke ikon pensil untuk mengedit atau mengganti foto profile Masuk settings untuk mengganti nama Pembahasan

Penjelasan langkah langkah mengedit profil edmodo dan mengganti nama pada edmodo:

Masuk profil siswa anda

Dengan memilih klik di pojok kanan atas gambar panah atau icon down-arrow png samping gambar foto profil mu serta tulis 'Me'. Setelah di klik, pilih menu Profile. Kemudian akan menampilkan profil anda.

Arahkan kursor ke ikon pensil untuk mengedit atau mengganti foto profile

Gerakan kursor ke ikon pensil, dan klik untuk memilih apa yang akan di edit. Jika sudah di klik, pilih yang akan di edit, semisal foto profil, klik foto profile dan di rubah sesuai foto yang akan di pilih

Untuk mengganti nama

Pertama-tama yaitu klik panah icon down arrow, pojok kanan atas samping foto profilemu. Dan kalimat me, setelah itu pilih menu settings. Setelah sudah memilih menu settings, setelah itu pilih untuk mengedit nama pertama dan terakhir

16. file apa yang harus di edit ketika mengganti port listening web server apache dari 80 menjadi 2000â€‹


Jawaban:

httpd.conf

Penjelasan:

Untuk mengubah port listening yang digunakan oleh Apache web server, maka yang harus diubah adalah file httpd.conf dengan mengubah baris bagian :

Listen 80

Menjadi :

Listen 2000

Untuk lokasi file httpd.conf berada, tergantung pada saat installasi, jika menggunakan xampp berada di folder installasi xampp/apache/conf/httpd.conf, sedangkan jika menggunakan Ubuntu, berada di /etc/apache2/apache2.conf
