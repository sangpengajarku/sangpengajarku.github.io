5. Sebuah trafo memiliki perbandingan lilitan sekunder dan lilitan primer 3:11, digunakan pada sebuah rumah yang memiliki daya 2200 watt dengan tegangan dari PLN 220V dengan efisiensi trafo 80%. Tentukan besar tegangan dan kuat arus sekundernya!


Mata pelajaran : fisika

Kelas: IX SMP

Kategori : kemagnetan

Kata kunci : transformator step step up, tegangan primer sekunder, kuat arus primer sekunder

Kode kategori berdasarkan kurikulum KTSP : 9.6.4





JAWABAN :





Berdasarkan soal diatas diketahui bahwa :

- Perbandingan jumlah lilitan pada kumparan primer dan sekunder adalah 3 : 11 -> Np : Ns = 3 : 11

- tegangan pada kumparan primer adalah 220 volt -> Vp = 220 volt

- daya masukan primer adalah 2200 wat -> Pp = 2200 watt

- efisiensi trafo adalah 80% -> η = 80%

Selanjutnya pada soal diatas yang ditanyakan adalah tegangan sekunder dan kuat arus sekunder, maka dari itu kita perlu mencari Vs pada trafo dan Is pada trafo.

- Untuk mencari tegangan pada kumparan sekunder caranya sebagai berikut :

Np/Ns = Vp/Vs

Np x Vs = Ns x Vp

Vs = Ns x Vp / Np

Vs = 11 x 220 volt / 3

Vs = 2420 / 3

Vs = 806,6666666667

Vs = 806,7 volt



- sebelum mencari kuat arus sekunder kita cari kuat arus primer pada  transformator caranya sebagai berikut :

Pp = Vp x Ip

Ip = Pp/Vp

Ip = 2200 watt / 220 volt

Ip = 10 A



- Untuk mencari kuat arus pada kumparan sekunder  caranya sebagai berikut :

η = (Is.Vs / Ip.Vp) x 100%

80% = (Is . 806,7 volt / 10 A . 220 volt ) x 100%

80% =( Is. 806,7 volt / 2200) x 100%

80 x 2200 = Is x 806,7 x 100

176.000 = Is x 80.670

Is = 176.000/80.670

Is = 2,1817280278 A

Is = 2,2 A

Berdasarkan uraian diatas dapat ditarik kesimpulan bahwa  tegangan pada kumparan sekunder adalah 806,7 volt sedangkan kuat arus pada kumparan sekunder adalah 2,2 A.





PEMBAHASAN LEBIH LANJUT :





Trafo merupakan alat yang digunakan untuk mengubah tegangan listrik bolak balik menjadi lebih besar atau lebih kecil.

Trafo sendiri dibagi menjadi 2 yaitu trafo step up (menaikan tegangan) dan trafo step down (menurunkan tegangan).

Adapun ciri ciri trafo step up yaitu nilai Vs > Vp (tegangan sekunder lebih besar dibandingkan tegangan primer) , Ns > Np (jumlah lilitan sekunder lebih besar dibandingkan jumlah lilitan primer), Is < Ip (kuat arus sekunder lebih kecil dibandingkan dengan kuat arus primer).

Untuk ciri ciri trafo step down yaitu nilai Vs < Vp (tegangan sekunder lebih kecil dibandingkan tegangan primer) , Ns < Np (jumlah lilitan sekunder lebih kecil dibandingkan jumlah lilitan primer), Is > Ip (kuat arus sekunder lebih besar dibandingkan dengan kuat arus primer).



Untuk mencari rumus pada transformator adalah sebagai berikut :

Np/Ns = Vp / Vs = Is/Ip

Ket:

Ns = jumlah lilitan pada kumparan primer

Np = jumlah lilitan pada kumparan sekunder

Vp = Tegangan pada kumparan primer (masukkan)

Vs = Tegangan pada kumparan sekunder (keluaran)

Ip = Arus pada kumparan primer

Is = Arus pada kumparan sekunder

Sedangkan untuk mencari efisiensi trafo rumusnya sebagai berikut :

η = (Is.Vs / Ip.Vp) x 100%

Keterangan :

η = efisiensi trafo

Vp = Tegangan pada kumparan primer (masukkan)

Vs = Tegangan pada kumparan sekunder (keluaran)

Ip = Arus pada kumparan primer

Is = Arus pada kumparan sekunder

22. Sebuah transformator memiliki efisiensi 80 %. Jika daya pada kumparan primer 1300 watt, maka daya sekunder trafo tersebut adalah.... ​


Jawaban:

[tex]\bf\large P_{Sekunder}=1.040\ Watt [/tex]

Penjelasan dengan langkah-langkah:

[tex]\sf\large Diketahui: [/tex]

[tex]\sf\large η=80\% [/tex]

[tex]\sf\large P_{P}=1.300\ Watt [/tex]

Jawab:

Gunakan rumus:

[tex]\sf\large η=\Large{\frac{P_{S}}{P_{P}}}\large×100\% [/tex]

[tex]\sf\large P_{S}=\Large{\frac{80\%×P_{P}}{100\%}} [/tex]

[tex]\sf\large P_{S}=\Large{\frac{80×1.300}{100}} [/tex]

[tex]\sf\large P_{S}=1.040\ Watt [/tex]

Jadi daya sekundernya adalah [tex]\sf\large 1.040\ Watt [/tex]
