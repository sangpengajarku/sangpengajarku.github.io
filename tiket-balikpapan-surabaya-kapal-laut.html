Jika harga tiket kelas 1 Rp 500.000,00 per orang dan kelas ekonomi Rp 200.000,00 per orang, maka pendapatan maksimum yang dapat diterima oleh pengusaha kapal adalah


Penumpang kelas 1 = x
Penumpang kelas ekonomi = y

x [tex] \geq [/tex] 0
y [tex] \geq [/tex] 0
x + y [tex] \leq [/tex] 300 orang
60x + 20y [tex] \leq [/tex] 14.000 kg
f(x,y) = 500.000x + 200.000y

Ditanyakan pendapatan maksimum .. f(x,y) = MAX ..
Kita Cari titik Potong nya dahulu (Biasanya di gambar)
Akan Mendapat kan 3 titik .. waktu x = 0 , y = 0 ,dan titik potongnya ..
Kita cari titik Potongnya dulu ..

x + y = 300 dikalikan 20
60x + 20y = 14000 dikalikan 1

20x + 20y = 6000
60x + 20y = 14000
= .. -40x = -8000
= .. x = 200
=> x + y = 300
= 200 + y = 300
= y = 300 - 200
= .. y = 100
Ketemu Titik Potong (x,y) = (200,100)
Dan daerah kurang dari .. saat x = 0 nilai y = 300 .. (0,300)
Dan daerah kurang dari .. saat y = 0 nilai x = 14000/60 .. (14000/60,0)

Jadi Tiga titik itu ..
f(x,y) = 500.000x + 200.000y
f(200,100) = ..
= 500.000(200) + 200.000(100)
= 100.000.000 + 20.000.000 = 120.000.000 (Maks)
f(0,300) = ..
= 500.000(0) + 200.000(300) = 60.000.000 (Min)
f(14000/60,0) =..
= 500.000(14000/60) + 200.000(0) = 116.666.667

Jadi pendapatan maksimum nya adalah Rp 120.000.000 ..
