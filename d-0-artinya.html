6. Tentukan nilai k dari persamaan kuadrat 2x^2- 4x + k = 0 jika akar- akarnya kembar ( artinya diskriminan D= 0)


Jawab:

Nilai k dari persamaan kuadrat 2x² - 4x + k = 0 jika akar- akarnya kembar adalah 2.

Penjelasan:

Dari bentuk umum persamaan kuadrat ax² +bx + c = 0, diskriminannya adalah D = b² - 4ac.

Syarat persamaan kuadrat mempunyai akar-akar kembar adalah D = 0.

2x² - 4x + k = 0 ⇒ a = 2, b = -4, dan c = k

D = 0

(-4)² - 4(2)(k) = 0

16 - 8k = 0

8k = 16

∴ Nilai k - 2

20. Nilai Diskminan suatu PK dapat dipergunakan untuk mengetahui jenis-jenis akar PK tersebut. Jika D > 0 maka PK memiliki akar nyata yang berbeda, jika D = 0 maka PK memiliki akar nyata yang sama (akar kembar) dan jika D < 0 berarti PK memiliki akar hayal. Berdasar hal tersebut, tentukan jenis akar-akar PK berikut :​


Penjelasan dengan langkah-langkah:

DISKRIMINAN PERSAMAAN KUADRAT

rumus: D = b² - 4ac

[tex] { - p}^{2} - 2p - 1 = 0 [/tex]

a = -1, b= -2, c = -1

D = b² - 4ac

= (-2)² - 4(-1)(-1)

= 4 - 4

= 0

(Karena D= 0, persamaan tersebut memiliki akar kembar)

[tex] { - 3n}^{2} + 4n + 1 = 0[/tex]

a = -3, b= 4, c = 1

D = 4² - 4(-3)(1)

= 16 + 12

= 38

(Karena D>0, persamaan tersebut memiliki akar nyata berbeda)

[tex] {4t}^{2} - 2t + 2 = 0[/tex]

a = 4, b= -2, c = 2

D = (-2)² - 4(4)(2)

= 4 - 32

= -28

(Karena D<0, persamaan tersebut memiliki akar hayal)

[tex] {4k}^{2} - 12k + 9 = 0[/tex]

a = 4, b = -12, c = 9

D= (-12)² - 4(4)(9)

= 144 - 144

= 0

(Karena D = 0, persamaan tersebut memiliki akar kembar)

[tex] { - 2x}^{2} + 5x - 3 = 0[/tex]

a = -2, b = 5, c = -3

D = 5² - 4(-2)(-3)

= 25 - 24

= 1

(Karena D>0, persamaan tersebut memiliki akar nyata berbeda)
